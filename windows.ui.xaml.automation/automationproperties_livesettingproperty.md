# Windows.UI.Xaml.Automation.AutomationProperties.LiveSettingProperty

## -description
Identifies the [AutomationProperties.LiveSetting](/uwp/api/windows.ui.xaml.automation.automationproperties#xaml-attached-properties) attached property, which is supported by an automation element that represents a live region.

## -property-value
The property identifier for the [AutomationProperties.LiveSetting](/uwp/api/windows.ui.xaml.automation.automationproperties#xaml-attached-properties) attached property.

## -remarks
The [LiveSetting](/uwp/api/windows.ui.xaml.automation.automationproperties#xaml-attached-properties) property indicates the "politeness" level that a client should use to notify the user of changes to the live region. This property can be one of the values from the [AutomationLiveSetting](../windows.ui.xaml.automation.peers/automationlivesetting.md) enumeration.
