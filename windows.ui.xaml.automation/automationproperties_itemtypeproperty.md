# Windows.UI.Xaml.Automation.AutomationProperties.ItemTypeProperty

## -description
Identifies the [AutomationProperties.ItemType](automationproperties_itemtype.md) attached property, which is a text string describing the type of the automation element.

## -property-value
The identifier for the [AutomationProperties.ItemType](automationproperties_itemtype.md) dependency property.

## -remarks
[ItemType](automationproperties_itemtype.md) is used to obtain information about items in a list, tree view, or data grid. For example, an item in a file directory view might be a "Document File" or a "Folder".

When [ItemType](automationproperties_itemtype.md) is supported, the string must match the application UI language or the operating system default UI language.
