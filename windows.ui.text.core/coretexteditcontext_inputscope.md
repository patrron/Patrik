# Windows.UI.Text.Core.CoreTextEditContext.InputScope

## -description
Gets or sets a value that indicates the input scope of the text input control.

## -property-value
One of the [CoreTextInputScope](coretextinputscope.md) enumeration values. The default is **Default**.
