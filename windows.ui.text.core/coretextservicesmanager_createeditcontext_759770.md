# Windows.UI.Text.Core.CoreTextServicesManager.CreateEditContext

## -description
Creates a context object used by a text input control to communicate with the text input server. Each text input control must create its own context object.

## -returns
A [CoreTextEditContext](coretexteditcontext.md) instance, which is the primary object used for communicating with the text input server.
