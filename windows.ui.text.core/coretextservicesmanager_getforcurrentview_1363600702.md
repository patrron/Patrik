# Windows.UI.Text.Core.CoreTextServicesManager.GetForCurrentView

## -description
Gets the [CoreTextServicesManager](coretextservicesmanager.md) object for the currently active view.

## -returns
A [CoreTextServicesManager](coretextservicesmanager.md) instance, which can be used to create further objects to support the app's text input scenarios.
