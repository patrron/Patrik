# Windows.Media.Audio.AudioFileOutputNode.Reset

## -description
Resets the audio file output node.

## -remarks
Calling this method causes [DiscardQueuedFrames](../windows.media.effects/ibasicaudioeffect_discardqueuedframes_1358903059.md) to be called on any audio effects applied to the [AudioFileOutputNode](audiofileoutputnode.md).
