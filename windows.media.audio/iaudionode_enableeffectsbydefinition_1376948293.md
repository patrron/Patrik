# Windows.Media.Audio.IAudioNode.EnableEffectsByDefinition

## -description
Enables all effects in the [EffectDefinitions](iaudionode_effectdefinitions.md) list with the specified effect definition.

## -parameters
### -param definition
The effect definition of the effects to enable.
