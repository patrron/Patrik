# Windows.Media.Audio.CreateAudioFileInputNodeResult

## -description
Represents the result of attempting to create an audio file input node.

## -remarks
Get an instance of this class by calling [AudioGraph.CreateFileInputNodeAsync](audiograph_createfileinputnodeasync_152724062.md). Check the [Status](createaudiofileinputnoderesult_status.md) property to determine if the node was successfully created, and if so, get the instance of the node with the [Node](createaudiofileinputnoderesult_node.md) property.
