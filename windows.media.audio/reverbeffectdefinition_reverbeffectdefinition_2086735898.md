# Windows.Media.Audio.ReverbEffectDefinition.ReverbEffectDefinition

## -description
Initializes a new instance of the [ReverbffectDefinition](reverbeffectdefinition.md) class.

## -parameters
### -param audioGraph
The [AudioGraph](audiograph.md) with which the effect will be used. Using effects outside of the audio graph it was created with is not supported.
