# Windows.ApplicationModel.Core.CoreApplication.Run

## -description
Runs an external view provider factory to obtain a view provider.

## -parameters
### -param viewSource
An implementation of a view provider factory.

## -remarks
### Windows Phone 8

This API is supported in native apps only.
