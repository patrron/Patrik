# Windows.Media.Playback.MediaPlayer.CopyFrameToVideoSurface

## -description
Copies the current video frame from the **MediaPlayer** to the provided **IDirect3DSurface**.

## -parameters

### -param destination
The **IDirect3DSurface** to which the video frame is copied.
