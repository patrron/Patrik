# Windows.Media.Playback.MediaPlaybackCommandManagerFastForwardReceivedEventArgs

## -description
Provides data or the [MediaPlaybackCommandManager.FastForwardReceived](mediaplaybackcommandmanager_fastforwardreceived.md) event, which occurs when the command manager receives a command from the **Fast-forward** button of the System Media Transport Controls.

## -remarks
Get an instance of this class by handling the [MediaPlaybackCommandManager.FastForwardReceived](mediaplaybackcommandmanager_fastforwardreceived.md) event.
