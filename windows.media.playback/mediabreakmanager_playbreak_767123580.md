# Windows.Media.Playback.MediaBreakManager.PlayBreak

## -description
Directs the [MediaPlayer](mediaplayer.md) to begin playing the specified [MediaBreak](mediabreak.md) immediately.

## -parameters
### -param value
The [MediaBreak](mediabreak.md) to be played.
