# Windows.Media.Playback.MediaPlaybackItem.CanSkip

## -description
Gets or sets a value indicating whether the [MediaPlaybackItem](mediaplaybackitem.md) can be skipped.

## -property-value
True if the [MediaPlaybackItem](mediaplaybackitem.md) can be skipped; otherwise, false.
