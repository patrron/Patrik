# Windows.Media.Playback.MediaPlaybackSession.PlaybackState

## -description
Gets a value indicating the current playback state of the [MediaPlaybackSession](mediaplaybacksession.md), such as buffering or playing.

## -property-value
The current playback state of the [MediaPlaybackSession](mediaplaybacksession.md)
