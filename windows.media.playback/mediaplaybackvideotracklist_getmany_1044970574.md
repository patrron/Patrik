# Windows.Media.Playback.MediaPlaybackVideoTrackList.GetMany

## -description
Retrieves the video tracks that start at the specified index in the list.

## -parameters
### -param startIndex
The zero-based index of the start of the video tracks in the list.

### -param items
The video tracks that start at *startIndex* in the list.

## -returns
The number of video tracks retrieved.
