# Windows.Media.Playback.MediaPlayer.RealTimePlayback

## -description
Gets or sets a value that configures the [MediaPlayer](mediaplayer.md) for real-time scenarios such as live streaming video.

## -property-value
True if the [MediaPlayer](mediaplayer.md) is being used for real-time media; otherwise, false.
