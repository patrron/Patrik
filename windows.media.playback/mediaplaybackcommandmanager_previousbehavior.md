# Windows.Media.Playback.MediaPlaybackCommandManager.PreviousBehavior

## -description
Gets an object that defines the behavior when the **Previous** button in the System Media Transport Controls is tapped.

## -property-value
An object that defines the behavior when the **Previous** button is tapped.
