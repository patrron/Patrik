# Windows.Media.Playback.MediaPlaybackCommandManager.AutoRepeatModeBehavior

## -description
Gets an object that defines the behavior when the **Auto-repeat** control in the System Media Transport Controls is toggled.

## -property-value
An object that defines the behavior when the **Auto-repeat** control is toggled.
