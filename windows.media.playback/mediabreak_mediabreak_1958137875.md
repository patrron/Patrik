# Windows.Media.Playback.MediaBreak.MediaBreak

## -description
Initializes a new instance of the [MediaBreak](mediabreak.md) class with the specified insertion method.

## -parameters
### -param insertionMethod
A value indicating whether the media break will pause the currently playing media content.
