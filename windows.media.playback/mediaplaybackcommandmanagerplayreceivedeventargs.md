# Windows.Media.Playback.MediaPlaybackCommandManagerPlayReceivedEventArgs

## -description
Provides data or the [MediaPlaybackCommandManager.PlayReceived](mediaplaybackcommandmanager_playreceived.md) event, which occurs when the command manager receives a command from the **Play** button of the System Media Transport Controls.

## -remarks
Get an instance of this class by handling the [MediaPlaybackCommandManager.PlayReceived](mediaplaybackcommandmanager_playreceived.md) event.
