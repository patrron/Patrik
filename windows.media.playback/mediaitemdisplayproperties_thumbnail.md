# Windows.Media.Playback.MediaItemDisplayProperties.Thumbnail

## -description
Gets or sets the thumbnail image for a [MediaPlaybackItem](mediaplaybackitem.md) that will be displayed by the System Media Transport Controls when the item is played.

## -property-value
A stream containing the thumbnail image data for a [MediaPlaybackItem](mediaplaybackitem.md).
