# MediaPlaybackState

## -description
Specifies the playback state of a [MediaPlaybackSession](mediaplaybacksession.md). 


## -enum-fields
### -field None:0
No current state.

### -field Opening:1
A media item is opening.

### -field Buffering:2
A media item is buffering.

### -field Playing:3
A media item is playing.

### -field Paused:4
Playback of a media item is paused.
