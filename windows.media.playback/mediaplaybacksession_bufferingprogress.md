# Windows.Media.Playback.MediaPlaybackSession.BufferingProgress

## -description
Gets a value that indicates the current buffering progress of a [MediaPlaybackSession](mediaplaybacksession.md).

## -property-value
The current buffering progress. The value ranges from 0 to 1.0. Multiply by 100 to obtain a percentage.
