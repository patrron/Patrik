# Windows.Media.Playback.MediaPlaybackItem.FindFromMediaSource

## -description
Retrieves the [MediaPlaybackItem](mediaplaybackitem.md) associated with the provided [MediaSource](../windows.media.core/mediasource.md).

## -parameters
### -param source
The [MediaSource](../windows.media.core/mediasource.md) associated with the [MediaPlaybackItem](mediaplaybackitem.md).

## -returns
The [MediaPlaybackItem](mediaplaybackitem.md) associated with the provided [MediaSource](../windows.media.core/mediasource.md).
