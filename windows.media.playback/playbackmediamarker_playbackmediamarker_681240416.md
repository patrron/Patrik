# Windows.Media.Playback.PlaybackMediaMarker.PlaybackMediaMarker

## -description
Initializes a new instance of the [PlaybackMediaMarker](playbackmediamarker.md) class with the specified offset into the media timeline, type string, and text.

## -parameters
### -param value
The media marker's offset into the media timeline.

### -param mediaMarketType
The type of the media marker.

### -param text
The text of the media marker.
