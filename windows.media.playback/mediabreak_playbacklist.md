# Windows.Media.Playback.MediaBreak.PlaybackList

## -description
Gets the [MediaPlaybackList](mediaplaybacklist.md) containing the [MediaPlaybackItem](mediaplaybackitem.md) objects that will be played as part of the [MediaBreak](mediabreak.md).

## -property-value
The [MediaPlaybackList](mediaplaybacklist.md) containing the [MediaPlaybackItem](mediaplaybackitem.md) objects that will be played as part of the [MediaBreak](mediabreak.md).
