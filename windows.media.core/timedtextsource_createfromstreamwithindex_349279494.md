# Windows.Media.Core.TimedTextSource.CreateFromStreamWithIndex

## -description
Creates a new instance of [TimedTextSource](timedtextsource.md) that uses image-based subtitles from the provided image and index streams and sets the default language.

## -parameters

### -param stream
A stream containing the image data for image-based subtitles.

### -param indexStream
A stream containing the index data for image-based subtitles.

### -param defaultLanguage
A string indicating the default language for the timed text source.

## -returns
The new timed text source.
