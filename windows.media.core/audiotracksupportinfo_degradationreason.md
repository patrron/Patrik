# Windows.Media.Core.AudioTrackSupportInfo.DegradationReason

## -description
Gets a value indicating the reason that the audio decoder is degrading the [AudioTrack](audiotrackopenfailedeventargs.md), if the audio is degraded.

## -property-value
The reason that the audio decoder is degrading the **AudioTrack**.
