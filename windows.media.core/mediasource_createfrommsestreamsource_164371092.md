# Windows.Media.Core.MediaSource.CreateFromMseStreamSource

## -description
Creates an instance of [MediaSource](mediasource.md) from the provided [MseStreamSource](msestreamsource.md).

## -parameters
### -param mediaSource
The [MediaStreamSource](mediastreamsource.md) from which the [MediaSource](mediasource.md) is created.

## -returns
The new media source.
