# Windows.Media.Core.MediaSource.CreateFromMediaBinder

## -description
Creates an instance of [MediaSource](mediasource.md) from the provided [MediaBinder](mediabinder.md).

## -parameters
### -param binder
The [MediaBinder](mediabinder.md) with which the **MediaSource** is associated.

## -returns
The new media source.
