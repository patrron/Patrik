# Windows.Media.Core.MediaBinder.Token

## -description
Gets or sets an app-specified string that is used to identify the media content that the app should bind when the [Binding](mediabinder_binding.md) event is raised.

## -property-value
An app-specified string that is used to identify the specific media content that the app should bind.
