# Windows.Media.Core.MediaSource.CreateFromUri

## -description
Creates an instance of [MediaSource](mediasource.md) from the provided [Uri](../windows.foundation/uri.md).

## -parameters
### -param uri
The URI from which the [MediaSource](mediasource.md) is created.

## -returns
The new media source.
