# Windows.Media.Core.MediaBinder.Binding

## -description
Occurs when a media player is ready for the media content to be bound to a [MediaSource](mediasource.md). Use the handler for this to set the media content for the source by calling [SetStream](mediabindingeventargs_setstream.md), [SetStreamReference](mediabindingeventargs_setstreamreference.md), or [SetUri](mediabindingeventargs_seturi.md).
