# Windows.Media.Core.MediaSource.CreateFromMediaStreamSource

## -description
Creates an instance of [MediaSource](mediasource.md) from the provided [MediaStreamSource](mediastreamsource.md).

## -parameters
### -param mediaSource
The [MediaStreamSource](mediastreamsource.md) from which the [MediaSource](mediasource.md) is created.

## -returns
The new media source.
