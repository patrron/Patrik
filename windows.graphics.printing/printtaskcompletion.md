# PrintTaskCompletion

## -description
Specifies the completion status of a print task.

## -enum-fields
### -field Abandoned:0
An abandoned print task.

### -field Canceled:1
A canceled print task.

### -field Failed:2
A failed print task.

### -field Submitted:3
A submitted print task.
