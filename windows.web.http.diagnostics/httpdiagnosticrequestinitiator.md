# HttpDiagnosticRequestInitiator

## -description
Provides the download request initiator type.

## -enum-fields
### -field ParsedElement:0
A parsed element

### -field Script:1
A script resource

### -field Image:2
An image resource

### -field Link:3
A link

### -field Style:4
A CSS element

### -field XmlHttpRequest:5
An XML HTTP request

### -field Media:6
A media resource

### -field HtmlDownload:7
An HTML download

### -field Prefetch:8
A pre-fetch request

### -field Other:9
An other resource

### -field CrossOriginPreFlight:10
A cross-origin pre-flight request

### -field Fetch:11
A fetch request

### -field Beacon:12
A one-way beacon request
