# Windows.Web.Http.Diagnostics.HttpDiagnosticProvider.CreateFromProcessDiagnosticInfo

## -description
Creates a new [HttpDiagnosticProvider](httpdiagnosticprovider.md) based on the specified [ProcessDiagnosticInfo](../windows.system.diagnostics/processdiagnosticinfo.md) object.

## -parameters
### -param processDiagnosticInfo
The [ProcessDiagnosticInfo](../windows.system.diagnostics/processdiagnosticinfo.md) that identifies the process to watch.

## -returns
The newly created [HttpDiagnosticProvider](httpdiagnosticprovider.md).
