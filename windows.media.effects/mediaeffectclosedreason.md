# MediaEffectClosedReason

## -description
Specifies the reason why a media effect was closed.

## -enum-fields
### -field Done:0
The operation is complete.

### -field UnknownError:1
An unknown error has occurred.

### -field UnsupportedEncodingFormat:2
The media effect does not support the required encoding format.

### -field EffectCurrentlyUnloaded:3
The effect has been unloaded.
