# PhotoImportContentTypeFilter

## -description
Specifies the content types that are included in a photo import operation.

## -enum-fields
### -field OnlyImages:0
Include only images.

### -field OnlyVideos:1
Include only videos.

### -field ImagesAndVideos:2
Include images and videos.


### -field ImagesAndVideosFromCameraRoll:3
Include images and videos located in the Camera Roll directory or any of its subdirectories.
