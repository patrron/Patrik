# Windows.Media.Import.PhotoImportOperation.ContinueImportingItemsAsync

## -description
Causes in-progress import items operations to continue.

## -property-value
An asynchronous operation that returns a [PhotoImportImportItemsResult](photoimportimportitemsresult.md) object on successful completion.
