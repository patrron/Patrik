# PhotoImportStorageMediumType

## -description
Specifies the type of a storage medium.

## -enum-fields
### -field Undefined:0
The storage medium type is undefined.

### -field Fixed:1
The storage medium is fixed.

### -field Removable:2
The storage medium is removable.
