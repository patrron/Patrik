# Windows.Media.Import.PhotoImportSource.FromFolderAsync

## -description
Creates a new instance of [PhotoImportSource](photoimportsource.md) from the specified root folder.

## -parameters
### -param sourceRootFolder
The root folder from which the photo import source is created.

## -returns
An asynchronous operation that returns a [PhotoImportSource](photoimportsource.md) upon successful completion.
