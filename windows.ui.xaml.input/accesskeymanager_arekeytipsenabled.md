# Windows.UI.Xaml.Input.AccessKeyManager.AreKeyTipsEnabled

## -description
Gets or sets a value that specifies whether KeyTips are shown for access keys.

## -property-value
**true** to show KeyTips; otherwise, **false**. The default is **true**.
