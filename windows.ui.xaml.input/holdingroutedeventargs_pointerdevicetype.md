# Windows.UI.Xaml.Input.HoldingRoutedEventArgs.PointerDeviceType

## -description
Gets the [PointerDeviceType](../windows.devices.input/pointerdevicetype.md) for the pointer device that initiated the associated input event.

Equivalent WinUI property: [Microsoft.UI.Xaml.Input.HoldingRoutedEventArgs.PointerDeviceType](/windows/winui/api/microsoft.ui.xaml.input.holdingroutedeventargs.pointerdevicetype).

## -property-value
The [PointerDeviceType](../windows.devices.input/pointerdevicetype.md) for this event occurrence.
