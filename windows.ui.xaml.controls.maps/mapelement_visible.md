# Windows.UI.Xaml.Controls.Maps.MapElement.Visible

## -description
Gets or sets a value that indicates whether the [MapElement](mapelement.md) is visible on the [MapControl](mapcontrol.md).

## -property-value
**true** if the [MapElement](mapelement.md) is visible on the [MapControl](mapcontrol.md); otherwise, **false**.
