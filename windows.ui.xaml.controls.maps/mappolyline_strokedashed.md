# Windows.UI.Xaml.Controls.Maps.MapPolyline.StrokeDashed

## -description
Gets or sets a value that indicates whether the line used to draw the [MapPolyline](mappolyline.md) is dashed.

## -property-value
**true** if the line used for drawing the [MapPolyline](mappolyline.md) is dashed; otherwise, **false**.
