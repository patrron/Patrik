# Windows.UI.Xaml.Controls.Maps.StreetsideExperience.StreetLabelsVisible

## -description
Gets or sets a value that indicates if street labels are visible in the [StreetsideExperience](streetsideexperience.md).

## -property-value
**true** if street labels are visible in the [StreetsideExperience](streetsideexperience.md); otherwise, **false**.
