# Windows.UI.Xaml.Controls.Maps.StreetsideExperience.ExitButtonVisible

## -description
Gets or sets a value that indicates if the exit button is visible in the [StreetsideExperience](streetsideexperience.md).

## -property-value
**true** if the exit button is visible in the [StreetsideExperience](streetsideexperience.md); otherwise, **false**.
