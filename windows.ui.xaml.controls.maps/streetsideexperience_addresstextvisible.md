# Windows.UI.Xaml.Controls.Maps.StreetsideExperience.AddressTextVisible

## -description
Gets or sets a value that indicates if address text is visible in the [StreetsideExperience](streetsideexperience.md).

## -property-value
**true** if address text is visible in the [StreetsideExperience](streetsideexperience.md); otherwise, **false**.
