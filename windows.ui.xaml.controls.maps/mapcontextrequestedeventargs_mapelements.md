# Windows.UI.Xaml.Controls.Maps.MapContextRequestedEventArgs.MapElements

Gets a collection of [MapElement](mapcontrol_mapelements.md) objects at the point on the map specified by the [Location](mapcontextrequestedeventargs_location.md) property.

## -property-value
A collection of [MapElement](mapcontrol_mapelements.md) objects at the point on the map specified by the [Location](mapcontextrequestedeventargs_location.md) property.
