# Windows.UI.Xaml.Controls.Maps.MapControl.TryZoomToAsync

## -description
Starts an asynchronous operation to zoom the map to a specific [ZoomLevel](mapcontrol_zoomlevel.md).

## -parameters
### -param zoomLevel
The final zoom level of the map, between 1 and 20.

## -returns
Contains the results of the operation: **true** if the operation completed successfully; otherwise, **false**.

## -remarks
See the remarks section of the [TryZoomInAsync](mapcontrol_tryzoominasync_1251712457.md) method.
