# Windows.UI.Xaml.Controls.Maps.MapTileSource.MapTileSource

## -description
Initializes a new instance of the [MapTileSource](maptilesource.md) class with the specified data source, zoom level range, bounding rectangle, and tile size.

## -parameters
### -param dataSource
The data source to use for the tiles.

### -param zoomLevelRange
The zoom level range for the tiles.

### -param bounds
The geographic area of tiles requested.

### -param tileSizeInPixels
The size of individual square tiles in pixels.
