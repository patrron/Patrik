# Windows.UI.Xaml.Controls.Maps.MapElement.IsEnabled

## -description
Indicates whether users can interact with the [MapElement](mapelement.md).

## -property-value
**true** if users can interact with the [MapElement](mapelement.md); otherwise, **false**.
