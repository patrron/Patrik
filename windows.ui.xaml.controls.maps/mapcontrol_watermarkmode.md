# Windows.UI.Xaml.Controls.Maps.MapControl.WatermarkMode

## -description
Gets or sets a value that determines when the watermark of the map is displayed.

## -xaml-syntax
```xaml
<!-- xmlns:Maps="using:Windows.UI.Xaml.Controls.Maps" -->

<Maps:MapControl WatermarkMode="watermarkMode"/>
- or -
<Maps:MapControl>
    <Maps:MapControl.WatermarkMode>watermarkMode</Maps:MapControl.WatermarkMode>
</Maps:MapControl>

```


## -xaml-values
<dl><dt>watermarkMode</dt><dd>watermarkModeA value from the MapWatermarkMode enumeration that determines when the watermark of the map is displayed.</dd>
</dl>
## -property-value
A value that determines when the watermark of the map is displayed.
