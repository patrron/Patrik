# Windows.UI.Xaml.Controls.Maps.CustomMapTileDataSource.BitmapRequested

## -description
Occurs when the bitmap is requested for a [CustomMapTileDataSource](custommaptiledatasource.md). An instance of [MapTileBitmapRequestedEventArgs](maptilebitmaprequestedeventargs.md) provides data for this event.
