# Windows.UI.Xaml.Controls.Maps.StreetsideExperience.CursorVisible

## -description
Gets or sets a value that indicates if the cursor is visible in the [StreetsideExperience](streetsideexperience.md).

## -property-value
**true** if the cursor is visible in the [StreetsideExperience](streetsideexperience.md); otherwise, **false**.
