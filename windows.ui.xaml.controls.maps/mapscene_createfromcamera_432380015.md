# Windows.UI.Xaml.Controls.Maps.MapScene.CreateFromCamera

## -description
Creates a scene to appear in a map based on the position of the specified [MapCamera](mapcamera.md).

## -parameters
### -param camera
The current position of the specified camera.

## -returns
Represents a view of a map in the [MapControl](mapcontrol.md).
