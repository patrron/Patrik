# Windows.UI.Xaml.Controls.Maps.MapTileSource.Visible

## -description
Gets or sets a value that indicates whether the [MapTileSource](maptilesource.md) is visible.

## -property-value
**true** if the [MapTileSource](maptilesource.md) is visible; otherwise, **false**.
