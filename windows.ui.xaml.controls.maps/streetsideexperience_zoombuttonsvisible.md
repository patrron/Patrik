# Windows.UI.Xaml.Controls.Maps.StreetsideExperience.ZoomButtonsVisible

## -description
Gets or sets a value that indicates if zoom buttons are visible in the [StreetsideExperience](streetsideexperience.md).

## -property-value
**true** if zoom buttons are visible in the [StreetsideExperience](streetsideexperience.md); otherwise, **false**.
