# Windows.UI.Xaml.Controls.Maps.StreetsideExperience.OverviewMapVisible

## -description
Gets or sets a value that indicates if the overview map is visible in the [StreetsideExperience](streetsideexperience.md).

## -property-value
**true** if the overview map is visible in the [StreetsideExperience](streetsideexperience.md); otherwise, **false**.
