# Windows.UI.Xaml.Controls.Maps.MapTileSource.IsRetryEnabled

## -description
Gets or sets a value that indicates whether retry is enabled for the tiles.

## -property-value
**true** if retry is enabled for the tiles; otherwise, **false**.
