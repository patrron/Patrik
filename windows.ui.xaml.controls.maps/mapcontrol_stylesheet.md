# Windows.UI.Xaml.Controls.Maps.MapControl.StyleSheet

## -description
Gets or sets an object that defines the style of the map control.

## -property-value
An object that defines the style of the map control.

## -remarks

## -see-also

## -examples
The following example sets the [StyleSheet](mapcontrol_stylesheet.md) property by using the static [MapControl.RoadDark](mapstylesheet_roaddark_376470101.md) method.

```csharp
  myMap.StyleSheet = MapStyleSheet.RoadDark();
```
