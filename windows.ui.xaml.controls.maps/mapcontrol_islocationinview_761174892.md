# Windows.UI.Xaml.Controls.Maps.MapControl.IsLocationInView

## -description
Determines whether the specified geographic location is located in the portion of the map currently visible in the [MapControl](mapcontrol.md).

## -parameters
### -param location
A geographic location.

### -param isInView
When this method returns, contains **true** if the specified geographic location is located in the portion of the map currently visible in the [MapControl](mapcontrol.md); otherwise, contains **false**.
