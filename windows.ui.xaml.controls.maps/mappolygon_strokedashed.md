# Windows.UI.Xaml.Controls.Maps.MapPolygon.StrokeDashed

## -description
Gets or sets a value that indicates whether the line used to draw the [MapPolygon](mappolygon.md) is dashed.

## -property-value
**true** if the line used for drawing the [MapPolygon](mappolygon.md) is dashed; otherwise, **false**.
