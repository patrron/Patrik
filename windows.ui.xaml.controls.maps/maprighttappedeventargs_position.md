# Windows.UI.Xaml.Controls.Maps.MapRightTappedEventArgs.Position

## -description
Gets the physical position on the [MapControl](mapcontrol.md) where it received user input.

## -property-value
The physical position on the [MapControl](mapcontrol.md) where it received user input, in terms of X and Y coordinates.
