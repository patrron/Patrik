# Windows.Devices.Bluetooth.BluetoothDevice.GetRfcommServicesForIdAsync

## -description
Retrieves all Rfcomm Services on the Remote Bluetooth Device matching the specified [RfcommServiceId](../windows.devices.bluetooth.rfcomm/rfcommserviceid.md).

## -parameters
### -param serviceId
The Rfcomm service id.

## -returns
After the asynchronous operation completes, returns an [RfcommDeviceServicesResult](../windows.devices.bluetooth.rfcomm/rfcommdeviceservicesresult.md) object.
