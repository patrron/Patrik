# Windows.Networking.Connectivity.NetworkAdapter.IanaInterfaceType

## -description
Gets a value indicating the network interface type as defined by the Internet Assigned Names Authority (IANA) for the [NetworkAdapter](networkadapter.md).

## -property-value
The interface type as defined by the Internet Assigned Names Authority (IANA).

## -remarks
This property represents the interface type as defined by the Internet Assigned Names Authority (IANA). For a list of possible values, see [http://www.iana.org/assignments/ianaiftype-mib](https://go.microsoft.com/fwlink/p/?linkid=84022).
