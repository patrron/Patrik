# Windows.Networking.Connectivity.ConnectivityManager

## -description
Methods defined by the ConnectivityManager class enable enforcement of traffic routing on a specific network adapter for the specified destination suffix. Once a policy is set using [AddHttpRoutePolicy](connectivitymanager_addhttproutepolicy_1422666154.md), traffic that matches the policy will either be routed or dropped.
