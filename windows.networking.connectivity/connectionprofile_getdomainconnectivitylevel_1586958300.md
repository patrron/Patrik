# Windows.Networking.Connectivity.ConnectionProfile.GetDomainConnectivityLevel

## -description
Gets the current domain authentication status for a network connection. Possible values are defined by [DomainConnectivityLevel](domainconnectivitylevel.md).

## -returns
A value indicating the authentication status for a connection to a network domain.
