# Windows.Networking.Connectivity.NetworkItem.NetworkId

## -description
Gets the network ID.

## -property-value
The network ID.

## -remarks
This value can be used to retrieve the unique ID of a network if [NetworkTypes](networktypes.md) is **Private**.
