# Windows.Networking.Connectivity.ConnectionSession.Close

## -description
Closes the connection to the access point.

## -remarks
This method is used by Windows Store app written in JavaScript. For apps written using the .NET FrameworkÂ 4.5 in C# and VB.NET, the [Close](connectionsession_close.md) method is exposed as the [Dispose](connectionsession_dispose.md) method on the [ConnectionSession](connectionsession.md). For apps written in C++, the this method is called when using the delete keyword on the object.
