# Windows.Networking.Connectivity.NetworkInformation.FindConnectionProfilesAsync

## -description
Returns an array of [ConnectionProfile](connectionprofile.md) objects that match the filtering criteria defined by [ConnectionProfileFilter](connectionprofilefilter.md).

## -parameters
### -param pProfileFilter
Provides the filtering criteria.

## -returns
An array of [ConnectionProfile](connectionprofile.md) objects.
