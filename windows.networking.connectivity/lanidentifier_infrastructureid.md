# Windows.Networking.Connectivity.LanIdentifier.InfrastructureId

## -description
Gets a [LanIdentifierData](lanidentifierdata.md) object containing locality identification information for the network adapter's connection.

## -property-value
Location information for the network adapter providing connectivity for the connection.
