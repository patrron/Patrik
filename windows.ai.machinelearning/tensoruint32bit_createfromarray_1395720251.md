# Windows.AI.MachineLearning.TensorUInt32Bit.CreateFromArray

## -description
Creates a 32-bit unsigned integer tensor object, allocates a buffer of size *shape*, and copies all of *data* into it.

## -parameters
### -param shape
The size of the buffer.

### -param data
The data to copy into the buffer.

## -returns
A 32-bit unsigned integer tensor object with a buffer containing the given data.
