# Windows.ApplicationModel.Wallet.WalletItemStore.ShowAsync

## -description
Displays the item with the given ID in the Wallet UI.

## -parameters
### -param id
The ID of the item to display.

## -returns
An asynchronous action. If you use [Asynchronous programming](http://msdn.microsoft.com/library/23fe28f1-89c5-4a17-a732-a722648f9c5e), the result type is **void**.
