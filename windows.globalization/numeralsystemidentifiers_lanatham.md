# Windows.Globalization.NumeralSystemIdentifiers.LanaTham

## -description
Gets the identifier for the Tai Tham Tham decimal numeral system.

## -property-value
The string "LanaTham", which identifies the Tai Tham Tham decimal numeral system. This system uses the following Unicode values:<table>
   <tr><th>Unicode code point</th><th>Corresponding Latin digit</th></tr>
   <tr><td>U+1A90</td><td>0</td></tr>
   <tr><td>U+1A91</td><td>1</td></tr>
   <tr><td>U+1A92</td><td>2</td></tr>
   <tr><td>U+1A93</td><td>3</td></tr>
   <tr><td>U+1A94</td><td>4</td></tr>
   <tr><td>U+1A95</td><td>5</td></tr>
   <tr><td>U+1A96</td><td>6</td></tr>
   <tr><td>U+1A97</td><td>7</td></tr>
   <tr><td>U+1A98</td><td>8</td></tr>
   <tr><td>U+1A99</td><td>9</td></tr>
</table>
