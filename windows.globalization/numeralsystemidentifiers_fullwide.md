# Windows.Globalization.NumeralSystemIdentifiers.FullWide

## -description
Gets the identifier for the Latin decimal numeral system using full-width characters.

## -property-value
The string "FullWide", which identifies the Latin decimal numeral system using full-width characters. This system has the digits ０１２３４５６７８９ which correspond to the Latin digits 0-9.
