# Windows.Globalization.CurrencyAmount.Currency

## -description

A currency in general use in a specific country.

## -property-value

A string containing a value from [CurrencyIdentifiers](currencyidentifiers.md) (such as [USD](currencyidentifiers_usd.md), or other **ISO 4217** alphabetic currency code), vendor-specific code prefixed with *"X-"* (for example, *"X-ContosoPoints"*), or another currency name (including cryptocurrency, such as *"Bitcoin"* or *"TRON"*).
