# Windows.Globalization.GeographicRegion.CodeTwoLetter

## -description
Gets the two-letter identifier for this region.

## -property-value
Returns the two-letter ISO 3166-1 alpha-2 code for this region, if one exists; otherwise, returns "ZZ".
