# Windows.Globalization.NumeralSystemIdentifiers.Sund

## -description
Gets the identifier for the Sundanese decimal numeral system.

## -property-value
The string "Sund", which identifies the Sundanese decimal numeral system. This system uses the following Unicode values:

<table>
   <tr><th>Unicode code point</th><th>Corresponding Latin digit</th></tr>
   <tr><td>U+1BB0</td><td>0</td></tr>
   <tr><td>U+1BB1</td><td>1</td></tr>
   <tr><td>U+1BB2</td><td>2</td></tr>
   <tr><td>U+1BB3</td><td>3</td></tr>
   <tr><td>U+1BB4</td><td>4</td></tr>
   <tr><td>U+1BB5</td><td>5</td></tr>
   <tr><td>U+1BB6</td><td>6</td></tr>
   <tr><td>U+1BB7</td><td>7</td></tr>
   <tr><td>U+1BB8</td><td>8</td></tr>
   <tr><td>U+1BB9</td><td>9</td></tr>
</table>
