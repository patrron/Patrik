# Windows.Globalization.CurrencyIdentifiers.BAM

## -description
Gets the currency identifier for the Bosnia and Herzegovina Convertible Mark.

## -property-value
The string "BAM", which identifies the Bosnia and Herzegovina Convertible Mark.
