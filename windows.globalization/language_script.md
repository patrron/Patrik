# Windows.Globalization.Language.Script

## -description
Gets the four-letter ISO 15924 script code of the language.

## -property-value
A string with the script code of the language. If the script code cannot be identified or inferred from the [language tag](language_languagetag.md), "Zyyy" is returned.
