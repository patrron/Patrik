# Windows.Globalization.NumeralSystemIdentifiers.Saur

## -description
Gets the identifier for the Saurashtra decimal numeral system.

## -property-value
The string "Saur", which identifies the Saurashtra decimal numeral system. This system uses the following Unicode values:<table>
   <tr><th>Unicode code point</th><th>Corresponding Latin digit</th></tr>
   <tr><td>U+A8D0</td><td>0</td></tr>
   <tr><td>U+A8D1</td><td>1</td></tr>
   <tr><td>U+A8D2</td><td>2</td></tr>
   <tr><td>U+A8D3</td><td>3</td></tr>
   <tr><td>U+A8D4</td><td>4</td></tr>
   <tr><td>U+A8D5</td><td>5</td></tr>
   <tr><td>U+A8D6</td><td>6</td></tr>
   <tr><td>U+A8D7</td><td>7</td></tr>
   <tr><td>U+A8D8</td><td>8</td></tr>
   <tr><td>U+A8D9</td><td>9</td></tr>
</table>
