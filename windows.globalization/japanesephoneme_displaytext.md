# Windows.Globalization.JapanesePhoneme.DisplayText

## -description
Gets the text to display for this Japanese word.

## -property-value
The display text of this Japanese word. It is a sub-string of the Japanese string originally analyzed by [JapanesePhoneticAnalyzer](japanesephoneticanalyzer.md).
