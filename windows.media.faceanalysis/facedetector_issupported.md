# Windows.Media.FaceAnalysis.FaceDetector.IsSupported

## -description
Gets a value indicating whether the [FaceDetector](facedetector.md) class is supported on the current device.

## -property-value
True if [FaceDetector](facedetector.md); otherwise, false.
