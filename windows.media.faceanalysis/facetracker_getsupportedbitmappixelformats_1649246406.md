# Windows.Media.FaceAnalysis.FaceTracker.GetSupportedBitmapPixelFormats

## -description
Returns a list of the bitmap pixels formats supported by the [FaceTracker](facetracker.md) on the current device.

## -returns
A list of the bitmap pixels formats supported by the [FaceTracker](facetracker.md) on the current device.
