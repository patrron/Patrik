# Windows.Media.FaceAnalysis.FaceDetector.DetectFacesAsync

## -description
Asynchronously detects faces in the provided [SoftwareBitmap](../windows.graphics.imaging/softwarebitmap.md) within the specified search area.

## -parameters
### -param image
The image data to be processed for face detection.

### -param searchArea
The bounds within the [SoftwareBitmap](../windows.graphics.imaging/softwarebitmap.md) in which face detection will be performed.

## -returns
An asynchronous operation that returns a list of [DetectedFace](detectedface.md) objects upon successful completion.
