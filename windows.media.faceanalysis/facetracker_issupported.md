# Windows.Media.FaceAnalysis.FaceTracker.IsSupported

## -description
Gets a value indicating whether the [FaceTracker](facetracker.md) class is supported on the current device.

## -property-value
True if [FaceTracker](facetracker.md); otherwise, false.
