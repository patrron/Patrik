# VoipCallState

## -description
Specifies the state of a VoIP call.

## -enum-fields
### -field Ended:0
The call has ended.

### -field Held:1
The call is on hold.

### -field Active:2
The call is active.

### -field Incoming:3
The call is incoming but not yet active.

### -field Outgoing:4
The call is outgoing but not yet active.
