# Windows.UI.ViewManagement.StatusBarProgressIndicator

## -description
Provides methods and properties for interacting with the progress indicator on the status bar on a window (app view).

## -remarks
Call the [StatusBar.ProgressIndicator](statusbar_progressindicator.md) method to obtain an instance of this class.
