# Windows.UI.ViewManagement.ApplicationView.SetDesiredBoundsMode

## -description
Sets a value indicating the bounds used by the framework to lay out the contents of the window (app view).

## -parameters
### -param boundsMode
The bounds that the framework should use to lay out the contents of the window (app view).

## -returns
**true** to confirm that the change to [DesiredBoundsMode](applicationview_desiredboundsmode.md) was successful; otherwise, **false**.
