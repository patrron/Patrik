# Windows.UI.ViewManagement.InputPane.GetForUIContext

## -description

Gets the [InputPane](inputpane.md) object associated with the view with the specified context identifier.

## -parameters
### -param context

The context identifier of the view to get the input pane for.

## -returns

The input pane.

## -remarks

This method fails if it is called in a windowless thread.
