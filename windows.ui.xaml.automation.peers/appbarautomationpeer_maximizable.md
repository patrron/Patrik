# Windows.UI.Xaml.Automation.Peers.AppBarAutomationPeer.Maximizable

## -description
Gets a Boolean value that indicates whether the app bar can be maximized.

Equivalent WinUI property: [Microsoft.UI.Xaml.Automation.Peers.AppBarAutomationPeer.Maximizable](/windows/winui/api/microsoft.ui.xaml.automation.peers.appbarautomationpeer.maximizable).

## -property-value
**true** if the app bar can be maximized; otherwise, **false**.
