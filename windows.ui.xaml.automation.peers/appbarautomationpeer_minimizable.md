# Windows.UI.Xaml.Automation.Peers.AppBarAutomationPeer.Minimizable

## -description
Gets a Boolean value that indicates whether the app bar can be minimized.

Equivalent WinUI property: [Microsoft.UI.Xaml.Automation.Peers.AppBarAutomationPeer.Minimizable](/windows/winui/api/microsoft.ui.xaml.automation.peers.appbarautomationpeer.minimizable).

## -property-value
**true** if the app bar can be minimized; otherwise, **false**.
