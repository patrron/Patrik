# Windows.UI.Xaml.Automation.Peers.AutomationPeer.IsKeyboardFocusable

## -description
Gets a value that indicates whether the element can accept keyboard focus.

Equivalent WinUI method: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.IsKeyboardFocusable](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.iskeyboardfocusable).

## -returns
**true** if the element can accept keyboard focus; otherwise, **false**.
