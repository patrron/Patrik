# Windows.UI.Xaml.Automation.Peers.AutomationPeer.GetCultureCore

## -description

Equivalent WinUI method: [Microsoft.UI.Xaml.Automation.Peers.AutomationPeer.GetCultureCore](/windows/winui/api/microsoft.ui.xaml.automation.peers.automationpeer.getculturecore).

## -returns
The value of the culture property for the element that is associated with the automation peer.
