# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.IncludeHour

## -description
Gets the [HourFormat](hourformat.md) in the template.

## -property-value
Specifies whether the hour is included. If it is, this property specifies the format for the hour in the template.
