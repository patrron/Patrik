# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.LongTime

## -description
Gets the [DateTimeFormatter](datetimeformatter.md) object that formats times according to the user's choice of long time pattern.

## -property-value
A [DateTimeFormatter](datetimeformatter.md) object equivalent to one constructed with the "longtime" template.
