# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.IncludeYear

## -description
Gets the [YearFormat](yearformat.md) in the template.

## -property-value
Specifies whether the year is included. If it is, this format specifies the format for the year in the template.
