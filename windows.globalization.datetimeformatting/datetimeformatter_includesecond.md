# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.IncludeSecond

## -description
Gets the [SecondFormat](secondformat.md) in the template.

## -property-value
Specifies whether the second is included. If it is, this property specifies the format for the second in the template.
