# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.LongDate

## -description
Gets the [DateTimeFormatter](datetimeformatter.md) object that formats dates according to the user's choice of long date pattern.

## -property-value
A [DateTimeFormatter](datetimeformatter.md) object equivalent to one constructed with the "longdate" template.
