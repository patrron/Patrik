# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.IncludeDayOfWeek

## -description
Gets the [DayOfWeekFormat](dayofweekformat.md) in the template.

## -property-value
Specifies whether the day of week is included. If it is, this property specifies the format for the day of week in the template.
