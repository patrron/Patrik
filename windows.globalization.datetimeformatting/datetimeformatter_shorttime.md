# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.ShortTime

## -description
Gets the [DateTimeFormatter](datetimeformatter.md) object that formats times according to the user's choice of short time pattern.

## -property-value
A [DateTimeFormatter](datetimeformatter.md) object equivalent to one constructed with the "shorttime" template.
