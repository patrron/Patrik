# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.IncludeMonth

## -description
Gets the [MonthFormat](monthformat.md) in the template.

## -property-value
Specifies whether the month is included. If it is, this property specifies the format for the month in the template.
