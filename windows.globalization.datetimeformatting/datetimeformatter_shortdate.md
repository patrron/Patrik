# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.ShortDate

## -description
Gets the [DateTimeFormatter](datetimeformatter.md) object that formats dates according to the user's choice of short date pattern.

## -property-value
A [DateTimeFormatter](datetimeformatter.md) object equivalent to one constructed with the "shortdate" template.
