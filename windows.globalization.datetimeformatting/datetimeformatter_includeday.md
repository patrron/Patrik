# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.IncludeDay

## -description
Gets the [DayFormat](dayformat.md) in the template.

## -property-value
Specifies whether the day is included. If it is, this property specifies the format for the day in the template.
