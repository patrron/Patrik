# Windows.Globalization.DateTimeFormatting.DateTimeFormatter.IncludeMinute

## -description
Gets the [MinuteFormat](minuteformat.md) in the template.

## -property-value
Specifies whether the minute is included. If it is, this property specifies the format for the minute in the template.
