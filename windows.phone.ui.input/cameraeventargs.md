# Windows.Phone.UI.Input.CameraEventArgs

## -description
Provides data for the [CameraHalfPressed](hardwarebuttons_camerahalfpressed.md), [CameraPressed](hardwarebuttons_camerapressed.md), and [CameraReleased](hardwarebuttons_camerareleased.md) events.

## -remarks
Handle the [HardwareButtons.CameraHalfPressed](hardwarebuttons_camerahalfpressed.md), the [HardwareButtons.CameraPressed](hardwarebuttons_camerapressed.md), or the [HardwareButtons.CameraReleased](hardwarebuttons_camerareleased.md) event to obtain an instance of this class.
