# Windows.Security.Cryptography.Certificates.Certificate.Certificate

## -description
Create a new instance of the [Certificate](certificate.md) class using the specified certificate data.

## -parameters
### -param certBlob
The certificate data as an ASN.1 DER encoded certificate blob (.cer or .p7b).
