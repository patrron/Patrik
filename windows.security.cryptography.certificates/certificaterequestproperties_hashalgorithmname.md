# Windows.Security.Cryptography.Certificates.CertificateRequestProperties.HashAlgorithmName

## -description
Gets or sets the hash algorithm used when creating the certificate request signature.

## -property-value
Hash algorithm.

## -remarks
The default value is SHA256.
