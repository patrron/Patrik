# Windows.System.Diagnostics.TraceReporting.PlatformDiagnosticTracePriority

## -description
Describes the priority of a trace.

## -enum-fields

## -field UserElevated:1
The trace priority has been elevated by user action.

## -field Normal:0
The trace is being collected at normal priority.

## -remarks
> [!IMPORTANT]
> Only first-party applications should use this type.
