# Windows.Storage.BulkAccess.FolderInformation.CreateFolderQuery

## -description
Creates an object for performing filtered search queries for child [StorageFolder](../windows.storage/storagefolder.md) objects of the current folder.

## -returns
An object for managing the search queries and accessing the results.
