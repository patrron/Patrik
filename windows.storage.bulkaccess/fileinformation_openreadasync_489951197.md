# Windows.Storage.BulkAccess.FileInformation.OpenReadAsync

## -description
Opens a read-only, random-access stream over the [StorageFile](../windows.storage/storagefile.md).

## -returns
When this method completes successfully, it returns a read-only, random-access stream (type [IRandomAccessStreamWithContentType](../windows.storage.streams/irandomaccessstreamwithcontenttype.md)).
