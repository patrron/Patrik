# Windows.Storage.BulkAccess.FolderInformation.GetFolderAsync

## -description
Retrieves the specified child folder from the current folder.

## -parameters
### -param name
The name of the child folder to retrieve.

## -returns
When this method completes successfully, it returns a [StorageFolder](../windows.storage/storagefolder.md) that represents the specified child folder.
