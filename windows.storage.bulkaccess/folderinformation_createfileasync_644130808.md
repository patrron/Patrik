# Windows.Storage.BulkAccess.FolderInformation.CreateFileAsync

## -description
Creates a new file in the current folder.

## -parameters
### -param desiredName
The name of the new file.

## -returns
When this method completes successfully, it returns a [StorageFile](../windows.storage/storagefile.md) that represents the new file.
