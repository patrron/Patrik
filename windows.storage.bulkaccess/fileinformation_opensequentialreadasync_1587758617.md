# Windows.Storage.BulkAccess.FileInformation.OpenSequentialReadAsync

## -description
Opens a read-only, sequential-access stream over the [StorageFile](../windows.storage/storagefile.md).

## -returns
When this method completes successfully, it returns a read-only, sequential-access stream (type [IInputStream](../windows.storage.streams/iinputstream.md)).
