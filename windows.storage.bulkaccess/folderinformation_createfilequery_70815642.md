# Windows.Storage.BulkAccess.FolderInformation.CreateFileQuery

## -description
Creates an object for performing filtered search queries for [StorageFile](../windows.storage/storagefile.md) in the current folder.

## -returns
An object for managing the search queries and accessing the results.
