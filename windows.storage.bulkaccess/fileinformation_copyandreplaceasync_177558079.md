# Windows.Storage.BulkAccess.FileInformation.CopyAndReplaceAsync

## -description
Replaces the specified [StorageFile](../windows.storage/storagefile.md) with a copy of the current file.

## -parameters
### -param fileToReplace
The [StorageFile](../windows.storage/storagefile.md) to be replaced.

## -returns
No object or value is returned when this method completes.
