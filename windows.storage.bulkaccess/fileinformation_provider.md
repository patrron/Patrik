# Windows.Storage.BulkAccess.FileInformation.Provider

## -description
Gets the [StorageProvider](../windows.storage/storageprovider.md) object that contains info about the service that stores the current file.

## -property-value
The [StorageProvider](../windows.storage/storageprovider.md) object that contains info about the service that stores the current file. The file may be stored by the local file system or by a remote service like Microsoft OneDrive.
