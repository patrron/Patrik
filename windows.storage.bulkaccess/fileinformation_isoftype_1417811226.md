# Windows.Storage.BulkAccess.FileInformation.IsOfType

## -description
Determines whether the loaded [StorageFile](../windows.storage/storagefile.md) is the specified type.

## -parameters
### -param type
The type of item to check against.

## -returns
True if the loaded [StorageFile](../windows.storage/storagefile.md) is the specified type; otherwise false.
