# Windows.Storage.BulkAccess.FolderInformation.GetItemAsync

## -description
Retrieves an item from the folder.

## -parameters
### -param name
The name of the item to retrieve.

## -returns
When this method completes successfully, it returns the item (type [IStorageItem](../windows.storage/istorageitem.md)).
