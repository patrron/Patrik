# Windows.Storage.BulkAccess.FolderInformation.CreateFileQuery

## -description
Creates an object for performing filtered search queries for [StorageFile](../windows.storage/storagefile.md) objects in the current folder. Search results are grouped based on the specified file attribute.

## -parameters
### -param query
A value that specifies the folder attribute to use to group the search results.

## -returns
An object for managing the search queries and accessing the results.
