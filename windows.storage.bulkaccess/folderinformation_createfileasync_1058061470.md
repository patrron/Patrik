# Windows.Storage.BulkAccess.FolderInformation.CreateFileAsync

## -description
Creates a new file in the current folder, and specifies what to do if a file with the same name already exists in the current folder.

## -parameters
### -param desiredName
The name of the new file.

### -param options
A value that indicates what to do if the file name already exists in the current folder.

## -returns
When this method completes successfully, it returns a [StorageFile](../windows.storage/storagefile.md) that represents the new file.
