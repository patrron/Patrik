# Windows.Storage.BulkAccess.FolderInformation.RenameAsync

## -description
Renames the [StorageFolder](../windows.storage/storagefolder.md), and specifies what to do if a folder with the same name already exists.

## -parameters
### -param desiredName
The new name.

### -param option
A value that indicates what to do if a folder with *desiredName* already exists.

## -returns
No object or value is returned when this method completes.
