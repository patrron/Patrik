# Windows.Storage.BulkAccess.FolderInformation.CreateFolderAsync

## -description
Creates a new child folder of the current folder.

## -parameters
### -param desiredName
The name of the new folder.

## -returns
When this method completes successfully, it returns a [StorageFolder](../windows.storage/storagefolder.md) that represents the new file.
