# Windows.Storage.BulkAccess.FolderInformation.GetBasicPropertiesAsync

## -description
Retrieves the basic properties of the [StorageFolder](../windows.storage/storagefolder.md) object.

## -returns
When this method completes successfully, it returns a [BasicProperties](../windows.storage.fileproperties/basicproperties.md) object.
