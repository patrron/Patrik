# Windows.Storage.BulkAccess.FileInformation.MoveAndReplaceAsync

## -description
Moves the current [StorageFile](../windows.storage/storagefile.md) from its location and uses it to replace the specified [StorageFile](../windows.storage/storagefile.md).

## -parameters
### -param fileToReplace
The [StorageFile](../windows.storage/storagefile.md) to be replaced.

## -returns
An object for managing the asynchronous move and replace operation.
