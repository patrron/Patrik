# Windows.Storage.BulkAccess.FileInformation.GetBasicPropertiesAsync

## -description
Retrieves the basic properties of the [StorageFile](../windows.storage/storagefile.md).

## -returns
When this method completes successfully, it returns a [BasicProperties](../windows.storage.fileproperties/basicproperties.md) object.
