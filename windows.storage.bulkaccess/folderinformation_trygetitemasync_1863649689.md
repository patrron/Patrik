# Windows.Storage.BulkAccess.FolderInformation.TryGetItemAsync

## -description
Try to get a specific file or sub-folder from the current folder using the name of the file or folder to get.

## -parameters
### -param name
The name (or path relative to the current folder) of the file or sub-folder to try to retrieve.

## -returns
When this method completes successfully, it returns the file or folder (type [IStorageItem](../windows.storage/istorageitem.md)).
