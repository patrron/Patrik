# Windows.Storage.BulkAccess.FolderInformation.IsEqual

## -description
Indicates whether the current folder is equal to the specified folder.

## -parameters
### -param item
The [IStorageItem](../windows.storage/istorageitem.md) object that represents the folder to compare against.

## -returns
Returns **true** if the current folder is equal to the specified folder; otherwise **false**.
