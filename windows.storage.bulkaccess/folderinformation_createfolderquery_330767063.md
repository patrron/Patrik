# Windows.Storage.BulkAccess.FolderInformation.CreateFolderQuery

## -description
Creates an object for performing filtered search queries for child [StorageFolder](../windows.storage/storagefolder.md) objects of the current folder. Search results are grouped based on the specified folder attribute.

## -parameters
### -param query
A value that specifies the folder attribute to use to group the search results.

## -returns
An object for managing the search queries and accessing the results.
