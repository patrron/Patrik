# Windows.Storage.BulkAccess.FolderInformation.Provider

## -description
Gets the [StorageProvider](../windows.storage/storageprovider.md) object that contains info about the service that stores the current folder.

## -property-value
The [StorageProvider](../windows.storage/storageprovider.md) object that contains info about the service that stores the current folder. The file may be stored by the local file system or by a remote service like Microsoft OneDrive.
