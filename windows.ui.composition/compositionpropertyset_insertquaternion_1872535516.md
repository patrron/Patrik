# Windows.UI.Composition.CompositionPropertySet.InsertQuaternion

## -description
Inserts a quaternion key-value pair.

Equivalent WinUI method: [Microsoft.UI.Composition.CompositionPropertySet.InsertQuaternion](/windows/winui/api/microsoft.ui.composition.compositionpropertyset.insertquaternion).

## -parameters
### -param propertyName
The key associated with the value. This key can be used to retrieve the value.

### -param value
The value to insert.
