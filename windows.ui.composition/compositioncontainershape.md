# Windows.UI.Composition.CompositionContainerShape

## -description

Represents a container for CompositionShapes, used to group items that share 2D transforms.

Equivalent WinUI class: [Microsoft.UI.Composition.CompositionContainerShape](/windows/winui/api/microsoft.ui.composition.compositioncontainershape).
