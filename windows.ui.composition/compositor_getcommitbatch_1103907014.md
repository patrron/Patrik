# Windows.UI.Composition.Compositor.GetCommitBatch

## -description
Retrieves a commit batch for the current commit cycle.

Equivalent WinUI method: [Microsoft.UI.Composition.Compositor.GetCommitBatch](/windows/winui/api/microsoft.ui.composition.compositor.getcommitbatch).

## -parameters
### -param batchType
The batch type.

## -returns
Returns a [CompositionCommitBatch](compositioncommitbatch.md) representing the retrieved commit batch.
