# FrameFlashMode

## -description
Specifies the flash mode for a frame in a variable photo sequence.

## -enum-fields
### -field Disable:0
The flash is disabled.

### -field Enable:1
The flash is enabled.

### -field Global:2
The flash uses the global flash mode.
