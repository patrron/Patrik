# Windows.UI.Input.Inking.IInkStrokeContainer.GetStrokes

## -description
Retrieves all ink strokes in the collection managed by the [InkStrokeContainer](inkstrokecontainer.md).

## -returns
The ink strokes managed by the [InkStrokeContainer](inkstrokecontainer.md) as a collection of [InkStroke](inkstroke.md) objects.
