# Windows.UI.Input.Inking.InkInputConfiguration.IsEraserInputEnabled

## -description
Gets or sets whether input from a pen's eraser tip is processed by the [InkPresenter](inkpresenter.md) object.

## -property-value
True, if input from the eraser tip is processed; otherwise, false.
