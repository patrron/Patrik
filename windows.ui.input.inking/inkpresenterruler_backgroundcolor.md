# Windows.UI.Input.Inking.InkPresenterRuler.BackgroundColor

## -description
Gets or sets the background color of the [InkPresenterRuler](inkpresenterruler.md).

## -property-value
The background color of the ruler as an ARGB value. The default is (255, 230, 230, 230).
