# Windows.UI.Input.Inking.IInkPresenterStencil.BackgroundColor

## -description
Gets or sets the background color of the [IInkPresenterStencil](iinkpresenterstencil.md) object.

## -property-value
The background color of the stencil as an ARGB value. The default is (255, 230, 230, 230).
