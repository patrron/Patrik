# Windows.UI.Input.Inking.InkPresenterRuler.ForegroundColor

## -description
Gets or sets the color of the foreground elements on an [InkPresenterRuler](inkpresenterruler.md).

## -property-value
The foreground color of the ruler as an ARGB value. The default is (255, 0, 0, 0).
