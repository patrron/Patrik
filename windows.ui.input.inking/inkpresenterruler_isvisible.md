# Windows.UI.Input.Inking.InkPresenterRuler.IsVisible

## -description
Gets or sets whether the [InkPresenterRuler](inkpresenterruler.md) is visible.

## -property-value
**true** if the ruler is visible. Otherwise, **false**.
