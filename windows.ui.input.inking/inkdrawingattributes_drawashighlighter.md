# Windows.UI.Input.Inking.InkDrawingAttributes.DrawAsHighlighter

## -description
Gets or sets a value that indicates whether the [InkStroke](inkstroke.md) is rendered as a highlighter overlay. The stroke is typically set to a bright color, such as yellow.

## -property-value
**true** to render the stroke as a highlighter; otherwise **false**. The default value is **false**.
