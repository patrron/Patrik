# Windows.UI.Input.Inking.InkDrawingAttributes.PencilProperties

## -description
Gets an instance of an [InkDrawingAttributesPencilProperties](inkdrawingattributespencilproperties.md) object for setting properties characteristic to a pencil stroke.

## -property-value
An [InkDrawingAttributesPencilProperties](inkdrawingattributespencilproperties.md) used to set pencil stroke properties.
