# Windows.UI.Input.Inking.IInkPresenterStencil.IsVisible

## -description
Gets or sets whether the [IInkPresenterStencil](iinkpresenterstencil.md) object is visible.

## -property-value
**true** if the stencil is visible. Otherwise, **false**.
