# Windows.UI.Input.Inking.IInkPresenterRulerFactory.Create

## -description
Generates an [InkPresenterRuler](inkpresenterruler.md) object that includes information used in the construction of an [InkPresenter](inkpresenter.md).

## -parameters
### -param inkPresenter
Provides properties, methods, and events for managing the input, processing, and rendering of ink input (standard and modified) for an [InkCanvas](../windows.ui.xaml.controls/inkcanvas.md) control.

## -returns
Represents a visual stencil, displayed as an semi-transparent overlay on an [InkCanvas](../windows.ui.xaml.controls/inkcanvas.md).
