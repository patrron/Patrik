# Windows.UI.Input.Inking.IInkPresenterStencil.ForegroundColor

## -description
Gets or sets the color of the foreground elements on an [IInkPresenterStencil](iinkpresenterstencil.md) object.

## -property-value
The foreground color of the stencil as an ARGB value. The default is (255, 0, 0, 0).
