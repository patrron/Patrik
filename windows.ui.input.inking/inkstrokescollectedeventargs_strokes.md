# Windows.UI.Input.Inking.InkStrokesCollectedEventArgs.Strokes

## -description
Gets the collection of processed ink strokes ("wet" to "dry") associated with the [StrokesCollected](inkpresenter_strokescollected.md) event.

## -property-value
The processed ink strokes as a collection of [InkStroke](inkstroke.md) objects.
