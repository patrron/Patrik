# Windows.UI.Input.Inking.InkPresenterProtractor.BackgroundColor

## -description
Gets or sets the background color of the [InkPresenterProtractor](inkpresenterprotractor.md).

## -property-value
The background color of the protractor as an ARGB value. The default is (255, 230, 230, 230).
