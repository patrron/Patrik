# Windows.UI.Input.Inking.InkPresenterProtractor.AccentColor

## -description
Gets or sets the color of the rays and horizontal marks for the protractor stencil. 

![Protractor stencil](images\stencil-protractor.png)

Rays represent the angle of the drawn arc. They are visible only when ink is snapped to the protractor and the protractor has not been moved. 

Horizontal marks represent a horizontal line bisecting the protractor. The default positions are 3 and 9 oâ€™clock, but a rotation transform can be applied to the protractor, which also rotates the marks. 

## -property-value
The specified color.

The default is system-defined.
