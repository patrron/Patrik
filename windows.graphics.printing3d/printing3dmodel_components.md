# Windows.Graphics.Printing3D.Printing3DModel.Components

## -description
Gets all 3D Manufacturing Format (3MF) components used in the 3D model.

## -property-value
All 3MF components used in the 3D model, of type [Printing3DComponent](printing3dcomponent.md).
