# Windows.Graphics.Printing3D.Printing3D3MFPackage.LoadAsync

## -description
Creates a [Printing3D3MFPackage](printing3d3mfpackage.md) object from a 3D Manufacturing Format (3MF) file stream.

## -parameters
### -param value
A 3MF file stream.

## -returns
A [Printing3D3MFPackage](printing3d3mfpackage.md) created from the specified 3MF package stream.
