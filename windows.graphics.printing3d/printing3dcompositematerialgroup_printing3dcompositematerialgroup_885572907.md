# Windows.Graphics.Printing3D.Printing3DCompositeMaterialGroup.Printing3DCompositeMaterialGroup

## -description
Creates an instance of the [Printing3DCompositeMaterialGroup](printing3dcompositematerialgroup.md) class.

## -parameters
### -param MaterialGroupId
The identifier (ID) of the composite material group; a value greater than zero.

## -remarks
Set *MaterialGroupId* to a value greater than zero. A value of zero is not valid and triggers an **E_INVALIADARG** error.
