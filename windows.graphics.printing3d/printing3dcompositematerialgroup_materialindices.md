# Windows.Graphics.Printing3D.Printing3DCompositeMaterialGroup.MaterialIndices

## -description
Gets an ordered list of base materials that are used to make the composite materials in the group. The order of base materials is maintained in the mixture values defined by [Composites](printing3dcompositematerialgroup_composites.md).

## -property-value
Index values of base materials from the base material group specified by [MaterialGroupId](printing3dcompositematerialgroup_materialgroupid.md).
