# Windows.Graphics.Printing3D.Printing3D3MFPackage.LoadModelFromPackageAsync

## -description
Creates a [Printing3DModel](printing3dmodel.md) object from a 3D Manufacturing Format (3MF) file stream.

## -parameters
### -param value
A 3MF file stream.

## -returns
A [Printing3DModel](printing3dmodel.md) object created from the specified 3MF object stream.
