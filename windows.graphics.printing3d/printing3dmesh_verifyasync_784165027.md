# Windows.Graphics.Printing3D.Printing3DMesh.VerifyAsync

## -description
Verifies the mesh has manifold edges and normal triangles.

## -parameters
### -param value
Specifies how the mesh is verified.

## -returns
The results of the verification, of type [Printing3DMeshVerificationResult](printing3dmeshverificationresult.md).
