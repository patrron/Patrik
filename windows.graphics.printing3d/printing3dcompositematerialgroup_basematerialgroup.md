# Windows.Graphics.Printing3D.Printing3DCompositeMaterialGroup.BaseMaterialGroup

## -description
Gets the existing [Printing3DBaseMaterialGroup](printing3dbasematerialgroup.md) whose [Printing3DBaseMaterial](printing3dbasematerial.md) members will be used to produce composite materials.

## -property-value
The [Printing3DBaseMaterialGroup](printing3dbasematerialgroup.md) from which to produce composite materials.
