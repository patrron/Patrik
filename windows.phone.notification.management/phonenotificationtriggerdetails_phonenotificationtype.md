# Windows.Phone.Notification.Management.PhoneNotificationTriggerDetails.PhoneNotificationType

## -description
A value from the [PhoneNotificationType](phonenotificationtype.md) enumeration that indicates the type of notification received.

## -property-value
For a list of values, see [PhoneNotificationType](phonenotificationtype.md).

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.
