# Windows.Phone.Notification.Management.CalendarChangedNotificationTriggerDetails.TimeCreated

## -description
Gets the date and time at which the notification was created.

## -property-value
The date and time at which the notification was created.

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.
