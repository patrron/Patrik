# Windows.Phone.Notification.Management.MediaControlsTriggerDetails.MediaMetadata

## -description
A [MediaMetadata](mediametadata.md) object that contains additional information about the media.

## -property-value
A [MediaMetadata](mediametadata.md) object that contains additional information about the media.

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.
