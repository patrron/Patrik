# Windows.Phone.Notification.Management.CortanaTileNotificationTriggerDetails.AccessoryNotificationType

## -description
An [AccessoryNotificationType](accessorynotificationtype.md) that specifies the type of notification received. For Cortana Tile notifications, this value is AccessoryNotificationType.CortanaTile.

## -property-value
AccessoryNotificationType.CortanaTile

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.
