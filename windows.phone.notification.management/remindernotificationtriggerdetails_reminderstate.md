# Windows.Phone.Notification.Management.ReminderNotificationTriggerDetails.ReminderState

## -description
A [ReminderState](reminderstate.md) value that represents the state of the reminder.

## -property-value
For possible values, see [ReminderState](reminderstate.md).

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.
