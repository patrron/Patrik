# Windows.Phone.Notification.Management.EmailNotificationTriggerDetails.AccessoryNotificationType

## -description
An [AccessoryNotificationType](accessorynotificationtype.md) that specifies the type of notification received. For email notifications, this value is AccessoryNotificationType.Email.

## -property-value
AccessoryNotificationType.Email

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.
