# Windows.Phone.Notification.Management.MediaControlsTriggerDetails.PlaybackStatus

## -description
The current playback status of the corresponding media.

## -property-value
For a list of possible values, see [PlaybackStatus](playbackstatus.md).

## -remarks
Calling this API requires that the **ID_CAP_SMS** and **ID_CAP_SMS_COMPANION** capabilities be specified in the application manifest.
