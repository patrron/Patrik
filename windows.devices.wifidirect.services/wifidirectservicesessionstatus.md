# WiFiDirectServiceSessionStatus

## -description
Values used to describe the status of a Wi-Fi Direct Service Session.

## -enum-fields
### -field Closed:0
The session has been closed.

### -field Initiated:1
The session has been initiated.

### -field Requested:2
A session has been requested.

### -field Open:3
The session is open.
