# Windows.Media.Capture.Frames.MediaFrameReader.StartAsync

## -description
Asynchronously starts the reading of frames from a [MediaFrameSource](mediaframesource.md).

## -returns
An asynchronous operation that returns a [MediaFrameReaderStartStatus](mediaframereaderstartstatus.md) on completion.
