# Windows.Media.Capture.Frames.MediaFrameSourceInfo.DeviceInformation

## -description
Gets a [DeviceInformation](../windows.devices.enumeration/deviceinformation.md) object that can be used to access properties of a [MediaFrameSource](mediaframesource.md) device.

## -property-value
A [DeviceInformation](../windows.devices.enumeration/deviceinformation.md) object representing a media frame source.
