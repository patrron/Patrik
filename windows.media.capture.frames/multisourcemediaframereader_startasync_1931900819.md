# Windows.Media.Capture.Frames.MultiSourceMediaFrameReader.StartAsync

## -description
Asynchronously starts the reading of time-corellated frames from one or more [MediaFrameSource](mediaframesource.md) objects.

## -returns
An asynchronous operation that returns a [MultiSourceMediaFrameReaderStartStatus](multisourcemediaframereaderstartstatus.md) on completion.
