# Windows.Media.Capture.Frames.MediaFrameSourceInfo.MediaStreamType

## -description
Gets a value indicating the stream type of a [MediaFrameSource](mediaframesource.md), such as photo, video, or audio.

## -property-value
A value indicating the stream type of a media frame source.
