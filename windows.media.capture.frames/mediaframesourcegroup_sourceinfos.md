# Windows.Media.Capture.Frames.MediaFrameSourceGroup.SourceInfos

## -description
Gets a list of [MediaFrameSourceInfo](mediaframesourceinfo.md) objects that describe each media frame source in the group.

## -property-value
An asynchronous action that returns a list of [MediaFrameSourceInfo](mediaframesourceinfo.md) upon successful completion.
