# Windows.Media.Capture.Frames.VideoMediaFrame.CameraIntrinsics

## -description
Gets an object that describes the distortion model of the camera that generated the [VideoMediaFrame](videomediaframe.md).

## -property-value
An object that describes the distortion model of the camera that generated the [VideoMediaFrame](videomediaframe.md).
