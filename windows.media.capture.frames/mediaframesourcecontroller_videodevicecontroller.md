# Windows.Media.Capture.Frames.MediaFrameSourceController.VideoDeviceController

## -description
Gets a [VideoDeviceController](../windows.media.devices/videodevicecontroller.md) for a [MediaFrameSource](mediaframesource.md), which provides camera controls such as exposure, focus, and zoom.

## -property-value
A [VideoDeviceController](../windows.media.devices/videodevicecontroller.md) for the media frame source.
