# Windows.Media.Capture.Frames.MediaFrameSourceInfo.SourceKind

## -description
Gets a value indicating the kind of media frames generated by a [MediaFrameSource](mediaframesource.md), such as color, depth, or infrared.

## -property-value
A value indicating the kind of media frames generated by a media frame source.
