# Windows.Media.Capture.Frames.MediaFrameSourceInfo.CoordinateSystem

## -description
Gets an object representing the spatial coordinate system in which the data from a [MediaFrameSource](mediaframesource.md) is expressed.

## -property-value
An object representing the spatial coordinate system of the media frame source.
