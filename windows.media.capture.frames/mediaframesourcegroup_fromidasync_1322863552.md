# Windows.Media.Capture.Frames.MediaFrameSourceGroup.FromIdAsync

## -description
Asynchronously gets the media frame source group with the specified ID.

## -parameters
### -param id
The unique identifier of a media frame source group.

## -returns
An asynchronous operation that returns a [MediaFrameSourceGroup](mediaframesourcegroup.md) upon successful completion.
