# Windows.Foundation.Diagnostics.LoggingChannel.LoggingChannel

## -description
Initializes a new instance of the [LoggingChannel](loggingchannel.md) class with the specified options.

## -parameters
### -param name
The name of the logging channel.

### -param options
The channel options. Pass **null** to specify the default options.
