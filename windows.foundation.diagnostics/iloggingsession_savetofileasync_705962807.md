# Windows.Foundation.Diagnostics.ILoggingSession.SaveToFileAsync

## -description
Save the current logging session to a file.

## -parameters
### -param folder
The folder to save the log file in.

### -param fileName
The name of the file.

## -returns
When this method completes, it returns the new file as a [StorageFile](../windows.storage/storagefile.md).
