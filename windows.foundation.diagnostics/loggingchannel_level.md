# Windows.Foundation.Diagnostics.LoggingChannel.Level

## -description
Gets the minimum level at which any session is listening to events from this channel.

## -property-value
The event level required for the session to listen to an event from this channel.

## -remarks
This value is only valid if the [Enabled](loggingchannel_enabled.md) property is set to **true**.
