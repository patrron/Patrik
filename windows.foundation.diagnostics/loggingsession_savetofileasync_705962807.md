# Windows.Foundation.Diagnostics.LoggingSession.SaveToFileAsync

## -description
Saves the current logging session to a file.

## -parameters
### -param folder
The folder that contains the log file.

### -param fileName
The name of the log file.

## -returns
When this method completes, it returns the new file as a [StorageFile](../windows.storage/storagefile.md).
