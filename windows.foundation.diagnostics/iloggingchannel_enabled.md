# Windows.Foundation.Diagnostics.ILoggingChannel.Enabled

## -description
Gets a value that indicates whether messages are being logged on the current [LoggingChannel](loggingchannel.md).

## -property-value
**True** if messages are being logged on the current channel; otherwise, **false**.
