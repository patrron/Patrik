# Windows.Foundation.Diagnostics.IFileLoggingSession.CloseAndSaveToFileAsync

## -description
Ends the current logging session and saves it to a file.

## -returns
When this method completes, it returns the new file as a [StorageFile](../windows.storage/storagefile.md).
