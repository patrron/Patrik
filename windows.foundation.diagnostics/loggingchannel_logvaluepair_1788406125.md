# Windows.Foundation.Diagnostics.LoggingChannel.LogValuePair

## -description
Logs data to the current [LoggingChannel](loggingchannel.md) with the specified [LoggingLevel](logginglevel.md).

## -parameters
### -param value1
The string to associate with *value2*.

### -param value2
The value to associate with *value1*.

### -param level
The logging level.
