# Windows.Web.Syndication.SyndicationFeed.LoadFromXml

## -description
Initializes the [SyndicationFeed](syndicationfeed.md) object from the given DOM object, which can contain XML content in either *RSS 2.0 * or *Atom 1.0* format.

## -parameters
### -param feedDocument
The DOM object which contains XML content in either *RSS 2.0* or *Atom 1.0* format.
