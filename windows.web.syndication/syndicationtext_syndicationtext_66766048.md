# Windows.Web.Syndication.SyndicationText.SyndicationText

## -description
Creates a new [SyndicationText](syndicationtext.md) object with the specified **Text** and **Type** property values.

## -parameters
### -param text
The content of a text content construct like **atom:title**.

### -param type
The type of the content. This value can be "text", "html", and "xhtml".
