# Windows.Web.Syndication.SyndicationGenerator.SyndicationGenerator

## -description
Creates a new [SyndicationGenerator](syndicationgenerator.md) object with the specified **Text** property value.

## -parameters
### -param text
Identifies the generator. This parameter represents the text content in the **atom:generator** element or the **generator** element in *RSS 2.0*.
