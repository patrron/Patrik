# Windows.Web.Syndication.SyndicationNode.SyndicationNode

## -description
Creates a new [SyndicationNode](syndicationnode.md) object with the specified **NodeName**, **NodeNamespace**, and **NodeValue** property values.

## -parameters
### -param nodeName
The local name of the element. It must be valid according to XML 1.0.

### -param nodeNamespace
The namespace of the element.

### -param nodeValue
The text content of the element. If the element contains only child elements, this parameter is NULL.
