# Windows.Web.Syndication.SyndicationCategory.SyndicationCategory

## -description
Creates a [SyndicationCategory](syndicationcategory.md) object with the specified **term** property.

## -parameters
### -param term
A string that identifies the category. This parameter represents the required **term** attribute on **atom:category** or the text content of the **category** element in *RSS 2.0*.
