# Windows.UI.Xaml.Markup.IXamlType.IsArray

## -description
Gets a value that indicates whether the [IXamlType](ixamltype.md) represents an array.

## -property-value
**true** if the [IXamlType](ixamltype.md) represents an array; otherwise, **false**.
