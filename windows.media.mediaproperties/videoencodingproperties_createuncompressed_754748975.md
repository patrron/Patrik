# Windows.Media.MediaProperties.VideoEncodingProperties.CreateUncompressed

## -description
Creates an instance of [VideoEncodingProperties](videoencodingproperties.md) for a uncompressed video.

## -parameters
### -param subtype
The media subtype of the format.

### -param width
The video image width.

### -param height
The video image height.

## -returns
The video encoding properties.
