# Windows.Media.MediaProperties.AudioEncodingProperties.CreateAacAdts

## -description
Creates an instance of [AudioEncodingProperties](audioencodingproperties.md) for Advanced Audio Coding (AAC) audio in Audio Data Transport Stream (ADTS) format.

## -parameters
### -param sampleRate
The audio sampling rate.

### -param channelCount
The number of audio channels.

### -param bitrate
The audio bit rate.

## -returns
The audio encoding properties.
