# Windows.Media.MediaProperties.AudioEncodingProperties.CreateWma

## -description
Creates an instance of [AudioEncodingProperties](audioencodingproperties.md) for Windows Media Audio (WMA).

## -parameters
### -param sampleRate
The audio sampling rate.

### -param channelCount
The number of audio channels.

### -param bitrate
The audio bit rate.

## -returns
The audio encoding properties.
