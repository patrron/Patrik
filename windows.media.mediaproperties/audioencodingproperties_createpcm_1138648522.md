# Windows.Media.MediaProperties.AudioEncodingProperties.CreatePcm

## -description
Creates an instance of [AudioEncodingProperties](audioencodingproperties.md) for Pulse Code Modulation (PCM) audio.

## -parameters
### -param sampleRate
The audio sampling rate.

### -param channelCount
The number of audio channels.

### -param bitsPerSample
The audio bit depth.

## -returns
The audio encoding properties.
