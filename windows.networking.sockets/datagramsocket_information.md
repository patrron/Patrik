# Windows.Networking.Sockets.DatagramSocket.Information

## -description
Gets socket information on the local and remote hostnames and local and remote service names for the [DatagramSocket](datagramsocket.md) object.

## -property-value
Socket information for the [DatagramSocket](datagramsocket.md) object.
