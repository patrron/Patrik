# Windows.Networking.Sockets.ServerMessageWebSocketControl.MessageType

## -description
Gets and sets the WebSocket message type to be configured on a [ServerMessageWebSocket](servermessagewebsocket.md) object for write operations.

## -property-value
The WebSocket message type. The default is binary.
