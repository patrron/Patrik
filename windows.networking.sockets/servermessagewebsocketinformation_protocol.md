# Windows.Networking.Sockets.ServerMessageWebSocketInformation.Protocol

## -description
Gets the WebSocket sub-protocol negotiated with the server during WebSocket handshake on a [ServerMessageWebSocket](servermessagewebsocket.md) object.

## -property-value
The sub-protocol negotiated with the server during the WebSocket handshake.
