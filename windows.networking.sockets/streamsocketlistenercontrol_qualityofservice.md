# Windows.Networking.Sockets.StreamSocketListenerControl.QualityOfService

## -description
The quality of service to be set on a [StreamSocket](streamsocket.md) object created when a connection is received by the [StreamSocketListener](streamsocketlistener.md) object.

## -property-value
The quality of service set on a [StreamSocket](streamsocket.md) object created when a connection is received by the [StreamSocketListener](streamsocketlistener.md) object.
