# Windows.Networking.Sockets.IWebSocket.Close

## -description
Closes the [IWebSocket](iwebsocket.md).

## -parameters
### -param code
Status code indicating the reason for closure.

### -param reason
Optional UTF-8-encoded data with additional information about the closure.
