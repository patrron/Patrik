# Windows.Networking.Sockets.DatagramSocketMessageReceivedEventArgs.GetDataStream

## -description
Gets an [IInputStream](../windows.storage.streams/iinputstream.md) object that represents a sequential stream of bytes to be read as a message from the remote network destination on a [DatagramSocket](datagramsocket.md) object.

## -returns
An [IInputStream](../windows.storage.streams/iinputstream.md) object that represents a sequential stream of bytes to be read as a message.
