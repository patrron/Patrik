# Windows.Networking.Sockets.ServerMessageWebSocket

## -description
Supports network communication that allows reading and writing whole messages using a WebSocket object owned by a web server. You can't register for an arbitrary WebSocket connection; instead, an instance of **ServerMessageWebSocket** is created by system code such as the [Windows Device Portal](/windows/uwp/debug-test-perf/device-portal?branch=live) (WDP).
