# Windows.Networking.Sockets.StreamWebSocket.Close

## -description
Closes the [StreamWebSocket](streamwebsocket.md) and indicates a reason for the closure.

## -parameters
### -param code
Status code indicating the reason for closure.

### -param reason
Optional UTF-8-encoded data with additional information about the closure.
