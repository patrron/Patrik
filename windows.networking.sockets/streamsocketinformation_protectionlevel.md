# Windows.Networking.Sockets.StreamSocketInformation.ProtectionLevel

## -description
Gets the protection level that represents the integrity and encryption set on a [StreamSocket](streamsocket.md) object.

## -property-value
The protection level that represents the integrity and encryption set on a [StreamSocket](streamsocket.md) object.
