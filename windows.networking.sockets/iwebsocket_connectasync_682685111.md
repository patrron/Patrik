# Windows.Networking.Sockets.IWebSocket.ConnectAsync

## -description
Starts an asynchronous operation to connect to a remote network destination on a [IWebSocket](iwebsocket.md) object.

## -parameters
### -param uri
An absolute [Uri](../windows.foundation/uri.md) for the server to connect to.

## -returns
An asynchronous connect operation on a [IWebSocket](iwebsocket.md) object.
