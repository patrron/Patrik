# TokenBindingKeyType

## -description
Gets the type of key used for token binding.

## -enum-fields
### -field Rsa2048:0
RSA-2048.

### -field EcdsaP256:1
ECDsaP256.

### -field AnyExisting:2
Any existing key type.
