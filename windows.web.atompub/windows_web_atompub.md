# Windows.Web.AtomPub

## -description
Enables HTTP CRUD access to Web resources using the AtomPub protocol. The [AtomPub sample](http://go.microsoft.com/fwlink/p/?linkid=245061) demonstrates this implementation.

## -remarks
[Windows.Web.AtomPub](windows_web_atompub.md) leverages [Windows.Web.Syndication](../windows.web.syndication/windows_web_syndication.md) classes to represent individual content elements.
