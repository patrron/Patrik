# Windows.Storage.Pickers.FilePickerSelectedFilesArray.GetMany

## -description
Retrieves the [StorageFile](../windows.storage/storagefile.md) objects that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the objects in the collection to return.

### -param items
The items in the collection that start at *startIndex*.

## -returns
The number of items returned.
