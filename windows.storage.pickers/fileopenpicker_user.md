# Windows.Storage.Pickers.FileOpenPicker.User

## -description
Gets info about the user for which the [FileOpenPicker](fileopenpicker.md) was created. Use this property for [multi-user applications](https://docs.microsoft.com/windows/uwp/xbox-apps/multi-user-applications).

## -property-value
The user for which the [FileOpenPicker](fileopenpicker.md) was created.
