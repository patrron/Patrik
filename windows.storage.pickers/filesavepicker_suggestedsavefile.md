# Windows.Storage.Pickers.FileSavePicker.SuggestedSaveFile

## -description
Gets or sets the [storageFile](../windows.storage/storagefile.md) that the file picker suggests to the user for saving a file.

## -property-value
The suggested storage file object to save.
