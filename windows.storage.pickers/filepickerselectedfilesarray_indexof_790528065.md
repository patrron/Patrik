# Windows.Storage.Pickers.FilePickerSelectedFilesArray.IndexOf

## -description
Retrieves the index of a specified [StorageFile](../windows.storage/storagefile.md) object in the collection.

## -parameters
### -param value
The object to find in the collection.

### -param index
The zero-based index of the object if found. Zero is returned if the object is not found.

## -returns
True if the object is found; otherwise false.
