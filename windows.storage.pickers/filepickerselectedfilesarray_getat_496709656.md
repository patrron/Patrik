# Windows.Storage.Pickers.FilePickerSelectedFilesArray.GetAt

## -description
Returns the [StorageFile](../windows.storage/storagefile.md) object at the specified index in the collection.

## -parameters
### -param index
The zero-based index of the object to retrieve.

## -returns
The object at the specified index.
