# Windows.Storage.Pickers.FileSavePicker.PickSaveFileAndContinue

## -description
Shows the file picker so that the user can save a file, deactivating and the app and reactivating it when the operation is complete. (Windows Phone Store app)

## -remarks
> Windows Phone Store app only. Use the [ContinuationData](filesavepicker_continuationdata.md) property to store context information that can be retrieved when the app is reactivated. For more info, see [How to continue your Windows Phone app after calling a file picker](http://msdn.microsoft.com/library/465bbb7a-9ed1-4b57-b60f-e5c6e7cd1470).
