# Windows.Devices.Enumeration.DeviceInformation.Update

## -description
Updates the properties of an existing [DeviceInformation](deviceinformation.md) object.

## -parameters
### -param updateInfo
Indicates the properties to update.

## -remarks
For more info on what the properties represent, see [Device information properties](http://msdn.microsoft.com/library/4a4c2802-e674-4c04-8a6d-d7c1bbf1bd20).
