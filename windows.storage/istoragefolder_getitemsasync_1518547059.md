# Windows.Storage.IStorageFolder.GetItemsAsync

## -description
Gets the items from the current folder.

## -returns
When this method completes successfully, it returns a list of the files and folders (type [IVectorView](../windows.foundation.collections/ivectorview_1.md)). The files and folders in the list are represented by objects of type [IStorageItem](istorageitem.md).
