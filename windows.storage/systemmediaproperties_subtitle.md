# Windows.Storage.SystemMediaProperties.SubTitle

## -description
Gets the name of the [System.Media.SubTitle](https://docs.microsoft.com/windows/desktop/properties/props-system-media-subtitle) property (one of the [Windows media file properties](https://docs.microsoft.com/windows/desktop/properties/media-bumper).

## -property-value
The name of the [System.Media.SubTitle](https://docs.microsoft.com/windows/desktop/properties/props-system-media-subtitle) file property.
