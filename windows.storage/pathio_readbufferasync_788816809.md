# Windows.Storage.PathIO.ReadBufferAsync

## -description
Reads the contents of the file at the specified path or Uniform Resource Identifier (URI) and returns a buffer.

## -parameters
### -param absolutePath
The path of the file to read.

## -returns
When this method completes, it returns an object (type [IBuffer](../windows.storage.streams/ibuffer.md)) that represents the contents of the file.
