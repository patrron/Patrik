# Windows.Storage.ApplicationData.GetForUserAsync

## -description
Static method that returns the [ApplicationData](applicationdata.md) for a [User](../windows.system/user.md).

## -parameters
### -param user
The [User](../windows.system/user.md) for which the [ApplicationData](applicationdata.md) is returned.

## -returns
When this method completes, it returns the requested [ApplicationData](applicationdata.md) object.
