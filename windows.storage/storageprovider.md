# Windows.Storage.StorageProvider

## -description
Contains info about the service that stores files and folders. Files and folders may be stored either by the local file system or by a remote service like Microsoft OneDrive.

## -remarks
For example, if a file or folder is stored on Microsoft OneDrive, [StorageProvider](storageprovider.md) will contain a display name, id, and thumbnail for Microsoft OneDrive.

Typically, you access [StorageProvider](storageprovider.md) objects as the result of calls to asynchronous methods and functions.
