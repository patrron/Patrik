# Windows.Storage.IStorageItem.IsOfType

## -description
Determines whether the current [IStorageItem](istorageitem.md) matches the specified [StorageItemTypes](storageitemtypes.md) value.

## -parameters
### -param type
The value to match against.

## -returns
True if the [IStorageItem](istorageitem.md) matches the specified value; otherwise false.
