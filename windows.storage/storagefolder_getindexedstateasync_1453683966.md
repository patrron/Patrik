# Windows.Storage.StorageFolder.GetIndexedStateAsync

## -description
Gets the indexed state of the current folder.

## -returns
When this method completes successfully, it returns an [IndexedState](../windows.storage.search/indexedstate.md) enum value that describes the state of the folder.
