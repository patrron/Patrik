# Windows.Storage.IStorageFolder.GetFileAsync

## -description
Gets the specified file from the current folder.

## -parameters
### -param name
The name (or path relative to the current folder) of the file to retrieve.

## -returns
When this method completes successfully, it returns a [StorageFile](storagefile.md) that represents the file.
