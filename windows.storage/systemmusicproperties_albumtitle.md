# Windows.Storage.SystemMusicProperties.AlbumTitle

## -description
Gets the name of the [System.Music.AlbumTitle](https://msdn.microsoft.com/library/windows/desktop/bb787292.aspx) property (one of the [Windows music file properties](http://msdn.microsoft.com/library/affb4d7a-b8ef-432c-bcf4-18d90424f9b8).

## -property-value
The name of the [System.Music.AlbumTitle](https://msdn.microsoft.com/library/windows/desktop/bb787292.aspx) file property.
