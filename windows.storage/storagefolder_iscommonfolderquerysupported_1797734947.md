# Windows.Storage.StorageFolder.IsCommonFolderQuerySupported

## -description
Indicates whether the current folder supports the specified [CommonFolderQuery](../windows.storage.search/commonfolderquery.md).

## -parameters
### -param query
The value to test.

## -returns
True if the folder supports the specified [CommonFolderQuery](../windows.storage.search/commonfolderquery.md); otherwise, false.
