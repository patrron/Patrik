# Windows.Storage.StorageFolder.GetBasicPropertiesAsync

## -description
Gets the basic properties of the current folder or file group.

## -returns
When this method completes successfully, it returns the basic properties of the current folder or file group as a [BasicProperties](../windows.storage.fileproperties/basicproperties.md) object.
