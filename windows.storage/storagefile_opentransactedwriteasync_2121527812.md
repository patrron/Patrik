# Windows.Storage.StorageFile.OpenTransactedWriteAsync

## -description
Opens a random-access stream to the file that can be used for transacted-write operations with the specified options.

## -parameters
### -param options
A bitwise combination of the enumeration values that specify options for opening the stream.

## -returns
When this method completes, it returns a [StorageStreamTransaction](storagestreamtransaction.md) that contains the random-access stream and methods that can be used to complete transactions.
