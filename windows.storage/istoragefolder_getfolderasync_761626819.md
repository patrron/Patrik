# Windows.Storage.IStorageFolder.GetFolderAsync

## -description
Gets the specified folder from the current folder.

## -parameters
### -param name
The name of the child folder to retrieve.

## -returns
When this method completes successfully, it returns a [StorageFolder](storagefolder.md) that represents the child folder.
