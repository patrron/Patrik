# Windows.Storage.SystemProperties.Media

## -description
Gets an object that provides the indexing names of [system media file properties](https://msdn.microsoft.com/library/cd71978d-09d9-43f1-ab1b-4b213e3a193b) such as [System.Media.Duration](https://msdn.microsoft.com/library/windows/desktop/bb787399.aspx).

## -property-value
A helper object that provides names for [system media file properties](https://msdn.microsoft.com/library/cd71978d-09d9-43f1-ab1b-4b213e3a193b).
