# Windows.Storage.StorageLibraryChange.IsOfType

## -description
Indicates whether the current [StorageFolder](storagefolder.md) matches the specified [StorageItemTypes](storageitemtypes.md) value.

## -parameters
### -param type
The enum value that determines the object type to match against.

## -returns
True if the [StorageFolder](storagefolder.md) matches the specified [StorageItemTypes](storageitemtypes.md) value; otherwise false.
