# Windows.Storage.StorageFolder.Provider

## -description
Gets the [StorageProvider](storageprovider.md) object that contains info about the service that stores the current folder.

## -property-value
The [StorageProvider](storageprovider.md) object that contains info about the service that stores the current folder. The folder may be stored by the local file system or by a remote service like Microsoft OneDrive.
