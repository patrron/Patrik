# Windows.Storage.SystemMusicProperties.Genre

## -description
Gets the name of the [System.Music.Genre](https://docs.microsoft.com/windows/desktop/properties/props-system-music-genre) property (one of the [Windows music file properties](https://docs.microsoft.com/windows/desktop/properties/music-bumper).

## -property-value
The name of the [System.Music.Genre](https://docs.microsoft.com/windows/desktop/properties/props-system-music-genre) file property.
