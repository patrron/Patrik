# Windows.Storage.StorageFolder.IsCommonFileQuerySupported

## -description
Indicates whether the current folder supports the specified [CommonFileQuery](../windows.storage.search/commonfilequery.md).

## -parameters
### -param query
The value to test.

## -returns
True if the folder supports the specified [CommonFileQuery](../windows.storage.search/commonfilequery.md); otherwise, false.
