# StreamedFileFailureMode

## -description
Indicates the reason that data could not be streamed.

## -enum-fields
### -field Failed:0
Streaming the data failed.

### -field CurrentlyUnavailable:1
The data could not be accessed.

### -field Incomplete:2
Streaming the data failed to complete.
