# Windows.Storage.SystemProperties.ItemNameDisplay

## -description
Gets the name of the [System.ItemNameDisplay](https://msdn.microsoft.com/library/windows/desktop/bb760770.aspx) property (one of the [Windows file properties](https://msdn.microsoft.com/library/7532e58f-bbf2-4e36-9d81-c8e04b92cb7c).

## -property-value
The name of the [System.ItemNameDisplay](https://msdn.microsoft.com/library/windows/desktop/bb760770.aspx) file property.
