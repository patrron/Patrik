# Windows.Storage.SystemProperties.ItemNameDisplay

## -description
Gets the name of the [System.ItemNameDisplay](https://docs.microsoft.com/windows/desktop/properties/props-system-itemnamedisplay) property (one of the [Windows file properties](https://docs.microsoft.com/windows/desktop/properties/props).

## -property-value
The name of the [System.ItemNameDisplay](https://docs.microsoft.com/windows/desktop/properties/props-system-itemnamedisplay) file property.
