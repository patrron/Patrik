# Windows.Storage.SystemMediaProperties.Writer

## -description
Gets the name of the [System.Media.Writer](https://msdn.microsoft.com/library/windows/desktop/bb787422.aspx) property (one of the [Windows media file properties](http://msdn.microsoft.com/library/cd71978d-09d9-43f1-ab1b-4b213e3a193b).

## -property-value
The name of the [System.Media.Writer](https://msdn.microsoft.com/library/windows/desktop/bb787422.aspx) file property.
