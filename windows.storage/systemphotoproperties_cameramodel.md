# Windows.Storage.SystemPhotoProperties.CameraModel

## -description
Gets the name of the [System.Photo.CameraModel](https://msdn.microsoft.com/library/windows/desktop/bb760388.aspx) property (one of the [Windows photo file properties](https://msdn.microsoft.com/library/aad47c36-fd1d-478d-ad7d-a40c3984a1d9).

## -property-value
The name of the [System.Photo.CameraModel](https://msdn.microsoft.com/library/windows/desktop/bb760388.aspx) file property.
