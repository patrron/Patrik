# Windows.Storage.SystemProperties.GPS

## -description
Gets an object that provides the indexing names of [Windows system file properties](https://docs.microsoft.com/windows/desktop/properties/props) for **System.GPS**.

## -property-value
A helper object that provides names for [GPS-related file properties](https://docs.microsoft.com/windows/desktop/properties/gps-bumper).
