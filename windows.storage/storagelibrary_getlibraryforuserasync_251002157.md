# Windows.Storage.StorageLibrary.GetLibraryForUserAsync

## -description
Gets the specified library for a [User](../windows.system/user.md).

## -parameters
### -param user
The [User](../windows.system/user.md) for which the library is retrieved.

### -param libraryId
A [KnownLibraryId](knownlibraryid.md) value that identifies the library to retrieve, like Music or Videos.

## -returns
When this method completes successfully, it returns the library (type [StorageLibrary](storagelibrary.md)).
