# Windows.Storage.SystemAudioProperties.EncodingBitrate

## -description
Gets the name of the [System.Audio.EncodingBitrate](https://msdn.microsoft.com/library/windows/desktop/bb773922.aspx) property (one of the [Windows audio file properties](https://msdn.microsoft.com/library/ed8fe840-f26f-43dc-b94f-f57941c2801f)).

## -property-value
The name of the [System.Audio.EncodingBitrate](https://msdn.microsoft.com/library/windows/desktop/bb773922.aspx) property.
