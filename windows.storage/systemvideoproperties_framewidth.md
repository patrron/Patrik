# Windows.Storage.SystemVideoProperties.FrameWidth

## -description
Gets the name of the [System.Video.FrameWidth](https://docs.microsoft.com/windows/desktop/properties/props-system-video-framewidth) property (one of the [Windows video file properties](https://docs.microsoft.com/windows/desktop/properties/video-bumper).

## -property-value
The name of the [System.Video.FrameWidth](https://docs.microsoft.com/windows/desktop/properties/props-system-video-framewidth) file property.
