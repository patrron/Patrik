# Windows.Storage.SystemImageProperties.VerticalSize

## -description
Gets the name of the [System.Image.VerticalSize](https://msdn.microsoft.com/library/windows/desktop/bb787470.aspx) property (one of the [Windows image file properties](https://msdn.microsoft.com/library/278eadc0-62ab-4b28-b238-93e96ce9da00)).

## -property-value
The name of the [System.Image.VerticalSize](https://msdn.microsoft.com/library/windows/desktop/bb787470.aspx) file property.
