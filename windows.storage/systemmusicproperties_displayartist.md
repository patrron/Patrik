# Windows.Storage.SystemMusicProperties.DisplayArtist

## -description
Gets the name of the [System.Music.DisplayArtist](https://msdn.microsoft.com/library/windows/desktop/dd807135.aspx) property (one of the [Windows music file properties](https://msdn.microsoft.com/library/affb4d7a-b8ef-432c-bcf4-18d90424f9b8).

## -property-value
The name of the [System.Music.DisplayArtist](https://msdn.microsoft.com/library/windows/desktop/dd807135.aspx) file property.
