# Windows.Storage.SystemPhotoProperties.Orientation

## -description
Gets the name of the [System.Photo.Orientation](https://docs.microsoft.com/windows/desktop/properties/props-system-photo-orientation) property (one of the [Windows photo file properties](https://docs.microsoft.com/windows/desktop/properties/photo-bumper).

## -property-value
The name of the [System.Photo.Orientation](https://docs.microsoft.com/windows/desktop/properties/props-system-photo-orientation) file property.
