# Windows.Storage.SystemProperties.Author

## -description
Gets the name of the [System.Author](https://msdn.microsoft.com/library/windows/desktop/bb760652.aspx) property (one of the [Windows file properties](http://msdn.microsoft.com/library/7532e58f-bbf2-4e36-9d81-c8e04b92cb7c).

## -property-value
The name of the [System.Author](https://msdn.microsoft.com/library/windows/desktop/bb760652.aspx) file property.
