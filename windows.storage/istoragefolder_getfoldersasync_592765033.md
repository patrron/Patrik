# Windows.Storage.IStorageFolder.GetFoldersAsync

## -description
Gets the folders in the current folder.

## -returns
When this method completes successfully, it returns a list of the files (type [IVectorView](../windows.foundation.collections/ivectorview_1.md)). Each folder in the list is represented by a [StorageFolder](storagefolder.md).
