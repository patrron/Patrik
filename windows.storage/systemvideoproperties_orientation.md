# Windows.Storage.SystemVideoProperties.Orientation

## -description
Gets the name of the [System.Video.Orientation](https://msdn.microsoft.com/library/windows/desktop/mt805791.aspx) property (one of the [Windows video file properties](https://msdn.microsoft.com/library/5d36b057-3712-4d00-99b7-3ddf45620b7a).

## -property-value
The name of the [System.Video.Orientation](https://msdn.microsoft.com/library/windows/desktop/mt805791.aspx) file property.
