# Windows.Storage.SystemVideoProperties.Orientation

## -description
Gets the name of the [System.Video.Orientation](https://docs.microsoft.com/windows/desktop/properties/props-system-video-orientation) property (one of the [Windows video file properties](https://docs.microsoft.com/windows/desktop/properties/video-bumper).

## -property-value
The name of the [System.Video.Orientation](https://docs.microsoft.com/windows/desktop/properties/props-system-video-orientation) file property.
