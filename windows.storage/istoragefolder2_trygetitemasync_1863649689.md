# Windows.Storage.IStorageFolder2.TryGetItemAsync

## -description
Try to get a single file or sub-folder from the current folder by using the name of the item.

## -parameters
### -param name
The name (or path relative to the current folder) of the file or sub-folder to try to retrieve.

## -returns
When this method completes successfully, it returns the file or folder (type [IStorageItem](istorageitem.md)).
