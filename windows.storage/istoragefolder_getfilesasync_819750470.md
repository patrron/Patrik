# Windows.Storage.IStorageFolder.GetFilesAsync

## -description
Gets the files from the current folder.

## -returns
When this method completes successfully, it returns a list of the files (type [IVectorView](../windows.foundation.collections/ivectorview_1.md)) in the folder. Each file in the list is represented by a [StorageFile](storagefile.md) object.
