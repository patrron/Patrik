# Windows.Storage.SystemProperties.Audio

## -description
Gets an object that provides the indexing names of [Windows file properties](https://msdn.microsoft.com/library/7532e58f-bbf2-4e36-9d81-c8e04b92cb7c) for **System.Audio**.

## -property-value
A helper object that provides names for [Windows file properties](https://msdn.microsoft.com/library/7532e58f-bbf2-4e36-9d81-c8e04b92cb7c) for **System.Audio**.
