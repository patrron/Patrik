# Windows.Phone.Management.Deployment.Enterprise.WorkplaceId

## -description
Gets the unique ID of the enterprise, as it appears on the enterprise certificate.

## -property-value
The unique ID of the enterprise, as it appears on the enterprise certificate.

## -remarks
The [Id](enterprise_id.md) property and the WorkplaceId property return the same ID value in different formats.
