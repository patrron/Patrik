# PosPrinterLineDirection

## -description
Describes the possible directions that a receipt or slip printer station can use to print a ruled line.

## -enum-fields
### -field Horizontal:0
Prints a horizontal ruled line.

### -field Vertical:1
Prints a vertical ruled line.
