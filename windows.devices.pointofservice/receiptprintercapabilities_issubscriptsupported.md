# Windows.Devices.PointOfService.ReceiptPrinterCapabilities.IsSubscriptSupported

## -description
Gets whether the receipt printer station can print characters in subscript style.

## -property-value
True if the receipt printer station can print characters in subscript style; otherwise false.
