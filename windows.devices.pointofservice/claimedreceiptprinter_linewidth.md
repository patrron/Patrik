# Windows.Devices.PointOfService.ClaimedReceiptPrinter.LineWidth

## -description
Gets the current width of the printed line for the receipt printer station, in the units that the [ClaimedPosPrinter.MapMode](claimedposprinter_mapmode.md) property specifies.

## -property-value
The current width of the printed line for the receipt printer station, in the units that the [ClaimedPosPrinter.MapMode](claimedposprinter_mapmode.md) property specifies.
