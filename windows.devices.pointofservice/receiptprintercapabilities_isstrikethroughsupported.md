# Windows.Devices.PointOfService.ReceiptPrinterCapabilities.IsStrikethroughSupported

## -description
Gets whether the receipt printer station can print characters in strikethrough style.

## -property-value
True if the receipt printer station can print characters in strikethrough style; otherwise false.
