# Windows.Devices.PointOfService.PosPrinterPrintOptions.CharacterSet

## -description
Gets or sets a numeric value that indicates the character set that the application wants to use for printing characters.

## -property-value
The character set that the application wants to use for printing characters. The value must be 0 or one of the supported character set values of the point-of-service printer.
