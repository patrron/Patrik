# PosPrinterMapMode

## -description
Describes the valid units of measure for point-of-service printers.

## -enum-fields
### -field Dots:0
The dot width for the point-of-service printer.

### -field Twips:1
1/1440 of an inch.

### -field English:2
0.001 inch.

### -field Metric:3
0.01 millimeter.
