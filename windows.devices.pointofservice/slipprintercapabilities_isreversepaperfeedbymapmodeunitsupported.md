# Windows.Devices.PointOfService.SlipPrinterCapabilities.IsReversePaperFeedByMapModeUnitSupported

## -description
Gets whether the slip printer station can reverse paper feed by map mode unit.

## -property-value
True if the slip printer station can reverse paper feed by map mode unit; otherwise false.
