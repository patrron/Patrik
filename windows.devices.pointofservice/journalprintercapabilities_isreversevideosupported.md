# Windows.Devices.PointOfService.JournalPrinterCapabilities.IsReverseVideoSupported

## -description
Gets whether the journal printer station can print characters in reverse-video style.

## -property-value
True if the journal printer station can print characters in reverse-video style; otherwise false.
