# PosPrinterAlignment

## -description
Describes the possible horizontal alignments of the text that a point-of-service printer prints on the page.

## -enum-fields
### -field Left:0
Text is left-aligned.

### -field Center:1
Text is centered.

### -field Right:2
Text is right-aligned.
