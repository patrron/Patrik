# Windows.Devices.PointOfService.SlipPrinterCapabilities.IsStrikethroughSupported

## -description
Gets whether the slip printer station can print characters in strikethrough style.

## -property-value
True if the slip printer station can print characters in strikethrough style; otherwise false.
