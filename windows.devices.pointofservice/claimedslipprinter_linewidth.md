# Windows.Devices.PointOfService.ClaimedSlipPrinter.LineWidth

## -description
Gets the current width of the printed line for the slip printer station, in the units that the [ClaimedPosPrinter.MapMode](claimedposprinter_mapmode.md) property specifies.

## -property-value
The current width of the printed line for the slip printer station, in the units that the [ClaimedPosPrinter.MapMode](claimedposprinter_mapmode.md) property specifies.
