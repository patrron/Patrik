# Windows.Devices.PointOfService.ClaimedSlipPrinter.LinesNearEndToEnd

## -description
Gets the number of lines that the slip printer station can print after the [ClaimedSlipPrinter.IsPaperNearEnd](claimedslipprinter_ispapernearend.md) property is set to **true** but before the printer reaches the end of the slip.

## -property-value
The number of lines that the slip printer station can print after the [ClaimedSlipPrinter.IsPaperNearEnd](claimedslipprinter_ispapernearend.md) property is set to **true** but before the printer reaches the end of the slip.
