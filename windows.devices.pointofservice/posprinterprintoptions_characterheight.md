# Windows.Devices.PointOfService.PosPrinterPrintOptions.CharacterHeight

## -description
Gets or sets a numeric value that indicates the character height that the application wants to use for printing characters.

## -property-value
The character height that the application wants to use for printing characters. The value must be 0 or one of the supported height values of the font if the font is not scalable.
