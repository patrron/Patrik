# Windows.Phone.PersonalInformation.KnownContactProperties.DisplayName

## -description
Gets the **DisplayName** property name.

## -property-value
The **DisplayName** property name.

## -remarks
The value returned for the **DisplayName** key in a contact **Properties** collection is a string.
