# ContactStoreSystemAccessMode

## -description
Specifies the system access mode for a custom contact store created with [CreateOrOpenAsync](contactstore_createoropenasync_214074668.md).

## -enum-fields
### -field ReadOnly:0
The operating system can only read from the contact store.

### -field ReadWrite:1
The operating system can modify contacts in the store.
