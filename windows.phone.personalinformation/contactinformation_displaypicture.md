# Windows.Phone.PersonalInformation.ContactInformation.DisplayPicture

## -description
Gets the display picture of a contact.

## -property-value
A stream containing the image data.

## -remarks
To set this value, use [SetDisplayPictureAsync](contactinformation_setdisplaypictureasync_974033820.md).
