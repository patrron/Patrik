# Windows.Phone.PersonalInformation.StoredContact.Id

## -description
Gets the local identifier of the stored contact.

## -property-value
The local identifier of the contact.

## -remarks
This identifier is assigned to the contact by the system. Your app can set a custom identifier by using the [RemoteId](storedcontact_remoteid.md) property. Use this to associate contacts in the contact store with contacts in a remote, cloud-based contact store.
