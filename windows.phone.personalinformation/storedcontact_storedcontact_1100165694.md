# Windows.Phone.PersonalInformation.StoredContact.StoredContact

## -description
Initializes a new instance of the [StoredContact](storedcontact.md) class and initializes the property values from the provided [ContactInformation](contactinformation.md) object.

## -parameters
### -param store
The contact store in which the contact should be created.

### -param contact
The contact information object with which the new contact object properties will be initialized.
