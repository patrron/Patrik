# Windows.Phone.PersonalInformation.KnownContactProperties.Nickname

## -description
Gets the **Nickname** property name.

## -property-value
The **Nickname** property name.

## -remarks
The value returned for the **Nickname** key in a contact **Properties** collection is a string.
