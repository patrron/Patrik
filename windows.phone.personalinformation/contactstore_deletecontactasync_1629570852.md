# Windows.Phone.PersonalInformation.ContactStore.DeleteContactAsync

## -description
Deletes the contact with the specified ID from the contact store.

## -parameters
### -param id
The ID of the contact to be deleted.

## -returns
An asynchronous action. If you use [Asynchronous programming](http://msdn.microsoft.com/library/23fe28f1-89c5-4a17-a732-a722648f9c5e), the result type is **void**.
