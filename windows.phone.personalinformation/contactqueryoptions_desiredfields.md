# Windows.Phone.PersonalInformation.ContactQueryOptions.DesiredFields

## -description
Gets or sets the list of properties that should be returned with each contact.

## -property-value
The list of properties that should be returned with each contact.

## -remarks
The default set of fields are those that are properties of the [StoredContact](storedcontact.md) class. Additional field names should be specified using one of the properties of [KnownContactProperties](knowncontactproperties.md) class.
