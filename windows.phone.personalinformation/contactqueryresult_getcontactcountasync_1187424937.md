# Windows.Phone.PersonalInformation.ContactQueryResult.GetContactCountAsync

## -description
Gets the number of contacts in the contact store.

## -returns
When this method completes, it returns the number of contacts in the contact store. If you use [Asynchronous programming](https://msdn.microsoft.com/library/23fe28f1-89c5-4a17-a732-a722648f9c5e), the result type is an unsigned integer number.
