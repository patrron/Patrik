# Windows.Devices.Sensors.Accelerometer.GetDefault

## -description
Returns the default accelerometer of a specific type of sensor. The possible accelerometer sensors are defined by [AccelerometerReadingType](accelerometerreadingtype.md).

## -parameters
### -param readingType
Indicates the type of accelerometer to retrieve.

## -returns
The default accelerometer or **null** if no integrated accelerometers are found.
