# Windows.Devices.Sensors.ActivitySensor.GetCurrentReadingAsync

## -description
Asynchronously gets the current sensor reading.

## -returns
Asynchronously returns a [ActivitySensorReading](activitysensorreading.md) object that represents info about the sensor.
