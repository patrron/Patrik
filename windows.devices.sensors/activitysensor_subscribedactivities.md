# Windows.Devices.Sensors.ActivitySensor.SubscribedActivities

## -description
Gets the list of activity types that the sensor pledges to perform.

## -property-value
The list of [ActivityType](activitytype.md)-typed values for the activity types that the sensor pledges to perform.
