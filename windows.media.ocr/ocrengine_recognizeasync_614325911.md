# Windows.Media.Ocr.OcrEngine.RecognizeAsync

## -description
Scans the specified image for text in the language specified by the [RecognizerLanguage](ocrengine_recognizerlanguage.md) property.

## -parameters
### -param bitmap
Represents an uncompressed bitmap.

## -returns
The result of the OCR that was initiated by the [OcrEngine](ocrengine.md) object.
