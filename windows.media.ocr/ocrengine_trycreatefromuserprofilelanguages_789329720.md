# Windows.Media.Ocr.OcrEngine.TryCreateFromUserProfileLanguages

## -description
Creates a new instance of the [OcrEngine](ocrengine.md) class.

## -returns
If any language from **GlobalizationPreferences.Languages** can be resolved to any of the available OCR languages returns new instance of **OcrEngine** class, otherwise returns **null**.
