# Windows.UI.Xaml.Navigation.NavigationFailedEventArgs.SourcePageType

## -description
Gets the data type of the target page.

Equivalent WinUI property: [Microsoft.UI.Xaml.Navigation.NavigationFailedEventArgs.SourcePageType](/windows/winui/api/microsoft.ui.xaml.navigation.navigationfailedeventargs.sourcepagetype).

## -property-value
The data type of the target page, as a type reference ([System.Type](/dotnet/api/system.type?view=dotnet-uwp-10.0&preserve-view=true) for Microsoft .NET, a [TypeName](../windows.ui.xaml.interop/typename.md) helper struct for Visual C++ component extensions (C++/CX)).

## -remarks
> [!NOTE]
> If you are programming using a Microsoft .NET language (C# or Microsoft Visual Basic), the [TypeName](../windows.ui.xaml.interop/typename.md) type projects as [System.Type](/dotnet/api/system.type?view=dotnet-uwp-10.0&preserve-view=true). If you're using Visual C++ component extensions (C++/CX)), this is a [TypeName](../windows.ui.xaml.interop/typename.md) helper struct.
