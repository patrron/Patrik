# ClosedCaptionOpacity

## -description
Specifies the opacity of closed caption text.

## -enum-fields
### -field Default:0
Default opacity.

### -field OneHundredPercent:1
100% opaque.

### -field SeventyFivePercent:2
75% opaque.

### -field TwentyFivePercent:3
25% opaque.

### -field ZeroPercent:4
Completely transparent.
