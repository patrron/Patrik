# ClosedCaptionSize

## -description
Specifies the size of closed caption text.

## -enum-fields
### -field Default:0
Default text size.

### -field FiftyPercent:1
50% text size.

### -field OneHundredPercent:2
100% text size.

### -field OneHundredFiftyPercent:3
150% text size.

### -field TwoHundredPercent:4
200% text size.
