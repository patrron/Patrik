# ClosedCaptionStyle

## -description
Specifies the style of closed caption text.

## -enum-fields
### -field Default:0
The default style.

### -field MonospacedWithSerifs:1
Monospaced with serifs.

### -field ProportionalWithSerifs:2
Proportional with serifs.

### -field MonospacedWithoutSerifs:3
Monospaced without serifs

### -field ProportionalWithoutSerifs:4
Proportional without serifs.

### -field Casual:5
Casual.

### -field Cursive:6
Cursive.

### -field SmallCapitals:7
Small capitals.
