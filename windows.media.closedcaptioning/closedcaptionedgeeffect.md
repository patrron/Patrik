# ClosedCaptionEdgeEffect

## -description
Specifies the edge effect of closed caption text.

## -enum-fields
### -field Default:0
The default edge effect.

### -field None:1
No edge effect.

### -field Raised:2
Raised edge.

### -field Depressed:3
Depressed edge.

### -field Uniform:4
Uniform edge

### -field DropShadow:5
Drop shadow.
