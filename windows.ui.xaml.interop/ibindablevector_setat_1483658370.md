# Windows.UI.Xaml.Interop.IBindableVector.SetAt

## -description
Sets the item value at the specified index of the vector.

## -parameters
### -param index
The zero-based index of the vector, at which to set the value.

### -param value
The item value to set.

## -remarks
When programming with .NET, this interface is hidden and developers should use the [System.Collections.IList](https://msdn.microsoft.com/library/system.collections.ilist.aspx) interface.
