# Windows.Media.SystemMediaTransportControls.GetForCurrentView

## -description
The system media transport controls for the current view.

## -returns
The system media transport controls for the current view.

## -remarks
For information about using this API to get a reference to manually integrate your app with the system media transport controls,
see [Manual control of the System Media Transport Controls](https://docs.microsoft.com/windows/uwp/audio-video-camera/system-media-transport-controls).
