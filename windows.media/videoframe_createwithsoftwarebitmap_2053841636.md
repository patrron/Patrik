# Windows.Media.VideoFrame.CreateWithSoftwareBitmap

## -description
Creates a new instance of the [VideoFrame](videoframe.md) class from the provided [SoftwareBitmap](../windows.graphics.imaging/softwarebitmap.md).

## -parameters
### -param bitmap
The [SoftwareBitmap](../windows.graphics.imaging/softwarebitmap.md) from which the **VideoFrame** is created.

## -returns
A **VideoFrame**.

## -remarks
If the *bitmap* parameter is null, an argument exception will be thrown.
