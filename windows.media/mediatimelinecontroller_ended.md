# Windows.Media.MediaTimelineController.Ended

## -description
Occurs when the timeline controller reaches the end of the time window specified by the [Duration](mediatimelinecontroller_duration.md) if [IsLoopingEnabled](mediatimelinecontroller_isloopingenabled) is set to false.
