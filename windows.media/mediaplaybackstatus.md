# MediaPlaybackStatus

## -description
Defines values for the status of the media playback.

## -enum-fields
### -field Closed:0
The media is closed.

### -field Changing:1
The media is changing.

### -field Stopped:2
The media is stopped.

### -field Playing:3
The media is playing.

### -field Paused:4
The media is paused.
