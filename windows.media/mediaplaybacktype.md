# MediaPlaybackType

## -description
Defines values for the types of media playback.

## -enum-fields
### -field Unknown:0
The media type is unknown.

### -field Music:1
The media type is audio music.

### -field Video:2
The media type is video.

### -field Image:3
The media type is an image.
