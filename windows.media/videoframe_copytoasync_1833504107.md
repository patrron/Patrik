# Windows.Media.VideoFrame.CopyToAsync

## -description
Copies the specified rectangle in the current [VideoFrame](videoframe.md) into the specified rectangle of a provided [VideoFrame](videoframe.md) object.

## -parameters
### -param frame
The target video frame into which the current frame will be copied.

### -param sourceBounds
A [BitmapBounds](../windows.graphics.imaging/bitmapbounds.md) reference specifying the rectangle within the source frame that should be copied.

### -param destinationBounds
A [BitmapBounds](../windows.graphics.imaging/bitmapbounds.md) reference specifying the rectangle within the target frame into which the source content is copied.

## -returns
An asynchronous action.
