# Windows.UI.Xaml.Data.PropertyChangedEventArgs.PropertyName

## -description
Gets the name of the property that changed.

> **.NET**
> This type is hidden, use [System.ComponentModel.PropertyChangedEventArgs](https://msdn.microsoft.com/library/system.componentmodel.propertychangedeventargs.aspx).

## -property-value
The name of the property that changed.
