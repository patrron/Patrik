# Windows.UI.Xaml.Data.ItemIndexRange.LastIndex

## -description

Equivalent WinUI property: [Microsoft.UI.Xaml.Data.ItemIndexRange.LastIndex](/windows/winui/api/microsoft.ui.xaml.data.itemindexrange.lastindex).

## -property-value
