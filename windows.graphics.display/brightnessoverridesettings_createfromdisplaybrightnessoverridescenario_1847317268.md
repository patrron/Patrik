# Windows.Graphics.Display.BrightnessOverrideSettings.CreateFromDisplayBrightnessOverrideScenario



## -description


Creates a [BrightnessOverrideSettings](brightnessoverridesettings.md) object with a brightness level corresponding to the specified brightness scenario.

## -parameters
### -param overrideScenario


The override scenario on which to base the settings.


## -returns


The override settings.
