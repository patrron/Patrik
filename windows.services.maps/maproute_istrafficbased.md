# Windows.Services.Maps.MapRoute.IsTrafficBased

## -description
Gets a value indicating whether the [MapRoute](maproute.md) is based on traffic.

## -property-value
**true** if the [MapRoute](maproute.md) is based on traffic; otherwise, **false**.
