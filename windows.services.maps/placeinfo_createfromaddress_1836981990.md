# Windows.Services.Maps.PlaceInfo.CreateFromAddress

## -description
Creates a [PlaceInfo](placeinfo.md) instance by using an address and a display name.

## -parameters
### -param displayAddress
The address of the location that you want to appear in the placecard.

### -param displayName
The name that you want to appear as a label in the placecard.
