# Windows.Services.Maps.MapLocationFinder.FindLocationsAtAsync

## -description
Converts a geographic location to a collection of addresses (reverse geocoding).

## -parameters
### -param queryPoint
The point for which you want to get locations.

## -returns
When this method completes successfully, it returns a list of locations contained in the [MapLocationFinderResult](maplocationfinderresult.md).
