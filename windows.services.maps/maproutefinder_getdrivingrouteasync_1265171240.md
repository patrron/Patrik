# Windows.Services.Maps.MapRouteFinder.GetDrivingRouteAsync

## -description
Gets a driving route for the specified start and end coordinates (using [MapRouteDrivingOptions](maproutedrivingoptions.md)).

## -parameters
### -param startPoint
The starting point of the driving route.

### -param endPoint
The ending point of the driving route.

### -param options
The advanced routing options to be applied to this route.

## -returns
When this method completes successfully, it returns one or more driving routes contained in the [MapRouteFinderResult](maproutefinderresult.md).
