# Windows.Services.Maps.MapLocationFinderResult.Locations

## -description
Gets the list of locations found by a [MapLocationFinder](maplocationfinder.md) query.

## -property-value
The list of locations found by a [MapLocationFinder](maplocationfinder.md) query. This property returns a collection of [MapLocation](maplocation.md) objects.
