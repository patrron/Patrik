# ManualFocusDistance

## -description
Defines the options for manual focus distance.

## -enum-fields
### -field Infinity:0
Focus to infinity.

### -field Hyperfocal:1
Focus to the hyperfocal distance.

### -field Nearest:2
Focus to the nearest distance.
