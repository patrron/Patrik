# Windows.Media.Devices.VideoTemporalDenoisingControl.Supported

## -description
Gets a value that indicates if the capture device supports the [VideoTemporalDenoisingControl](videotemporaldenoisingcontrol.md).

## -property-value
True if the [VideoTemporalDenoisingControl](videotemporaldenoisingcontrol.md) is supported; otherwise, false.
