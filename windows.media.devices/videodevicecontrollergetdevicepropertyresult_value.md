# Windows.Media.Devices.VideoDeviceControllerGetDevicePropertyResult.Value

## -description
Gets an object representing the result value of an operation to get the value of a video device controller driver property.

## -property-value
An object representing the result value of an operation to get the value of a video device controller driver property. If the value of [Status](videodevicecontrollergetdevicepropertyresult_status.md) is not **Success**, this value will be null.
