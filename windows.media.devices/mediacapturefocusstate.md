# MediaCaptureFocusState

## -description
Defines the different focus states the capture device can be in.

## -enum-fields
### -field Uninitialized:0
The focus state is uninitialized.

### -field Lost:1
The focus has been lost.

### -field Searching:2
Searching for focus.

### -field Focused:3
Focused.

### -field Failed:4
The attempt to focus has failed.
