# CaptureUse

## -description
Defines the values for the primary use of the capture device.

## -enum-fields
### -field None:0
The capture device does not have a primary use.

### -field Photo:1
The capture device is used primarily for photos.

### -field Video:2
The capture device is used primarily for video.
