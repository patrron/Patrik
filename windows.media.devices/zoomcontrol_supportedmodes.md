# Windows.Media.Devices.ZoomControl.SupportedModes

## -description
Gets the list of [ZoomTransitionMode](zoomtransitionmode.md) values indicating the modes supported by the capture device.

## -property-value
The list of supported [ZoomTransitionMode](zoomtransitionmode.md) values.
