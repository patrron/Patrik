# Windows.Media.Devices.InfraredTorchControl.IsSupported

## -description
Gets a value that indicates if the capture device supports the [InfraredTorchControl](infraredtorchcontrol.md).

## -property-value
**true** if the infrared control is supported; otherwise, **false**.
