# ZoomTransitionMode

## -description
Defines the zoom transition modes.

## -enum-fields
### -field Auto:0
The default zoom transition mode for the device. Can be Direct or Smooth, depending on the device.

### -field Direct:1
The zoom transition is instantaneous.

### -field Smooth:2
The zoom transition is smooth.
