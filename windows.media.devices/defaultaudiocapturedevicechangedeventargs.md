# Windows.Media.Devices.DefaultAudioCaptureDeviceChangedEventArgs

## -description
Contains information about a default audio capture device change event.

## -remarks
This object is not instantiated directly. It is created by the [MediaDevice.DefaultAudioCaptureDeviceChanged](mediadevice_defaultaudiocapturedevicechanged.md) event and is returned as an argument to the [TypedEventHandler](../windows.foundation/typedeventhandler_2.md) delegate.
