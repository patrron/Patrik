# Windows.Media.Devices.AudioDeviceModule.SendCommandAsync

## -description
Asynchronously sends data to the audio device module and receives the result.

## -parameters

### -param Command
A buffer containing the command data to send to the audio device module.

## -returns
An asynchronous operation the returns a [ModuleCommandResult](modulecommandresult.md) upon successful completion.
