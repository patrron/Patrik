# TelephonyKey

## -description
Indicates the keypad button that was pressed on a telephony device.

## -enum-fields

The "0" keypad button.

### -field D1:1
The "1" keypad button.

### -field D2:2
The "2" keypad button.

### -field D3:3
The "3" keypad button.

### -field D4:4
The "4" keypad button.

### -field D5:5
The "5" keypad button.

### -field D6:6
The "6" keypad button.

### -field D7:7
The "7" keypad button.

### -field D8:8
The "8" keypad button.

### -field D9:9
The "9" keypad button.

### -field Star:10
The "*" keypad button.

### -field Pound:11
The "#" keypad button.

### -field A:12
Keypad button A.

### -field B:13
Keypad button B.

### -field C:14
Keypad button C.

### -field D:15
Keypad button D.
