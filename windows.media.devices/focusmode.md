# FocusMode

## -description
Defines the values for focus mode.

## -enum-fields
### -field Auto:0
Use autofocus.

### -field Single:1
Focus once.

### -field Continuous:2
Continuously adjust focus.

### -field Manual:3
Use manual focus.
