# Windows.UI.Input.GestureRecognizer.GestureSettings

## -description
Gets or sets a value that indicates the gesture and manipulation settings supported by an application.

## -property-value
The gesture settings supported by an application. The value of this property is a bitwise OR of members of [GestureSettings](gesturerecognizer_gesturesettings.md) enumeration.
