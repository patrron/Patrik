# Windows.UI.Input.ManipulationStartedEventArgs.Cumulative

## -description
Gets values that indicate the accumulated transformation deltas (translation, rotation, scale) for a manipulation before the [ManipulationStarted](gesturerecognizer_manipulationstarted.md) event.

## -property-value
The accumulated transformation values up to the [ManipulationStarted](gesturerecognizer_manipulationstarted.md) event.
