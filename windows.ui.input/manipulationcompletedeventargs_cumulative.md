# Windows.UI.Input.ManipulationCompletedEventArgs.Cumulative

## -description
Gets values that indicate the accumulated transformation deltas (translation, rotation, scale) of a completed manipulation (from the start of the manipulation to the end of inertia).

## -property-value
The accumulated transformation values since a [ManipulationStarted](gesturerecognizer_manipulationstarted.md) event.
