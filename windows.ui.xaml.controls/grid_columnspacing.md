# Windows.UI.Xaml.Controls.Grid.ColumnSpacing

## -description

Gets or sets the uniform distance (in pixels) between grid columns.

## -xaml-syntax

```xaml
<Grid ColumnSpacing="double" .../>
```

## -property-value

The uniform distance (in pixels) between grid columns.
