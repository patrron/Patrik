# Windows.UI.Xaml.Controls.ScrollViewer.VerticalOffset

## -description
Gets the distance the content has been scrolled vertically.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.ScrollViewer.VerticalOffset](/windows/winui/api/microsoft.ui.xaml.controls.scrollviewer.verticaloffset).

## -property-value
The distance the content has been scrolled vertically.
