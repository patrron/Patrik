# Windows.UI.Xaml.Controls.TextBlock.SelectionStart

## -description
Gets the starting position of the text selected in the [TextBlock](textblock.md).

## -property-value
An object that represents the selection start, or null if no selection exists.

## -remarks
Use [Select](textblock_select_694729799.md) to set the selection start and end.
