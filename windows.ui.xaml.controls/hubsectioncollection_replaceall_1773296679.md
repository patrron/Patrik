# Windows.UI.Xaml.Controls.HubSectionCollection.ReplaceAll

## -description

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.HubSectionCollection.ReplaceAll](/windows/winui/api/microsoft.ui.xaml.controls.hubsectioncollection.replaceall).

### -param items
