# Windows.UI.Xaml.Controls.SemanticZoomViewChangedEventArgs.SourceItem

## -description
Provides information about the item and its bounds, for the item as represented in the previous view.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.SemanticZoomViewChangedEventArgs.SourceItem](/windows/winui/api/microsoft.ui.xaml.controls.semanticzoomviewchangedeventargs.sourceitem).

## -property-value
