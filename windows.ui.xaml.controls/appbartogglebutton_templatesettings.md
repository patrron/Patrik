# Windows.UI.Xaml.Controls.AppBarToggleButton.TemplateSettings

## -description

Gets an object that provides calculated values that can be referenced as [{TemplateBinding} markup extension](https://docs.microsoft.com/windows/uwp/xaml-platform/templatebinding-markup-extension) sources when defining templates for an AppBarToggleButton control.

## -property-value

An object that provides calculated values for templates.
