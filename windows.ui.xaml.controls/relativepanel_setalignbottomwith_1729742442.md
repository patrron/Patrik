# Windows.UI.Xaml.Controls.RelativePanel.SetAlignBottomWith

## -description
Sets the value of the [RelativePanel.AlignBottomWith](/uwp/api/windows.ui.xaml.controls.relativepanel#xaml-attached-properties) XAML attached property for the target element.

## -parameters
### -param element
The object to which the property value is written.

### -param value
The value to set. (The element to align this element's bottom edge with.)

## -remarks
This method is a utility method for the property system, and isn't used in most app scenarios. In most cases you set the [RelativePanel.AlignBottomWith](/uwp/api/windows.ui.xaml.controls.relativepanel#xaml-attached-properties) XAML attached property in XAML and won't need this method. For more info, see the [RelativePanel.AlignBottomWith](/uwp/api/windows.ui.xaml.controls.relativepanel#xaml-attached-properties) attached property.
