# Windows.UI.Xaml.Controls.ISemanticZoomInformation.SemanticZoomOwner

## -description
Gets or sets the [SemanticZoom](semanticzoom.md) owner that hosts the implementing view.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.ISemanticZoomInformation.SemanticZoomOwner](/windows/winui/api/microsoft.ui.xaml.controls.isemanticzoominformation.semanticzoomowner).

## -property-value
The [SemanticZoom](semanticzoom.md) that hosts this view.
