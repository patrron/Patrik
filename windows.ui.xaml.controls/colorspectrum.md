# Windows.UI.Xaml.Controls.ColorSpectrum

## -description

Represents a control that lets a user choose a color from a visual spectrum.

## -remarks

ColorSpectrum is a component of the [ColorPicker](colorpicker.md) control. Use ColorPicker for a complete color picking experience.
