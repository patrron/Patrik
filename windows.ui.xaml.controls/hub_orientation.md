# Windows.UI.Xaml.Controls.Hub.Orientation

## -description
Gets or sets the orientation of a [Hub](hub.md).

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.Hub.Orientation](/windows/winui/api/microsoft.ui.xaml.controls.hub.orientation).

## -xaml-syntax
```xaml
<Hub Orientation="Horizontal"/>
-or-
<Hub Orientation="Vertical"/>
```


## -property-value
One of the [Orientation](orientation.md) values. The default is **Horizontal**.
