# Windows.UI.Xaml.Controls.MediaElement.PlayToSource

## -description
Gets the information that is transmitted if the [MediaElement](mediaelement.md) is used for a "PlayTo" scenario.

## -property-value
A reference object that carries the "PlayTo" source information.
