# Windows.UI.Xaml.Controls.WebViewSettings.IsIndexedDBEnabled

## -description
Gets or sets a value that indicates whether the use of **IndexedDB** is allowed in the [WebView](webview.md).

## -property-value
**true** if **IndexedDB** is allowed in the [WebView](webview.md); otherwise, **false**. The default is **true**.
