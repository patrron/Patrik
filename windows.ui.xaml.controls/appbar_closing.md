# Windows.UI.Xaml.Controls.AppBar.Closing

## -description
Occurs when the [AppBar](appbar.md) starts to change from visible to hidden.

Equivalent WinUI event: [Microsoft.UI.Xaml.Controls.AppBar.Closing](/windows/winui/api/microsoft.ui.xaml.controls.appbar.closing).

## -xaml-syntax
```xaml
<AppBar Closing="eventhandler"/>
```
