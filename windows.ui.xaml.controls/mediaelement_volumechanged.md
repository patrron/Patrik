# Windows.UI.Xaml.Controls.MediaElement.VolumeChanged

## -description
Occurs when the value of the [Volume](mediaelement_volume.md) property changes.

Equivalent WinUI event: [Microsoft.UI.Xaml.Controls.MediaElement.VolumeChanged](/windows/winui/api/microsoft.ui.xaml.controls.mediaelement.volumechanged).
