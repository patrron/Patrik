# Windows.UI.Xaml.Controls.ItemClickEventHandler

## -description
Represents the method that will handle an [ItemClick](listviewbase_itemclick.md) event.

Equivalent WinUI delegate: [Microsoft.UI.Xaml.Controls.ItemClickEventHandler](/windows/winui/api/microsoft.ui.xaml.controls.itemclickeventhandler).

## -parameters
### -param sender
The object where the handler is attached.

### -param e
Event data for the event.
