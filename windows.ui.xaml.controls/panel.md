# Windows.UI.Xaml.Controls.Panel

## -description
Provides a base class for all [Panel](panel.md) elements. Use [Panel](panel.md) elements to position and arrange child objects in a UI page.

## -remarks
### **Panel** derived classes

[Panel](panel.md) is the parent class for several immediately derived classes. Some of these are practical layout containers/panels. Others are intermediate base classes for specialized panels.
+ [Canvas](canvas.md)
+ [Grid](grid.md)
+ [ItemsStackPanel](itemsstackpanel.md)
+ [ItemsWrapGrid](itemswrapgrid.md)
+ [RelativePanel](relativepanel.md)
+ [StackPanel](stackpanel.md)
+ [VariableSizedWrapGrid](variablesizedwrapgrid.md)
+ [VirtualizingPanel](virtualizingpanel.md)
