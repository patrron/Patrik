# Windows.UI.Xaml.Controls.WebViewNavigationStartingEventArgs.Cancel

## -description
Gets or sets a value indicating whether to cancel the [WebView](webview.md) navigation.

## -property-value
**true** to cancel the navigation; otherwise, **false**.
