# Windows.UI.Xaml.Controls.WebView.ContainsFullScreenElement

## -description
Gets a value that indicates whether the [WebView](webview.md) contains an element that supports full screen.

## -property-value
**true** if the [WebView](webview.md) contains an element that supports full screen; otherwise, **false**.
