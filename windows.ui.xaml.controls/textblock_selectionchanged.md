# Windows.UI.Xaml.Controls.TextBlock.SelectionChanged

## -description

Occurs when the text selection has changed.

Equivalent WinUI event: [Microsoft.UI.Xaml.Controls.TextBlock.SelectionChanged](/windows/winui/api/microsoft.ui.xaml.controls.textblock.selectionchanged).

## -xaml-syntax

```xaml
<TextBlockÂ SelectionChanged="eventhandler"/>
```

## -remarks

> [!IMPORTANT]
> If using a keyboard for text selection within a TextBlock, the user must first activate *Caret Browsing* (with the app in the foreground, press F7).
