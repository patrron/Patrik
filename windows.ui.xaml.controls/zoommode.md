# ZoomMode

## -description
Defines constants that specify the current mode for zoom behavior in the [ScrollViewer](scrollviewer.md) content.

## -xaml-syntax
```xaml
<objectproperty="enumMemberName" />
```


## -enum-fields
### -field Disabled:0
Zoom of content is disabled.

### -field Enabled:1
Zoom of content is enabled.
