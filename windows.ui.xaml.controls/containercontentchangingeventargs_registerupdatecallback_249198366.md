# Windows.UI.Xaml.Controls.ContainerContentChangingEventArgs.RegisterUpdateCallback

## -description
Registers the event handler to be called again during the specified phase.

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.ContainerContentChangingEventArgs.RegisterUpdateCallback](/windows/winui/api/microsoft.ui.xaml.controls.containercontentchangingeventargs.registerupdatecallback).

## -parameters
### -param callbackPhase
The phase during which the callback should occur.

### -param callback
The event handler function.
