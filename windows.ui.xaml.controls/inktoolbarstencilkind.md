# Windows.UI.Xaml.Controls.InkToolbarStencilKind

## -description
Specifies the types of [InkToolbar](inktoolbar.md) stencils.

## -enum-fields
### -field Ruler:0
A straight rule for drawing straight lines.

### -field Protractor:1
A protractor for drawing arcs and curves
