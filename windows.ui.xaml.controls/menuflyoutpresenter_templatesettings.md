# Windows.UI.Xaml.Controls.MenuFlyoutPresenter.TemplateSettings

## -description
Gets an object that provides calculated values that can be referenced as [TemplateBinding](http://msdn.microsoft.com/library/33a52463-699a-4634-9dd0-3fd62476e069) sources when defining templates for a [MenuFlyoutPresenter](menuflyoutpresenter.md) control.

## -property-value
An object that provides calculated values for templates.
