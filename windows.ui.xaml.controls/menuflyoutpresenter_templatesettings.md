# Windows.UI.Xaml.Controls.MenuFlyoutPresenter.TemplateSettings

## -description
Gets an object that provides calculated values that can be referenced as [TemplateBinding](/previous-versions/windows/apps/hh758288(v=win.10)) sources when defining templates for a [MenuFlyoutPresenter](menuflyoutpresenter.md) control.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.MenuFlyoutPresenter.TemplateSettings](/windows/winui/api/microsoft.ui.xaml.controls.menuflyoutpresenter.templatesettings).

## -property-value
An object that provides calculated values for templates.
