# Windows.UI.Xaml.Controls.CalendarDatePickerDateChangedEventArgs.NewDate

## -description

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.CalendarDatePickerDateChangedEventArgs.NewDate](/windows/winui/api/microsoft.ui.xaml.controls.calendardatepickerdatechangedeventargs.newdate).

## -property-value
The date that is currently selected in the [CalendarDatePicker](calendardatepicker.md).
