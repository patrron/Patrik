# Windows.UI.Xaml.Controls.WebViewPermissionRequest.State

## -description
Gets the current state of the permission request.

## -property-value
An enumeration value that indicates the current state of the permission request. The default is **Unknown**.

## -remarks
The default [State](webviewpermissionrequest_state.md) value is **Unknown**. Calling the [Allow](webviewpermissionrequest_allow.md), [Deny](webviewpermissionrequest_deny.md), or [Defer](webviewpermissionrequest_defer.md) method sets the [State](webviewpermissionrequest_state.md) property to the corresponding [WebViewPermissionState](webviewpermissionstate.md).
