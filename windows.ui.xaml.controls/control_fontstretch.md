# Windows.UI.Xaml.Controls.Control.FontStretch

## -description
Gets or sets the degree to which a font is condensed or expanded on the screen.

## -xaml-syntax
```xaml
<control FontStretch="fontStretchMemberName"/>
```


## -xaml-values
<dl><dt>fontStretchMemberName</dt><dd>fontStretchMemberNameA FontStretch constant name, such as Condensed, Normal, or Expanded.</dd>
</dl>
## -property-value
One of the values that specifies the degree to which a font is condensed or expanded on the screen. The default is **Normal**.
