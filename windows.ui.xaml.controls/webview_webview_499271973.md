# Windows.UI.Xaml.Controls.WebView.WebView

## -description
Initializes a new instance of the [WebView](webview.md) class with the specified execution mode.

## -parameters
### -param executionMode
A value of the enumeration that indicates whether the [WebView](webview.md) hosts content on the UI thread or a non-UI thread.
