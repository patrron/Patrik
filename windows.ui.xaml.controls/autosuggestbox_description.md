# Windows.UI.Xaml.Controls.AutoSuggestBox.Description

## -description

Gets or sets content that is shown below the control. The content should provide guidance about the input expected by the control.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.AutoSuggestBox.Description](/windows/winui/api/microsoft.ui.xaml.controls.autosuggestbox.description).

## -property-value

The content to be displayed below the control. The default is **null**.
