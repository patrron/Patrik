# Windows.UI.Xaml.Controls.ContentPresenter.TextWrapping

## -description
Gets or sets how the element wraps text.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.ContentPresenter.TextWrapping](/windows/winui/api/microsoft.ui.xaml.controls.contentpresenter.textwrapping).

## -property-value
A value that indicates how the element wraps text. The default is **NoWrap**.
