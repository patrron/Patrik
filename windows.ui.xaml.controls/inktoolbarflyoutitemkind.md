# Windows.UI.Xaml.Controls.InkToolbarFlyoutItemKind

## -description
Specifies the types of buttons that can be used on an [InkToolbar](inktoolbar.md) flyout.

## -enum-fields
### -field Simple:0
A basic button.

### -field RadioCheck:3
A radio button that can be toggled on or off.

### -field Radio:1
A basic radio button.

### -field Check:2
A check box.
