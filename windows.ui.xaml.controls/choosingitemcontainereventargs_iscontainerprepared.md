# Windows.UI.Xaml.Controls.ChoosingItemContainerEventArgs.IsContainerPrepared

## -description
Gets or sets a value that indicates whether the container is ready for use.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.ChoosingItemContainerEventArgs.IsContainerPrepared](/windows/winui/api/microsoft.ui.xaml.controls.choosingitemcontainereventargs.iscontainerprepared).

## -property-value
**true** if the container is ready for use; otherwise, **false**.
