# Windows.UI.Xaml.Controls.NavigationViewItemType

## -description

Defines constants that specify what kind of item a [NavigationViewItemData](navigationviewitemdata.md) represents.

## -enum-fields

### -field Separator:1

The item represents a line that separates navigation items.

### -field Header:2

The item represents a header for a group of navigation items.

### -field Default:0

The item represents a regular navigtation menu item.

## -remarks

This enumeration provides values for the [NavigationViewItemData.ItemType](navigationviewitemdata_itemtype.md) property.
