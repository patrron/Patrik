# Windows.UI.Xaml.Controls.ItemCollection.ReplaceAll

## -description
Initially clears the collection, then inserts the provided array as new items.

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.ItemCollection.ReplaceAll](/windows/winui/api/microsoft.ui.xaml.controls.itemcollection.replaceall).

## -parameters
### -param items
The new collection items.

## -remarks
For implementations that track a "changed" event, the initial reset fires an event, but the items added don't fire discrete per-item events.
