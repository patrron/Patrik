# Windows.UI.Xaml.Controls.NotifyEventArgs.CallingUri

## -description
Gets the Uniform Resource Identifier (URI) of the page containing the script that raised the [ScriptNotify](webview_scriptnotify.md) event.

## -property-value
The Uniform Resource Identifier (URI) of the page that raised the event.
