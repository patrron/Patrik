# Windows.UI.Xaml.Controls.WebViewPermissionRequest.Allow

## -description
Grants the requested permission.

## -remarks
When you allow the requested permission, subsequent calls to [Allow](webviewpermissionrequest_allow.md), [Deny](webviewpermissionrequest_deny.md), or [Defer](webviewpermissionrequest_defer.md) the permission request result in an error.
