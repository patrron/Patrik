# Windows.UI.Xaml.Controls.RichTextBlock.Focus

## -description
Focuses the [RichTextBlock](richtextblock.md), as if it were a conventionally focusable control.

## -parameters
### -param value
Specifies the desired target for focus state, as a value of the enumeration.

## -returns
**true** if focus was set to the [RichTextBlock](richtextblock.md), or focus was already there. **false** if the [RichTextBlock](richtextblock.md) is not focusable.
