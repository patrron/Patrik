# Windows.UI.Xaml.Controls.WebViewSettings.IsJavaScriptEnabled

## -description
Gets or sets a value that indicates whether the use of JavaScript is allowed in the [WebView](webview.md).

## -property-value
**true** if JavaScript is allowed in the [WebView](webview.md); otherwise, **false**. The default is **true**.
