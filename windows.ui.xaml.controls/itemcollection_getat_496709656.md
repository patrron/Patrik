# Windows.UI.Xaml.Controls.ItemCollection.GetAt

## -description
Returns the item located at the specified index.

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.ItemCollection.GetAt](/windows/winui/api/microsoft.ui.xaml.controls.itemcollection.getat).

## -parameters
### -param index
The integer index for the value to retrieve.

## -returns
The item value at the specified index.
