# Windows.UI.Xaml.Controls.HubSectionCollection.First

## -description
Returns the iterator for iteration over the items in the collection.

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.HubSectionCollection.First](/windows/winui/api/microsoft.ui.xaml.controls.hubsectioncollection.first).

## -returns
The iterator object. The iterator's current position is at the 0-index position, or at the collection end if the collection is empty.
