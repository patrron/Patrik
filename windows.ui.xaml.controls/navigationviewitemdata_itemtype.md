# Windows.UI.Xaml.Controls.NavigationViewItemData.ItemType

## -description

Gets or sets a value that indicates what kind of menu item this data represents.

## -property-value

A value of the enumeration that indicates what kind of menu item this data represents. The default is **Default**.
