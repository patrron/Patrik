# Windows.UI.Xaml.Controls.HsvColor

## -description

Describes a color in terms of hue, saturation, and value channels.

## -struct-fields

### -field Hue

Gets or sets the HSV hue channel value of the color.

### -field Saturation

Gets or sets the HSV saturation channel value of the color.

### -field Value

Gets or sets the HSV value channel value of the color.
