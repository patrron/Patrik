# Windows.UI.Xaml.Controls.DragItemsCompletedEventArgs.Items

## -description
Gets the loosely typed collection of objects that are selected for the item drag action.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.DragItemsCompletedEventArgs.Items](/windows/winui/api/microsoft.ui.xaml.controls.dragitemscompletedeventargs.items).

## -property-value
A loosely typed collection of objects.
