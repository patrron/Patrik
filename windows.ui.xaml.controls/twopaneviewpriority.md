# Windows.UI.Xaml.Controls.TwoPaneViewPriority

## -description

Defines constants that specify which pane has priority in a TwoPaneView.

## -enum-fields
### -field Pane2:1

Pane 2 has priority.

### -field Pane1:0

Pane 1 has priority.
