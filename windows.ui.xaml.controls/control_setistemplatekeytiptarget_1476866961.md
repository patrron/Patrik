# Windows.UI.Xaml.Controls.Control.SetIsTemplateKeyTipTarget

## -description
Sets the value of the [Control.IsTemplateKeyTipTarget](/uwp/api/windows.ui.xaml.controls.control#xaml-attached-properties) XAML attached property for a target element.

## -parameters
### -param element
The object to which the property value is written.

### -param value
The value to set.

## -remarks
This method is a utility method for the property system, and isn't used in most app scenarios. In most cases you set the **Control.IsTemplateFocusTarget** XAML attached property in XAML and won't need this method. For more info, see the [Control.IsTemplateKeyTipTarget](/uwp/api/windows.ui.xaml.controls.control#xaml-attached-properties) attached property.
