# Windows.UI.Xaml.Controls.ColumnDefinitionCollection.ReplaceAll

## -description
Initially clears the collection, then inserts the provided array as new items.

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.ColumnDefinitionCollection.ReplaceAll](/windows/winui/api/microsoft.ui.xaml.controls.columndefinitioncollection.replaceall).

## -parameters
### -param items
