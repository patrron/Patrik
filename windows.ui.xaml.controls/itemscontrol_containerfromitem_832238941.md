# Windows.UI.Xaml.Controls.ItemsControl.ContainerFromItem

## -description
Returns the container corresponding to the specified item.

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.ItemsControl.ContainerFromItem](/windows/winui/api/microsoft.ui.xaml.controls.itemscontrol.containerfromitem).

## -parameters
### -param item
The item to retrieve the container for.

## -returns
A container that corresponds to the specified item, if the item has a container and exists in the collection; otherwise, **null**.
