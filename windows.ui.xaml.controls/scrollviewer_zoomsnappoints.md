# Windows.UI.Xaml.Controls.ScrollViewer.ZoomSnapPoints

## -description
Gets the observable collection of zoom snap point factors that are held by the [ScrollViewer](scrollviewer.md).

## -property-value
A collection of [Single](https://docs.microsoft.com/dotnet/api/system.single?redirectedfrom=MSDN) values that represent zoom factors as snap points.
