# Windows.UI.Xaml.Controls.InkToolbarCustomPenButton.CustomPen

## -description

## -property-value
The [InkToolbarCustomPen](inktoolbarcustompen.md) that is enabled or disabled by the [InkToolbarCustomPenButton](inktoolbarcustompenbutton.md).
