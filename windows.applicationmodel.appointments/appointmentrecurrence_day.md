# Windows.ApplicationModel.Appointments.AppointmentRecurrence.Day

## -description
Gets or sets the day on which an appointment occurs. Day is of type [UInt32](https://docs.microsoft.com/dotnet/api/system.uint32?redirectedfrom=MSDN), has a default value of 1, and can be a value from 1 to 31.

## -property-value
The day on which an appointment occurs.
