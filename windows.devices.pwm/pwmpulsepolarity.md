# PwmPulsePolarity

## -description
Describes which polarity the PWM signal should start in.

## -enum-fields
### -field ActiveHigh:0
Configures the PWM signal to start in the active high state.

### -field ActiveLow:1
Configures the PWM signal to start in the active low state.
