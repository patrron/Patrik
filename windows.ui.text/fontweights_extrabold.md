# Windows.UI.Text.FontWeights.ExtraBold

Specifies a font weight value of 800.

Equivalent WinUI property: [Microsoft.UI.Text.FontWeights.ExtraBold](/windows/winui/api/microsoft.ui.text.fontweights.extrabold).

## -property-value
