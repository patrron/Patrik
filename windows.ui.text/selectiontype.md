# SelectionType

## -description
Specifies the type of a selection.

## -enum-fields
### -field None:0
No selection and no insertion point.

### -field InsertionPoint:1
An insertion point.

### -field Normal:2
A single nondegenerate range.

### -field InlineShape:7
An image (see [ITextRange.InsertImage](itextrange_insertimage_631868789.md)).

### -field Shape:8
A shape.
