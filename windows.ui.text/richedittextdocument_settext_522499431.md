# Windows.UI.Text.RichEditTextDocument.SetText

## -description
Sets the text of the document.

Equivalent WinUI method: [Microsoft.UI.Text.RichEditTextDocument.SetText](/windows/winui/api/microsoft.ui.text.richedittextdocument.settext).

## -parameters

### -param options

Options that control how the text is inserted into the document.

### -param value

The new text.

## -remarks

If the string is **NULL**, the text in the document is deleted.
