# Windows.UI.Text.ITextCharacterFormat.FontStyle

## -description
Gets or sets the style of the font face, such as normal or italic.

Equivalent WinUI property: [Microsoft.UI.Text.ITextCharacterFormat.FontStyle](/windows/winui/api/microsoft.ui.text.itextcharacterformat.fontstyle).

## -property-value
The font style.
