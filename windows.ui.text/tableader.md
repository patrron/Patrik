# TabLeader

## -description
The character that is used to fill the space taken by a tab character.

## -enum-fields
### -field Spaces:0
Spaces are used. This is the default.

### -field Dots:1
Dots are used.

### -field Dashes:2
A dashed line is used.

### -field Lines:3
A solid line is used.

### -field ThickLines:4
A thick line is used.

### -field Equals:5
An equal sign is used.
