# Windows.UI.Text.ContentLinkInfo.Uri

## -description

Gets or sets the Uniform Resource Identifier (URI) of the app to launch when the link is activated.

## -property-value

The Uniform Resource Identifier (URI) of the app to launch when the link is activated. The default is **null**.
