# Windows.UI.Text.FontWeights.Thin

## -description

Specifies a font weight value of 100.

Equivalent WinUI property: [Microsoft.UI.Text.FontWeights.Thin](/windows/winui/api/microsoft.ui.text.fontweights.thin).

## -property-value

A [FontWeight](fontweight.md) with a Weight value of 100.
