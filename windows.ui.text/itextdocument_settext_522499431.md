# Windows.UI.Text.ITextDocument.SetText

## -description
Sets the text of the document.

## -parameters
### -param options
Options controlling how the text is inserted into the document.

### -param value
The new text.

## -remarks
If the string is **NULL**, the text in the document is deleted.
