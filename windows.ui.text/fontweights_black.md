# Windows.UI.Text.FontWeights.Black

## -description

Specifies a font weight value of 900.

Equivalent WinUI property: [Microsoft.UI.Text.FontWeights.Black](/windows/winui/api/microsoft.ui.text.fontweights.black).

## -property-value

A [FontWeight](fontweight.md) with a Weight value of 900.
