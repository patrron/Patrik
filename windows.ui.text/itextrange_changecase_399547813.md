# Windows.UI.Text.ITextRange.ChangeCase

## -description
Changes the case of letters in a text range.

Equivalent WinUI method: [Microsoft.UI.Text.ITextRange.ChangeCase](/windows/winui/api/microsoft.ui.text.itextrange.changecase).

## -parameters
### -param value
The new case of letters in the text range. The default value is **Lower**.
