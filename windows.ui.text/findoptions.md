# FindOptions

## -description
Specifies the options to use when doing a text search.

## -enum-fields
### -field None:0
Use the default text search options; namely, use case- independent, arbitrary character boundaries.

### -field Word:2
Match whole words.

### -field Case:4
Match case; that is, a case-sensitive search.
