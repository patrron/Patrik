# DeviceAccountIconId

## -description
Specifies the type of icon to use for the device account.

## -enum-fields
### -field Exchange:0
Uses the Exchange icon.

### -field Msa:1
Uses the Microsoft account icon.

### -field Outlook:2
Uses the Outlook icon.

### -field Generic:3
Uses a generic account icon.
