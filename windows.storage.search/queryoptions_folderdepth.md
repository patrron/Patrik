# Windows.Storage.Search.QueryOptions.FolderDepth

## -description
Indicates whether the search query should produce a shallow view of the folder contents or a deep recursive view of all files and subfolder.

## -property-value
A value that indicates how deeply to query the folder. The default value is [FolderDepth.Shallow](folderdepth.md). Predefined queries typically override this property and change it to [FolderDepth.Deep](folderdepth.md).
