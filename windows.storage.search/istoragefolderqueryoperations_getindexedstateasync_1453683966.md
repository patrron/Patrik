# Windows.Storage.Search.IStorageFolderQueryOperations.GetIndexedStateAsync

## -description
Retrieves a value that indicates the indexed state of the [StorageFolder](../windows.storage/storagefolder.md) location associated with the query.

## -returns
When this method completes successfully, it returns a value of the [IndexedState](indexedstate.md) enumeration.
