# ChatRestoreHistorySpan

## -description
Indicates the chat restore history span.

## -enum-fields
### -field LastMonth:0
Only messages from last month will be restored.

### -field LastYear:1
Only messages from last year will be restored.

### -field AnyTime:2
All messages will be restored.
