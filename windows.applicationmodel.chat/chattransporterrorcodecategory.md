# ChatTransportErrorCodeCategory

## -description
Specifies the category of a transport error.

## -enum-fields
### -field None:0
No specific category for the error code

### -field Http:1
An HTTP error

### -field Network:2
Can't connect to the network

### -field MmsServer:3
An MMS server error
