# Windows.System.Profile.AnalyticsInfo.GetClientPropertiesAsync

## -description
Asynchronously retrieves the requested client attributes.

## -parameters
### -param attributeNames
An [IIterable](../windows.foundation.collections/iiterable_1.md) list of strings containing the client attributes to retrieve.

## -returns
Returns an [IAsyncOperation](../windows.foundation/iasyncoperation_1.md) object indicating that the task has completed and containing an [IMapView](../windows.foundation.collections/imapview_2.md) object with the requested attributes as Key/Value pairs.
