# Windows.Web.Http.Headers.HttpMethodHeaderValueCollection.IndexOf

## -description
Retrieves the index of an [HttpMethod](../windows.web.http/httpmethod.md) in the collection.

## -parameters
### -param value
The [HttpMethod](../windows.web.http/httpmethod.md) to find in the [HttpMethodHeaderValueCollection](httpmethodheadervaluecollection.md).

### -param index
The index of the [HttpMethod](../windows.web.http/httpmethod.md) in the [HttpMethodHeaderValueCollection](httpmethodheadervaluecollection.md).

If the item is found, this is the zero-based index of the item; otherwise, this parameter is 0.

## -returns
Indicates whether the item is found.

**true** if the [HttpMethod](../windows.web.http/httpmethod.md) item is found; otherwise, **false**.
