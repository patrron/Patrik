# Windows.Web.Http.Headers.HttpCookiePairHeaderValue.HttpCookiePairHeaderValue

## -description
Initializes a new instance of the [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) class. with a cookie name and a value for the cookie.

## -parameters
### -param name
A token that represents the cookie name.

### -param value
A value for the cookie.
