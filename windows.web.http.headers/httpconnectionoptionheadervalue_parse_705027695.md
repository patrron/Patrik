# Windows.Web.Http.Headers.HttpConnectionOptionHeaderValue.Parse

## -description
Converts a string to an [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) instance.

## -parameters
### -param input
A string that represents the connection information in the **Connection** HTTP header.

## -returns
An [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) instance.

## -exceptions
### E_INVALIDARG

The *input* parameter is **null** (**Nothing** in Visual Basic).

The *input* parameter is not valid connection header information.
