# Windows.Web.Http.Headers.HttpMethodHeaderValueCollection.ReplaceAll

## -description
Replaces all the [HttpMethod](../windows.web.http/httpmethod.md) items in the collection with the specified [HttpMethod](../windows.web.http/httpmethod.md) items.

## -parameters
### -param items
The [HttpMethod](../windows.web.http/httpmethod.md) items to add to the collection.
