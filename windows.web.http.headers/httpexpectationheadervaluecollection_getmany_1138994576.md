# Windows.Web.Http.Headers.HttpExpectationHeaderValueCollection.GetMany

## -description
Retrieves the [HttpExpectationHeaderValue](httpexpectationheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpExpectationHeaderValue](httpexpectationheadervalue.md) items in the [HttpExpectationHeaderValueCollection](httpexpectationheadervaluecollection.md).

### -param items
An array of [HttpExpectationHeaderValue](httpexpectationheadervalue.md) items that start at *startIndex* in the [HttpExpectationHeaderValueCollection](httpexpectationheadervaluecollection.md).

## -returns
The number of items retrieved.
