# Windows.Web.Http.Headers.HttpCredentialsHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpCredentialsHeaderValue](httpcredentialsheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param credentialsHeaderValue
The [HttpCredentialsHeaderValue](httpcredentialsheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpCredentialsHeaderValue](httpcredentialsheadervalue.md) information; otherwise, **false**.
