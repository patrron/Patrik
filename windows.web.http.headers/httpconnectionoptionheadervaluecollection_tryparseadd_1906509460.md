# Windows.Web.Http.Headers.HttpConnectionOptionHeaderValueCollection.TryParseAdd

## -description
Tries to parse and add the specified item to the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md).

## -parameters
### -param input
The item to parse and add.

## -returns
**true** if the item successfully parsed and was added; otherwise **false**.
