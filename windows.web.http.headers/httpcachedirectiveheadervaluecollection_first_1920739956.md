# Windows.Web.Http.Headers.HttpCacheDirectiveHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) items in the collection. The iterator points to the first [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) item in the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md).
