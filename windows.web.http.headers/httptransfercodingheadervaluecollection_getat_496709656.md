# Windows.Web.Http.Headers.HttpTransferCodingHeaderValueCollection.GetAt

## -description
Returns the [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the collection.

## -returns
The [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) at the specified index in the [HttpTransferCodingHeaderValueCollection](httptransfercodingheadervaluecollection.md).
