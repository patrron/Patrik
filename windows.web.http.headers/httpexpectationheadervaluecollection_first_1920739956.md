# Windows.Web.Http.Headers.HttpExpectationHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpExpectationHeaderValue](httpexpectationheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpExpectationHeaderValue](httpexpectationheadervalue.md) items in the collection. The iterator points to the first [HttpExpectationHeaderValue](httpexpectationheadervalue.md) item in the [HttpExpectationHeaderValueCollection](httpexpectationheadervaluecollection.md).
