# Windows.Web.Http.Headers.HttpChallengeHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpChallengeHeaderValue](httpchallengeheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param challengeHeaderValue
The [HttpChallengeHeaderValue](httpchallengeheadervalue.md) version of the string.

## -returns
**true** if input is valid [HttpChallengeHeaderValue](httpchallengeheadervalue.md) information; otherwise, **false**.
