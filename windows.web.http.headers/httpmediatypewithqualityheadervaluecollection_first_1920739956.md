# Windows.Web.Http.Headers.HttpMediaTypeWithQualityHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) items in the collection. The iterator points to the first [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) item in the [HttpMediaTypeWithQualityHeaderValueCollection](httpmediatypewithqualityheadervaluecollection.md).
