# Windows.Web.Http.Headers.HttpRequestHeaderCollection.Authorization

## -description
Gets or sets the [HttpCredentialsHeaderValue](httpcredentialsheadervalue.md) object that represents the value of an **Authorization** HTTP header on an HTTP request.

## -property-value
The value of an **Authorization** HTTP header on an HTTP request. A null value means that the header is absent.
