# Windows.Web.Http.Headers.HttpConnectionOptionHeaderValueCollection.GetMany

## -description
Retrieves the [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) items in the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md).

### -param items
An array of [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) items that start at *startIndex* in the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md).

## -returns
The number of [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) items retrieved.
