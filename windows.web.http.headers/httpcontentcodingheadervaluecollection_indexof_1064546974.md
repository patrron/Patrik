# Windows.Web.Http.Headers.HttpContentCodingHeaderValueCollection.IndexOf

## -description
Retrieves the index of an [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) in the collection.

## -parameters
### -param value
The [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) to find in the [HttpContentCodingHeaderValueCollection](httpcontentcodingheadervaluecollection.md).

### -param index
The index of the [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) in the [HttpContentCodingHeaderValueCollection](httpcontentcodingheadervaluecollection.md).

If the item is found, this is the zero-based index of the item; otherwise, this parameter is 0.

## -returns
Indicates whether the item is found.

**true** if the [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) item is found; otherwise, **false**.
