# Windows.Web.Http.Headers.HttpLanguageHeaderValueCollection.First

## -description
Retrieves an iterator to the first [Language](../windows.globalization/language.md) item in the collection.

## -returns
An object that can be used to enumerate the [Language](../windows.globalization/language.md) items in the collection. The iterator points to the first [Language](../windows.globalization/language.md) item in the [HttpLanguageHeaderValueCollection](httplanguageheadervaluecollection.md).
