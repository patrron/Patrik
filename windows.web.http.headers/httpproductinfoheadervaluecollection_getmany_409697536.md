# Windows.Web.Http.Headers.HttpProductInfoHeaderValueCollection.GetMany

## -description
Retrieves the [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) items in the [HttpProductInfoHeaderValueCollection](httpproductinfoheadervaluecollection.md).

### -param items
An array of [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) items that start at *startIndex* in the [HttpProductInfoHeaderValueCollection](httpproductinfoheadervaluecollection.md).

## -returns
The number of items retrieved.
