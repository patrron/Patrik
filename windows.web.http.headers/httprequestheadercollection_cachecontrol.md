# Windows.Web.Http.Headers.HttpRequestHeaderCollection.CacheControl

## -description
Gets the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md) that represents the value of a **Cache-Control** HTTP header on an HTTP request.

## -property-value
The object that represents the value of a **Cache-Control** HTTP header. An empty collection means that the header is absent.
