# Windows.Web.Http.Headers.HttpTransferCodingHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param transferCodingHeaderValue
The [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) information; otherwise, **false**.
