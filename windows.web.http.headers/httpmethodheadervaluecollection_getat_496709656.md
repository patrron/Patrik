# Windows.Web.Http.Headers.HttpMethodHeaderValueCollection.GetAt

## -description
Returns the [HttpMethod](../windows.web.http/httpmethod.md) at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the collection.

## -returns
The [HttpMethod](../windows.web.http/httpmethod.md) at the specified index in the [HttpMethodHeaderValueCollection](httpmethodheadervaluecollection.md).
