# Windows.Web.Http.Headers.HttpProductInfoHeaderValue.Comment

## -description
Gets the product comment from the [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) used in the **User-Agent** HTTP header.

## -property-value
The product comment used in the **User-Agent** HTTP header.
