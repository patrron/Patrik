# Windows.Web.Http.Headers.HttpContentCodingHeaderValueCollection.GetAt

## -description
Returns the [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the [HttpContentCodingHeaderValueCollection](httpcontentcodingheadervaluecollection.md).

## -returns
The [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) at the specified index in the [HttpContentCodingHeaderValueCollection](httpcontentcodingheadervaluecollection.md).
