# Windows.Web.Http.Headers.HttpExpectationHeaderValueCollection.GetAt

## -description
Returns the [HttpExpectationHeaderValue](httpexpectationheadervalue.md) at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the collection.

## -returns
The [HttpExpectationHeaderValue](httpexpectationheadervalue.md) at the specified index in the [HttpExpectationHeaderValueCollection](httpexpectationheadervaluecollection.md).
