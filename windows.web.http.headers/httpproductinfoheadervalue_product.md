# Windows.Web.Http.Headers.HttpProductInfoHeaderValue.Product

## -description
Gets the product from the [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) used in the **User-Agent** HTTP header.

## -property-value
The product name and version used in the **User-Agent** HTTP header.
