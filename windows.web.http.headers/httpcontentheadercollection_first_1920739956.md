# Windows.Web.Http.Headers.HttpContentHeaderCollection.First

## -description
Retrieves an iterator to the first item in the [HttpContentHeaderCollection](httpcontentheadercollection.md).

## -returns
An object that can be used to enumerate the items in the collection. The iterator points to the first item in the [HttpContentHeaderCollection](httpcontentheadercollection.md).
