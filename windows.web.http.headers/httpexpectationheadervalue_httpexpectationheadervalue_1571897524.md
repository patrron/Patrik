# Windows.Web.Http.Headers.HttpExpectationHeaderValue.HttpExpectationHeaderValue

## -description
Initializes a new instance of the [HttpExpectationHeaderValue](httpexpectationheadervalue.md) class with a name and value for the name.

## -parameters
### -param name
A token that represents a name used in the **Expect** HTTP header.

### -param value
A value for the name used in the **Expect** HTTP header.
