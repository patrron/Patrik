# Windows.Web.Http.Headers.HttpLanguageHeaderValueCollection.IndexOf

## -description
Retrieves the index of a [Language](../windows.globalization/language.md) in the collection.

## -parameters
### -param value
The item to find in the [HttpLanguageHeaderValueCollection](httplanguageheadervaluecollection.md).

### -param index
The index of the [Language](../windows.globalization/language.md) item in the [HttpLanguageHeaderValueCollection](httplanguageheadervaluecollection.md).

If the item is found, this is the zero-based index of the item; otherwise, this parameter is 0.

## -returns
Indicates whether the item is found.

**true** if the [Language](../windows.globalization/language.md) item is found; otherwise, **false**.
