# Windows.Web.Http.Headers.HttpRequestHeaderCollection.AcceptLanguage

## -description
Gets the [HttpLanguageRangeWithQualityHeaderValueCollection](httplanguagerangewithqualityheadervaluecollection.md) of [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) objects that represent the value of an **Accept-Language** HTTP header on an HTTP request.

## -property-value
The collection of [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) objects that represent the value of an **Accept-Language** HTTP header. An empty collection means that the header is absent.
