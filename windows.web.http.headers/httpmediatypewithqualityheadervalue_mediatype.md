# Windows.Web.Http.Headers.HttpMediaTypeWithQualityHeaderValue.MediaType

## -description
Gets or sets the **media-type** of the content to accept that is used in the **Accept** HTTP header.

## -property-value
The **media-type** of the entity-body.
