# Windows.Web.Http.Headers.HttpChallengeHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpChallengeHeaderValue](httpchallengeheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpChallengeHeaderValue](httpchallengeheadervalue.md) items in the collection. The iterator points to the first [HttpChallengeHeaderValue](httpchallengeheadervalue.md) item in the [HttpChallengeHeaderValueCollection](httpchallengeheadervaluecollection.md).
