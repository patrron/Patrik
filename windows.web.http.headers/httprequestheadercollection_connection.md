# Windows.Web.Http.Headers.HttpRequestHeaderCollection.Connection

## -description
Gets the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md) of [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) objects that represent the value of a **Connection** HTTP header on an HTTP request.

## -property-value
The collection of [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) objects that represent the HTTP **Connection** header. An empty collection means that the header is absent.
