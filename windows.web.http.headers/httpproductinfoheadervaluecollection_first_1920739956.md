# Windows.Web.Http.Headers.HttpProductInfoHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) items in the collection. The iterator points to the first [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) item in the [HttpProductInfoHeaderValueCollection](httpproductinfoheadervaluecollection.md).
