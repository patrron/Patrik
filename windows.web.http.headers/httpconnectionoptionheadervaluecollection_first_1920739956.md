# Windows.Web.Http.Headers.HttpConnectionOptionHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) items in the collection. The iterator points to the first [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) item in the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md).
