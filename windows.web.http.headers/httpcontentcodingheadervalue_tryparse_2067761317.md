# Windows.Web.Http.Headers.HttpContentCodingHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param contentCodingHeaderValue
The [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) information; otherwise, **false**.
