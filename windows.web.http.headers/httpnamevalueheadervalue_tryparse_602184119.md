# Windows.Web.Http.Headers.HttpNameValueHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param nameValueHeaderValue
The [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) information; otherwise, **false**.
