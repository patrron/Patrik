# Windows.Web.Http.Headers.HttpCookiePairHeaderValueCollection.IndexOf

## -description
Retrieves the index of an [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) in the collection.

## -parameters
### -param value
The [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) to find in the [HttpCookiePairHeaderValueCollection](httpcookiepairheadervaluecollection.md).

### -param index
The index of the [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) in the [HttpCookiePairHeaderValueCollection](httpcookiepairheadervaluecollection.md).

If the item is found, this is the zero-based index of the item; otherwise, this parameter is 0.

## -returns
Indicates whether the item is found.

**true** if the [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) item is found; otherwise, **false**.
