# Windows.Web.Http.Headers.HttpMediaTypeHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpMediaTypeHeaderValue](httpmediatypeheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param mediaTypeHeaderValue
The [HttpMediaTypeHeaderValue](httpmediatypeheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpMediaTypeHeaderValue](httpmediatypeheadervalue.md) information; otherwise, **false**.
