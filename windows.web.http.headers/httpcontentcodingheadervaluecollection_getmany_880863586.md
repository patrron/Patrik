# Windows.Web.Http.Headers.HttpContentCodingHeaderValueCollection.GetMany

## -description
Retrieves the [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) items in the [HttpContentCodingHeaderValueCollection](httpcontentcodingheadervaluecollection.md).

### -param items
An array of [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) items that start at *startIndex* in the [HttpContentCodingHeaderValueCollection](httpcontentcodingheadervaluecollection.md).

## -returns
The number of [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) items retrieved.
