# Windows.Web.Http.Headers.HttpConnectionOptionHeaderValue.TryParse

## -parameters
### -param input
The string to validate.

### -param connectionOptionHeaderValue
The [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) version of the string.

## -returns
