# Windows.Web.Http.Headers.HttpLanguageHeaderValueCollection.GetAt

## -description
Returns the [Language](../windows.globalization/language.md) item at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the collection.

## -returns
The [Language](../windows.globalization/language.md) item at the specified index in the [HttpLanguageHeaderValueCollection](httplanguageheadervaluecollection.md).
