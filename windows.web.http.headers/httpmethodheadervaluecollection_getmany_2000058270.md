# Windows.Web.Http.Headers.HttpMethodHeaderValueCollection.GetMany

## -description
Retrieves the [HttpMethod](../windows.web.http/httpmethod.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpMethod](../windows.web.http/httpmethod.md) items in the [HttpMethodHeaderValueCollection](httpmethodheadervaluecollection.md).

### -param items
An array of [HttpMethod](../windows.web.http/httpmethod.md) items that start at *startIndex* in the [HttpMethodHeaderValueCollection](httpmethodheadervaluecollection.md).

## -returns
The number of items retrieved.
