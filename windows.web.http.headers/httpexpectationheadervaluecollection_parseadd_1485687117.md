# Windows.Web.Http.Headers.HttpExpectationHeaderValueCollection.ParseAdd

## -description
Parses and adds an entry to the [HttpExpectationHeaderValueCollection](httpexpectationheadervaluecollection.md).

## -parameters
### -param input
The entry to add.

## -exceptions
### E_INVALIDARG

The *input* parameter is **null** (**Nothing** in Visual Basic).

The *input* parameter is not a valid value for the **Expect** HTTP header.
