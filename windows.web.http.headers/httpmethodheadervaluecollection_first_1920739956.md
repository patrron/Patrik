# Windows.Web.Http.Headers.HttpMethodHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpMethod](../windows.web.http/httpmethod.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpMethod](../windows.web.http/httpmethod.md) items in the collection. The iterator points to the first [HttpMethod](../windows.web.http/httpmethod.md) item in the [HttpMethodHeaderValueCollection](httpmethodheadervaluecollection.md).
