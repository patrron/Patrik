# Windows.Web.Http.Headers.HttpCookiePairHeaderValueCollection.GetMany

## -description
Retrieves the [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) items in the [HttpCookiePairHeaderValueCollection](httpcookiepairheadervaluecollection.md).

### -param items
An array of [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) items that start at *startIndex* in the [HttpCookiePairHeaderValueCollection](httpcookiepairheadervaluecollection.md).

## -returns
The number of [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) items retrieved.
