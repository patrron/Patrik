# Windows.Web.Http.Headers.HttpLanguageRangeWithQualityHeaderValueCollection.GetMany

## -description
Retrieves the [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) items in the [HttpLanguageRangeWithQualityHeaderValueCollection](httplanguagerangewithqualityheadervaluecollection.md).

### -param items
An array of [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) items that start at *startIndex* in the [HttpLanguageRangeWithQualityHeaderValueCollection](httplanguagerangewithqualityheadervaluecollection.md).

## -returns
The number of items retrieved.
