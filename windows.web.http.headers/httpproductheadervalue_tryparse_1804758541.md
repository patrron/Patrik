# Windows.Web.Http.Headers.HttpProductHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpProductHeaderValue](httpproductheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param productHeaderValue
The [HttpProductHeaderValue](httpproductheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpProductHeaderValue](httpproductheadervalue.md) information; otherwise, **false**.
