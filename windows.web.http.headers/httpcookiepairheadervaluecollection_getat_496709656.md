# Windows.Web.Http.Headers.HttpCookiePairHeaderValueCollection.GetAt

## -description
Returns the [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the collection.

## -returns
The [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) at the specified index in the [HttpCookiePairHeaderValueCollection](httpcookiepairheadervaluecollection.md).
