# Windows.Web.Http.Headers.HttpContentRangeHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpContentRangeHeaderValue](httpcontentrangeheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param contentRangeHeaderValue
The [HttpContentRangeHeaderValue](httpcontentrangeheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpContentRangeHeaderValue](httpcontentrangeheadervalue.md) information; otherwise, **false**.
