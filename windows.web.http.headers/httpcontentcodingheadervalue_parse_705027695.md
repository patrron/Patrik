# Windows.Web.Http.Headers.HttpContentCodingHeaderValue.Parse

## -description
Converts a string to an [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) instance.

## -parameters
### -param input
A string that represents the content coding information in the **Content-Encoding** HTTP header.

## -returns
An [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) instance.

## -remarks
Below are the exceptions that this function throws.

### E_INVALIDARG

The *input* parameter is **null** (**Nothing** in Visual Basic).

The *input* parameter is not valid content coding information.
