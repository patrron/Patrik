# Windows.Web.Http.Headers.HttpConnectionOptionHeaderValueCollection.IndexOf

## -description
Retrieves the index of an [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) in the collection.

## -parameters
### -param value
The [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) to find in the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md).

### -param index
The index of the [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) in the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md).

If the item is found, this is the zero-based index of the item; otherwise, this parameter is 0.

## -returns
Indicates whether the item is found.

**true** if the [HttpConnectionOptionHeaderValue](httpconnectionoptionheadervalue.md) item is found; otherwise, **false**.
