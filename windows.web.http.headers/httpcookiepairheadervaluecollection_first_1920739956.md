# Windows.Web.Http.Headers.HttpCookiePairHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) items in the collection. The iterator points to the first [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) item in the [HttpCookiePairHeaderValueCollection](httpcookiepairheadervaluecollection.md).
