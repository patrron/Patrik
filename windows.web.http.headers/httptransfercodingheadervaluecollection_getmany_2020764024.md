# Windows.Web.Http.Headers.HttpTransferCodingHeaderValueCollection.GetMany

## -description
Retrieves the [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) items in the [HttpTransferCodingHeaderValueCollection](httptransfercodingheadervaluecollection.md).

### -param items
An array of [HttpTransferCodingHeaderValue](httptransfercodingheadervalue.md) items that start at *startIndex* in the [HttpTransferCodingHeaderValueCollection](httptransfercodingheadervaluecollection.md).

## -returns
The number of items retrieved.
