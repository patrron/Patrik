# Windows.Web.Http.Headers.HttpCredentialsHeaderValue.HttpCredentialsHeaderValue

## -description
Initializes a new instance of the [HttpCredentialsHeaderValue](httpcredentialsheadervalue.md) class with the scheme and user token information to use for authentication.

## -parameters
### -param scheme
The scheme to use for authentication.

### -param token
The user token information to use for authentication.
