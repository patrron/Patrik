# Windows.Web.Http.Headers.HttpLanguageHeaderValueCollection.ReplaceAll

## -description
Replaces all the [Language](../windows.globalization/language.md) items in the collection with the specified [Language](../windows.globalization/language.md) items.

## -parameters
### -param items
The [Language](../windows.globalization/language.md) items to add to the collection.
