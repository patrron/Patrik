# Windows.Web.Http.Headers.HttpMediaTypeWithQualityHeaderValueCollection.GetMany

## -description
Retrieves the [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) items in the [HttpMediaTypeWithQualityHeaderValueCollection](httpmediatypewithqualityheadervaluecollection.md).

### -param items
An array of [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) items that start at *startIndex* in the [HttpMediaTypeWithQualityHeaderValueCollection](httpmediatypewithqualityheadervaluecollection.md).

## -returns
The number of items retrieved.
