# Windows.Web.Http.Headers.HttpRequestHeaderCollection.Cookie

## -description
Gets the [HttpCookiePairHeaderValueCollection](httpcookiepairheadervaluecollection.md) of [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) objects that represent the value of an **Cookie** HTTP header sent on an HTTP request.

## -property-value
The collection of [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) objects that represent the HTTP cookies in a **Cookie** HTTP header. An empty collection means that the header is absent.
