# Windows.Web.Http.Headers.HttpCacheDirectiveHeaderValueCollection.GetAt

## -description
Returns the [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md).

## -returns
The [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) at the specified index in the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md).
