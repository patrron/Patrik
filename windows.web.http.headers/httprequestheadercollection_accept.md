# Windows.Web.Http.Headers.HttpRequestHeaderCollection.Accept

## -description
Gets the [HttpMediaTypeWithQualityHeaderValueCollection](httpmediatypewithqualityheadervaluecollection.md) of [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) objects that represent the value of an **Accept** HTTP header on an HTTP request.

## -property-value
The collection of [HttpMediaTypeWithQualityHeaderValue](httpmediatypewithqualityheadervalue.md) objects that represent the value of an **Accept** HTTP header. An empty collection means that the header is absent.
