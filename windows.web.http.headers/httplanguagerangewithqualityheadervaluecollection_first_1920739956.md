# Windows.Web.Http.Headers.HttpLanguageRangeWithQualityHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) items in the collection. The iterator points to the first [HttpLanguageRangeWithQualityHeaderValue](httplanguagerangewithqualityheadervalue.md) item in the [HttpLanguageRangeWithQualityHeaderValueCollection](httplanguagerangewithqualityheadervaluecollection.md).
