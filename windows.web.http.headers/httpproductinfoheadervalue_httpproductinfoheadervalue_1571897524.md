# Windows.Web.Http.Headers.HttpProductInfoHeaderValue.HttpProductInfoHeaderValue

## -description
Initializes a new instance of the [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) class with a product name and version.

## -parameters
### -param productName
The name of the product token used in the **User-Agent** HTTP header.

### -param productVersion
The version of the product token used in the **User-Agent** HTTP header.
