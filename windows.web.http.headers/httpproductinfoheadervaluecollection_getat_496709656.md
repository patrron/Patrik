# Windows.Web.Http.Headers.HttpProductInfoHeaderValueCollection.GetAt

## -description
Returns the [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) at the specified index in the collection.

## -parameters
### -param index
The zero-based index of a specified item in the collection.

## -returns
The [HttpProductInfoHeaderValue](httpproductinfoheadervalue.md) at the specified index in the [HttpProductInfoHeaderValueCollection](httpproductinfoheadervaluecollection.md).
