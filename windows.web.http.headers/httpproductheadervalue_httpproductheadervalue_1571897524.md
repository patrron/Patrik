# Windows.Web.Http.Headers.HttpProductHeaderValue.HttpProductHeaderValue

## -description
Initializes a new instance of the [HttpProductHeaderValue](httpproductheadervalue.md) class with a product name and a product version.

## -parameters
### -param productName
The name of the product token used in the **User-Agent** HTTP header.

### -param productVersion
The version of the product token used in the **User-Agent** HTTP header.
