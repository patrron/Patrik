# Windows.Web.Http.Headers.HttpContentCodingHeaderValueCollection.First

## -description
Retrieves an iterator to the first [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) item in the collection.

## -returns
An object that can be used to enumerate the [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) items in the collection. The iterator points to the first [HttpContentCodingHeaderValue](httpcontentcodingheadervalue.md) item in the [HttpContentCodingHeaderValueCollection](httpcontentcodingheadervaluecollection.md).
