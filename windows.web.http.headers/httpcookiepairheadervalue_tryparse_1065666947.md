# Windows.Web.Http.Headers.HttpCookiePairHeaderValue.TryParse

## -description
Determines whether a string is valid [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) information.

## -parameters
### -param input
The string to validate.

### -param cookiePairHeaderValue
The [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) version of the string.

## -returns
**true** if *input* is valid [HttpCookiePairHeaderValue](httpcookiepairheadervalue.md) information; otherwise, **false**.
