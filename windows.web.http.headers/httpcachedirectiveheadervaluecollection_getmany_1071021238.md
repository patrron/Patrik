# Windows.Web.Http.Headers.HttpCacheDirectiveHeaderValueCollection.GetMany

## -description
Retrieves the [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) items in the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md).

### -param items
An array of [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) items that start at *startIndex* in the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md).

## -returns
The number of [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) items retrieved.
