# Windows.Web.Http.Headers.HttpLanguageHeaderValueCollection.GetMany

## -description
Retrieves the [Language](../windows.globalization/language.md) items that start at the specified index in the collection.

## -parameters
### -param startIndex
The zero-based index of the start of the [Language](../windows.globalization/language.md) items in the [HttpLanguageHeaderValueCollection](httplanguageheadervaluecollection.md).

### -param items
An array of [Language](../windows.globalization/language.md) items that start at *startIndex* in the [HttpLanguageHeaderValueCollection](httplanguageheadervaluecollection.md).

## -returns
The number of items retrieved.
