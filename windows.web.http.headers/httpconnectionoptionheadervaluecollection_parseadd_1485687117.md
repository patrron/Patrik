# Windows.Web.Http.Headers.HttpConnectionOptionHeaderValueCollection.ParseAdd

## -description
Parses and adds an entry to the [HttpConnectionOptionHeaderValueCollection](httpconnectionoptionheadervaluecollection.md).

## -parameters
### -param input
The entry to add.

## -remarks
Below are exceptions that this function throws.

### E_INVALIDARG
The *input* parameter is **null** (**Nothing** in Visual Basic).

The *input* parameter is not a valid value for the **Connection** HTTP header.
