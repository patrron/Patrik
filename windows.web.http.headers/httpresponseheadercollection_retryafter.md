# Windows.Web.Http.Headers.HttpResponseHeaderCollection.RetryAfter

## -description
Gets or sets the [HttpDateOrDeltaHeaderValue](httpdateordeltaheadervalue.md) object that represent the value of a **Retry-After** HTTP header on an HTTP response.

## -property-value
The object that represents the value of a **Retry-After** HTTP header on an HTTP response. A **null** value means that the header is absent.
