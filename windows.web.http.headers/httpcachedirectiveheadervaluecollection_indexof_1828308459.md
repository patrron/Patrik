# Windows.Web.Http.Headers.HttpCacheDirectiveHeaderValueCollection.IndexOf

## -description
Retrieves the index of an [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) in the collection.

## -parameters
### -param value
The [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) to find in the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md).

### -param index
The index of the [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) in the [HttpCacheDirectiveHeaderValueCollection](httpcachedirectiveheadervaluecollection.md).

If the item is found, this is the zero-based index of the item; otherwise, this parameter is 0.

## -returns
Indicates whether the item is found.

**true** if the [HttpNameValueHeaderValue](httpnamevalueheadervalue.md) item is found; otherwise, **false**.
