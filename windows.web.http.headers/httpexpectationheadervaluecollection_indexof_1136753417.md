# Windows.Web.Http.Headers.HttpExpectationHeaderValueCollection.IndexOf

## -description
Retrieves the index of an [HttpExpectationHeaderValue](httpexpectationheadervalue.md) in the collection.

## -parameters
### -param value
The [HttpExpectationHeaderValue](httpexpectationheadervalue.md) to find in the [HttpExpectationHeaderValueCollection](httpexpectationheadervaluecollection.md).

### -param index
The index of the [HttpExpectationHeaderValue](httpexpectationheadervalue.md) in the [HttpExpectationHeaderValueCollection](httpexpectationheadervaluecollection.md).

If the item is found, this is the zero-based index of the item; otherwise, this parameter is 0.

## -returns
Indicates whether the item is found.

**true** if the [HttpExpectationHeaderValue](httpexpectationheadervalue.md) item is found; otherwise, **false**.
