# Windows.UI.StartScreen.JumpListItem.CreateSeparator

## -description
Creates a jump list item that is an inert separator for a custom group within the app's jump list.

## -returns
A jump list item that can be added to an app's jump list through the [items](jumplist_items.md) property.
