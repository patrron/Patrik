# Windows.UI.StartScreen.TileMixedRealityModel.ActivationBehavior

## -description
Gets or sets what the system does when the user selects this tile. 

## -property-value
A value that specifies how the system behaviors when the user selects this tile. The default behavior is to launch the associated app.
