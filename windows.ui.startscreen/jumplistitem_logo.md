# Windows.UI.StartScreen.JumpListItem.Logo

## -description
Gets or sets the jump list item's logo.

## -property-value
The Uri for the jump list item logo. This can be specified using one of these schemes:<termdeflist><termdef><name>ms-appx:///</name>A path within the deployed app package. This path is resolved for languages and DPI plateau supported by the app.</termdef><termdef><name>ms-appdata:///local/</name>A file found in the per-user app storage.</termdef></termdeflist>
