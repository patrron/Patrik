# Windows.Media.SpeechSynthesis.SpeechSynthesisStream.GetOutputStreamAt

## -description
Retrieves an output stream at a specified location in [SpeechSynthesisStream](speechsynthesisstream.md).

## -parameters
### -param position
The location in the stream at which to begin.

## -returns
The output stream.
