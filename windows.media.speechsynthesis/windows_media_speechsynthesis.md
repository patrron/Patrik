# Windows.Media.SpeechSynthesis

## -description

Provides support for initializing and configuring a speech synthesis engine (voice) to convert a text string to an audio stream, also known as text-to-speech (TTS). Voice characteristics, pronunciation, volume, pitch, rate or speed, emphasis, and so on are customized through [](http://go.microsoft.com/fwlink/p/?LinkID=201763).
