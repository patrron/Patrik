# Windows.Media.SpeechSynthesis.SpeechSynthesisStream.ContentType

## -description
Gets the MIME type of the content of [SpeechSynthesisStream](speechsynthesisstream.md).

## -property-value
The MIME type (audio/wav) of the stream.
