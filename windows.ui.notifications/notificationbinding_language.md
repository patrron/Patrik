# Windows.UI.Notifications.NotificationBinding.Language

## -description
Gets or sets the language for this notification binding.

## -property-value
The language of the text content. See the [National Language Support (NLS) API Reference](/openspecs/windows_protocols/ms-lcid/a9eac961-e77d-41a6-90a5-ce1a8b0cdb9c) for a list of valid values.
