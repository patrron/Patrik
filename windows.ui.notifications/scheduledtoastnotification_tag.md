# Windows.UI.Notifications.ScheduledToastNotification.Tag

## -description
Gets or sets a string that uniquely identifies a toast notification inside a [Group](scheduledtoastnotification_group.md).

## -property-value
The unique identifier for this notification within a [Group](scheduledtoastnotification_group.md).
