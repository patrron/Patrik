# Windows.UI.Notifications.ToastNotificationManagerForUser.CreateToastNotifier

## -description
Creates a new [ToastNotifier](toastnotifier.md) for the calling application and the bound [user](toastnotificationmanagerforuser_user.md) that lets you raise a toast notification.

## -returns
The object you will use to send the toast notification to the app for the bound [user](toastnotificationmanagerforuser_user.md).
