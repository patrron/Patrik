# Windows.UI.Notifications.TileUpdater.EnableNotificationQueueForWide310x150

## -description
Enables the tile to queue up to five notifications on the wide tile.

## -parameters
### -param enable
**True** to enable queuing on this tile size; otherwise **false**.
