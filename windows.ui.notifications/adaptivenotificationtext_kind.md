# Windows.UI.Notifications.AdaptiveNotificationText.Kind

## -description
Specifies the type of content the notification contains. This property always returns [AdaptiveNotificationContentKind.Text](adaptivenotificationcontentkind.md).

## -property-value
A value that specifies the type of content the notification contains. This property always returns [AdaptiveNotificationContentKind.Text](adaptivenotificationcontentkind.md).
