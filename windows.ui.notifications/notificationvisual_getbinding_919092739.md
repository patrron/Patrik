# Windows.UI.Notifications.NotificationVisual.GetBinding

## -description
Retrieves the binding with the specified template.

## -parameters
### -param templateName
The template for the binding visual. You can use the properties of the [KnownNotificationBindings](knownnotificationbindings.md) class to retrieve the string for each of the available template types.

## -returns
The binding with the specified template, if it exists; otherwise, **null**.
