# Windows.UI.Notifications.NotificationBinding.Template

## -description
Gets or sets the template for this binding visual.

## -property-value
The template for this binding visual. You can use the properties of the [KnownNotificationBindings](knownnotificationbindings.md) class to retrieve the string for each of the available template types.
