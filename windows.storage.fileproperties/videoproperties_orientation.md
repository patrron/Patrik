# Windows.Storage.FileProperties.VideoProperties.Orientation

## -description
Gets a [VideoOrientation](videoorientation.md) value that indicates how the video should be rotated to display it correctly.

## -property-value
The value that indicates how the video should be rotated to display it correctly.
