# Windows.Storage.FileProperties.GeotagHelper.GetGeotagAsync

## -description
Asynchronously retrieves a [Geopoint](../windows.devices.geolocation/geopoint.md) representing the geographic metadata of a file, if present.

## -parameters
### -param file
The file from which the geographic metadata is retrieved.

## -returns
An asynchronous operation that returns a [Geopoint](../windows.devices.geolocation/geopoint.md) on successful completion.
