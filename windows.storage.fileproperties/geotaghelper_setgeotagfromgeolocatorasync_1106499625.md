# Windows.Storage.FileProperties.GeotagHelper.SetGeotagFromGeolocatorAsync

## -description
Asynchronously sets the geographic metadata of a file to the device's current location using the provided [Geolocator](geotaghelper_setgeotagfromgeolocatorasync_1106499625.md) object.

## -parameters
### -param file
The file into which the geographic metadata is set.

### -param geolocator
The [Geolocator](geotaghelper_setgeotagfromgeolocatorasync_1106499625.md) object that will be used to determine the device's current location.

## -returns
An asychronous action.
