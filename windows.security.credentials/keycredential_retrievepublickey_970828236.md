# Windows.Security.Credentials.KeyCredential.RetrievePublicKey

## -description
Gets the public portion of the asymmetric [KeyCredential](keycredential.md).

## -parameters
### -param blobType
The blob type of the public key to retrieve.

## -returns
The public portion of the asymmetric key credential.
