# Windows.Security.Credentials

## -description

Provides a common way to securely store and manage your passcodes, passphrases, and other identification information.

> [!NOTE]
> Enterprise credentials can't be roamed.

The [Credential locker sample](https://code.msdn.microsoft.com/windowsapps/PasswordVault-f01be74a) gives specific scenarios of storing and retrieving credentials.
