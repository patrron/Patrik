# Windows.Storage.Provider.FileUpdateRequest.File

## -description
Gets a [StorageFile](../windows.storage/storagefile.md) object that represents the locally cached copy of the file to update.

## -property-value
The [StorageFile](../windows.storage/storagefile.md) object that represents the locally cached copy of the file to update.
