# Windows.Services.Store.StoreSku.Availabilities

## -description
Gets the availabilities for the current product SKU. Each product SKU can have one or more availabilities that have different prices.

## -property-value
A collection of [StoreAvailability](storeavailability.md) objects that represent the availabilities for the current product SKU.
