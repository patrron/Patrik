# Windows.Services.Store.StoreLicense.InAppOfferToken

## -description
Gets in the product ID for the add-on.

## -property-value
The in-app offer token for the add-on, if a token exists.

## -remarks
This property returns the product ID that you gave the add-on when you created it in the Windows Dev Center dashboard.
