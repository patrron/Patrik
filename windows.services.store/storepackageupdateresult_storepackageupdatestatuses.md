# Windows.Services.Store.StorePackageUpdateResult.StorePackageUpdateStatuses

## -description
Gets info about the status of each of the package updates that are associated with the completed request.

## -property-value
A collection of [StorePackageUpdateStatus](storepackageupdatestatus.md) objects that provide info about the status of each package update that is associated with the completed request.
