# Windows.Services.Store.StoreCanAcquireLicenseResult.LicensableSku

## -description
Gets the [SKU Store ID](https://docs.microsoft.com/windows/uwp/monetize/in-app-purchases-and-trials#store-ids) of the downloadable content (DLC) add-on, if a license can be acquired for the add-on for the current user.

## -property-value
The [SKU Store ID](https://docs.microsoft.com/windows/uwp/monetize/in-app-purchases-and-trials#store-ids) of the downloadable content (DLC) add-on, if a license can be acquired for the add-on for the current user.
