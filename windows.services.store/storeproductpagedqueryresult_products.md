# Windows.Services.Store.StoreProductPagedQueryResult.Products

## -description
Gets the collection of products returned by the request.

## -property-value
A dictionary of key and value pairs, where each key is a Store ID for the product and the value is a [StoreProduct](storeproduct.md) object that represents the add-on.
