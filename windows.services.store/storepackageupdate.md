# Windows.Services.Store.StorePackageUpdate

## -description
Provides info about a package for the current app that has an update available for download from the Windows Store.

## -remarks
The [StoreContext.GetAppAndOptionalStorePackageUpdatesAsync](storecontext_getappandoptionalstorepackageupdatesasync.md) method returns objects of this type. For more information, including a code example, see [Download and install package updates for your app](https://msdn.microsoft.com/windows/uwp/packaging/self-install-package-updates).
