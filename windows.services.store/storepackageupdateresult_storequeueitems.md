# Windows.Services.Store.StorePackageUpdateResult.StoreQueueItems

## -description
Gets installation queue info about each of the package updates that are associated with the completed request.

## -property-value
A collection of [StoreQueueItem](storequeueitem.md) objects that provide installation queue info about each of the package updates that are associated with the completed request.
