# Windows.Graphics.DirectX.DirectXPrimitiveTopology

## -description
A Windows Runtime equivalent of the [D3D_PRIMITIVE_TOPOLOGY](/windows/desktop/api/D3DCommon/ne-d3dcommon-d3d_primitive_topology) enumeration.

## -enum-fields
### -field Undefined:0

### -field PointList:1

### -field LineList:2

### -field LineStrip:3

### -field TriangleList:4

### -field TriangleStrip:5

## -remarks

Refer to [D3D_PRIMITIVE_TOPOLOGY](/windows/desktop/api/D3DCommon/ne-d3dcommon-d3d_primitive_topology) for descriptions of these topologies.
