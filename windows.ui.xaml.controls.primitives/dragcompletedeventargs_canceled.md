# Windows.UI.Xaml.Controls.Primitives.DragCompletedEventArgs.Canceled

## -description
Gets a value that indicates whether the drag operation was canceled.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.Primitives.DragCompletedEventArgs.Canceled](/windows/winui/api/microsoft.ui.xaml.controls.primitives.dragcompletedeventargs.canceled).

## -property-value
**true** if the drag operation was canceled; otherwise, **false**.
