# Windows.UI.Xaml.Controls.Primitives.DragCompletedEventHandler

## -description
Represents the method that will handle the [DragCompleted](thumb_dragcompleted.md) event of a [Thumb](thumb.md).

Equivalent WinUI delegate: [Microsoft.UI.Xaml.Controls.Primitives.DragCompletedEventHandler](/windows/winui/api/microsoft.ui.xaml.controls.primitives.dragcompletedeventhandler).

## -parameters
### -param sender
The object where the event handler is attached.

### -param e
The event data.
