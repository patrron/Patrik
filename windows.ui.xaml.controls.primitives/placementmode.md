# PlacementMode

## -description
Defines constants that specify the preferred location for positioning a [ToolTip](../windows.ui.xaml.controls/tooltip.md) relative to a visual element.

## -xaml-syntax
```xaml
<objectproperty = "enumMemberName"/>

```


## -enum-fields
### -field Bottom:2
The preferred location of the [ToolTip](../windows.ui.xaml.controls/tooltip.md) is at the bottom of the target element.

### -field Left:9
The preferred location of the [ToolTip](../windows.ui.xaml.controls/tooltip.md) is at the left of the target element.

### -field Mouse:7
The preferred location of the [ToolTip](../windows.ui.xaml.controls/tooltip.md) is at the mouse pointer location.

### -field Right:4
The preferred location of the [ToolTip](../windows.ui.xaml.controls/tooltip.md) is at the right of the target element.

### -field Top:10
The preferred location of the [ToolTip](../windows.ui.xaml.controls/tooltip.md) is at the top of the target element.
