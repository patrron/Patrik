# Windows.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.GetRegularSnapPoints

## -description
Gets the distance between regular snap points for a specified orientation and alignment.

Equivalent WinUI method: [Microsoft.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.GetRegularSnapPoints](/windows/winui/api/microsoft.ui.xaml.controls.primitives.orientedvirtualizingpanel.getregularsnappoints).

## -parameters
### -param orientation
The orientation for the desired snap point set.

### -param alignment
The alignment to use when applying the snap points.

### -param offset
Out parameter. The offset of the first snap point.

## -returns
The distance between the equidistant snap points. Returns 0 when no snap points are present.
