# Windows.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.HorizontalOffset

## -description
Gets the horizontal offset of the scrolled content.

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.Primitives.OrientedVirtualizingPanel.HorizontalOffset](/windows/winui/api/microsoft.ui.xaml.controls.primitives.orientedvirtualizingpanel.horizontaloffset).

## -property-value
A value in pixels.

## -remarks
To change the horizontal offset, call [SetHorizontalOffset](orientedvirtualizingpanel_sethorizontaloffset_1971679761.md).
