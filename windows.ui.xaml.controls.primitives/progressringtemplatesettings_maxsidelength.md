# Windows.UI.Xaml.Controls.Primitives.ProgressRingTemplateSettings.MaxSideLength

Equivalent WinUI property: [Microsoft.UI.Xaml.Controls.Primitives.ProgressRingTemplateSettings.MaxSideLength](/windows/winui/api/microsoft.ui.xaml.controls.primitives.progressringtemplatesettings.maxsidelength).

## -property-value
