# Windows.Devices.Usb.UsbDeviceClasses.Physical

## -description
Gets the device class object for a device that conforms to the Physical Interface Devices (PID) specification.

## -property-value
A [UsbDeviceClass](usbdeviceclass.md) object for a PID-compliant device.
