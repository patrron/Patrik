# Windows.Devices.Usb.UsbEndpointDescriptor.TryParse

## -description
Retrieves the USB endpoint descriptor in a [UsbEndpointDescriptor](usbendpointdescriptor.md) object that is contained in a [UsbDescriptor](usbdescriptor.md) object.

## -parameters
### -param descriptor
The [UsbDescriptor](usbdescriptor.md) object to parse.

### -param parsed
Receives a [UsbEndpointDescriptor](usbendpointdescriptor.md) object.

## -returns
True, if the specified [UsbDescriptor](usbdescriptor.md) object is a USB endpoint descriptor. Otherwise, false.
