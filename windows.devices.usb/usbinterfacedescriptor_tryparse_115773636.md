# Windows.Devices.Usb.UsbInterfaceDescriptor.TryParse

## -description
Retrieves information about the alternate setting in a [UsbInterfaceDescriptor](usbinterfacedescriptor.md) object that is contained in a [UsbDescriptor](usbdescriptor.md) object.

## -parameters
### -param descriptor
The [UsbDescriptor](usbdescriptor.md) object to parse.

### -param parsed
Receives a [UsbInterfaceDescriptor](usbinterfacedescriptor.md) object.

## -returns
True, if the specified [UsbDescriptor](usbdescriptor.md) object is USB interface descriptor. Otherwise, false.
