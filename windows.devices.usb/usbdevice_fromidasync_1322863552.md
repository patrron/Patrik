# Windows.Devices.Usb.UsbDevice.FromIdAsync

## -description
Starts an asynchronous operation that creates a [UsbDevice](usbdevice.md) object.

## -parameters
### -param deviceId
The device instance path of the device. To obtain that value, get the [DeviceInformation.Id](../windows.devices.enumeration/deviceinformation_id.md) property value.

## -returns
Returns an [IAsyncOperation(UsbDevice)](../windows.foundation/iasyncoperation_1.md) object that returns the results of the operation.
