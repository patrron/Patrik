# Windows.Devices.Usb.UsbDevice.Configuration

## -description
Gets an object that represents a USB configuration including all interfaces and their endpoints.

## -property-value
A [UsbConfiguration](usbconfiguration.md) object that contains information about a USB configuration including the configuration descriptor and array of all USB interfaces in that configuration.
