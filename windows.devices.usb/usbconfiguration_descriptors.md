# Windows.Devices.Usb.UsbConfiguration.Descriptors

## -description
Gets an array of objects that represent the full set of descriptors associated with a USB configuration.

## -property-value
An array of [UsbDescriptor](usbdescriptor.md) objects that contains the full set of descriptors associated with a USB configuration.
