# Windows.Devices.Usb.UsbInterruptOutEndpointDescriptor.Pipe

## -description
Gets the object that represents the pipe that the host opens to communicate with the interrupt OUT endpoint.

## -property-value
An [UsbInterruptInPipe](usbinterruptinpipe.md) object that is the pipe for the interrupt OUT endpoint.
