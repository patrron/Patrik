# Windows.Devices.Usb.UsbEndpointDescriptor.AsInterruptOutEndpointDescriptor

## -description
Gets an object that represents the endpoint descriptor for the USB interrupt OUT endpoint.

## -property-value
An [UsbInterruptOutEndpointDescriptor](usbinterruptoutendpointdescriptor.md) object that describes the interrupt OUT endpoint.
