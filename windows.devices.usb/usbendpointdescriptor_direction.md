# Windows.Devices.Usb.UsbEndpointDescriptor.Direction

## -description
Gets the direction of the USB endpoint.

## -property-value
A [UsbTransferDirection](usbtransferdirection.md) value that indicates the direction of the endpoint. This value is Bit 7 of the **bEndpointAddress** field of an endpoint descriptor. For information, see Table 9-13 in the Universal Serial Bus Specification (version 2.0) or Table 9-18 in the Universal Serial Bus 3.0 Specification.
