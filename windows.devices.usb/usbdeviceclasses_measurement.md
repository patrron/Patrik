# Windows.Devices.Usb.UsbDeviceClasses.Measurement

## -description
Gets the device class object for a device that conforms to the USB test and measurement class (USBTMC).

## -property-value
A [UsbDeviceClass](usbdeviceclass.md) object for the USBTMC-compliant device.
