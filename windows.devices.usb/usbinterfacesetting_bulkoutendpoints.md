# Windows.Devices.Usb.UsbInterfaceSetting.BulkOutEndpoints

## -description
Gets an array of objects that represent descriptors associated with USB bulk OUT endpoints that are defined in the alternate setting.

## -property-value
An array of [UsbBulkOutEndpointDescriptor](usbbulkoutendpointdescriptor.md) objects.
