# Windows.Devices.Usb.UsbDeviceClasses.PersonalHealthcare

## -description
Gets the device class object for a device that conforms to the USB Personal Healthcare Device Class (PHDC).

## -property-value
A [UsbDeviceClass](usbdeviceclass.md) object for a PHDC-compliant healthcare device.
