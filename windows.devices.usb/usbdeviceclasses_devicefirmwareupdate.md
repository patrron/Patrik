# Windows.Devices.Usb.UsbDeviceClasses.DeviceFirmwareUpdate

## -description
Gets the device class object for the device that conforms to the Device Firmware Update device class.

## -property-value
A [UsbDeviceClass](usbdeviceclass.md) object for the Device Firmware Update device class.
