# Windows.Devices.Usb.UsbEndpointDescriptor.AsInterruptInEndpointDescriptor

## -description
Gets an object that represents the endpoint descriptor for the USB interrupt IN endpoint.

## -property-value
An [UsbInterruptInEndpointDescriptor](usbinterruptinendpointdescriptor.md) that describes the USB interrupt IN endpoint.
