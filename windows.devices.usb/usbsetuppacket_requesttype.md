# Windows.Devices.Usb.UsbSetupPacket.RequestType

## -description
Gets or sets the **bmRequestType** field in the setup packet of the USB control transfer. That field is represented by a [UsbControlRequestType](usbcontrolrequesttype.md) object.

## -property-value
The **bmRequestType** field in the setup packet. The values that are assigned to this field are described in Table 9.2 and section 9.3.1 of the Universal Serial Bus (USB) specification.
