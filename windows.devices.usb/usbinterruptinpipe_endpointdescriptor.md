# Windows.Devices.Usb.UsbInterruptInPipe.EndpointDescriptor

## -description
Gets the object that represents the endpoint descriptor associated with the USB interrupt IN endpoint.

## -property-value
A [UsbInterruptInEndpointDescriptor](usbinterruptinendpointdescriptor.md) object that represents the endpoint descriptor associated with the USB interrupt IN endpoint.
