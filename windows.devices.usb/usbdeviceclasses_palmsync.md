# Windows.Devices.Usb.UsbDeviceClasses.PalmSync

## -description
Gets the device class object for the device that conforms to the Palm Sync device class.

## -property-value
A [UsbDeviceClass](usbdeviceclass.md) object for the Palm Sync device class.
