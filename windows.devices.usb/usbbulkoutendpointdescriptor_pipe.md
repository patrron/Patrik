# Windows.Devices.Usb.UsbBulkOutEndpointDescriptor.Pipe

## -description
Gets the object that represents the pipe that the host opens to communicate with the bulk IN endpoint.

## -property-value
A [UsbBulkOutPipe](usbbulkoutpipe.md) object that is the pipe for the bulk OUT endpoint.
