Gets the object that represents the endpoint descriptor associated with the USB bulk OUT endpoint.

## -property-value
A [UsbBulkOutEndpointDescriptor](usbbulkoutendpointdescriptor.md) that represents the endpoint descriptor associated with the USB bulk OUT endpoint.
