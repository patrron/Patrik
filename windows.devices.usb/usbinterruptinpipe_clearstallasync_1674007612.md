# Windows.Devices.Usb.UsbInterruptInPipe.ClearStallAsync

## -description
Starts an asynchronous operation to clear a stall condition (endpoint halt) on the USB interrupt IN endpoint that is associated with the pipe.

## -returns
An [IAsyncAction](../windows.foundation/iasyncaction.md) object that is used to control the asynchronous operation.
