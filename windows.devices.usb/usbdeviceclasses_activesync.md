# Windows.Devices.Usb.UsbDeviceClasses.ActiveSync

## -description
Gets the device class object for the device that conforms to the Active Sync device class.

## -property-value
A [UsbDeviceClass](usbdeviceclass.md) object for the Active Sync device class.
