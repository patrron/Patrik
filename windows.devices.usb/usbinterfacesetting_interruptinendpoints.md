# Windows.Devices.Usb.UsbInterfaceSetting.InterruptInEndpoints

## -description
Gets an array of objects that represent descriptors associated with USB interrupt IN endpoints that are defined in the alternate setting.

## -property-value
An array of [UsbInterruptInEndpointDescriptor](usbinterruptinendpointdescriptor.md) objects.
