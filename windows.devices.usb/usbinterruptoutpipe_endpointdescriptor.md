# Windows.Devices.Usb.UsbInterruptOutPipe.EndpointDescriptor

## -description
Gets the object that represents the endpoint descriptor associated with the USB interrupt OUT endpoint.

## -property-value
A [UsbInterruptOutEndpointDescriptor](usbinterruptoutendpointdescriptor.md) object that represents the endpoint descriptor associated with the USB interrupt OUT endpoint.
