# Windows.System.RemoteSystems.RemoteSystemSessionMessageChannel.RemoteSystemSessionMessageChannel

## -description
Initializes an instance of [RemoteSystemSessionMessageChannel](RemoteSystemSessionMessageChannel.md) to manage messaging for a given remote session, specifying the reliability type of the channel.

## -parameters

### -param session
The [RemoteSystemSession](RemoteSystemSession.md) to which this messaging channel will correspond.

### -param channelName
The user-defined name of the messaging channel. It should be simple and descriptive, such as "Bob's 3D App messaging channel."

### -param reliability
A [RemoteSystemSessionMessageChannelReliability](RemoteSystemSessionMessageChannelReliability.md) value describing the reliability type of this channel.
