# Windows.System.RemoteSystems.RemoteSystemSessionCreationResult.Session

## -description
Gets the [RemoteSystemSession](RemoteSystemSession.md) object related to this creation attempt. 

## -property-value
The [RemoteSystemSession](RemoteSystemSession.md) object representing the session that was created (if the attempt was successful). If the session was not created successfully, this value is **null**.
