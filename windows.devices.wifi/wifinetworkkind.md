# WiFiNetworkKind

## -description
Describes the kinds of Wi-Fi networks.

## -enum-fields
### -field Any:0
Either an infrastructure or independent network.

### -field Infrastructure:1
An infrastructure network.

### -field Adhoc:2
An independent (IBSS) network.
