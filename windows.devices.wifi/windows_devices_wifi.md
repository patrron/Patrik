# Windows.Devices.WiFi

## -description
Provides a means to scan for nearby WiFi access points, enumerate those that are found, and connect to an access point.

Note that you must specify that your app has the [wiFiControl](http://msdn.microsoft.com/library/25b18ba5-e584-4537-9f19-bb2c8c52dfe1) capability in order to use this API.
