# Windows.Devices.Gpio.Provider.IGpioPinProvider.IsDriveModeSupported

## -description
Determines if a drive mode is supported for the pin.

## -parameters
### -param driveMode
The desired drive mode.

## -returns
True if the drive mode is supported; otherwise false.
