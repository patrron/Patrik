# Windows.Data.Xml.Dom.XmlDocument.CreateTextNode

## -description
Creates a text node that contains the supplied data.

## -parameters
### -param data
The value to be supplied to the new text object's [NodeValue](xmltext_nodevalue.md).

## -returns
The created text node.
