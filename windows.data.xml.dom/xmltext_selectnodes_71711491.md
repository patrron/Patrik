# Windows.Data.Xml.Dom.XmlText.SelectNodes

## -description
Applies the specified pattern-matching operation to this node's context and returns the list of matching nodes as an [XmlNodeList](xmlnodelist.md).

## -parameters
### -param xpath
Specifies an XPath expression.

## -returns
The collection of nodes selected by applying the given pattern-matching operation. If no nodes are selected, this method returns an empty collection.
