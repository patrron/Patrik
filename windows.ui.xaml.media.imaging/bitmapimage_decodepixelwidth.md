# Windows.UI.Xaml.Media.Imaging.BitmapImage.DecodePixelWidth

## -description
Gets or sets the width to use for image decoding operations.

## -xaml-syntax
```xaml

```


## -property-value
The width (in pixels) to use for image decoding operations.
