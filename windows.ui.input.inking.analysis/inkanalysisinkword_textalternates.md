# Windows.UI.Input.Inking.Analysis.InkAnalysisInkWord.TextAlternates

## -description
Gets the collection of strings identified as potential matches of the InkAnalysisInkWord object.

## -property-value
The recognition matches as a collection of [String](https://docs.microsoft.com/dotnet/api/system.string?redirectedfrom=MSDN) objects. The most likely candidate is topmost in the collection.
