# Windows.Security.Authentication.Web.Provider.WebProviderTokenRequest.GetApplicationTokenBindingKeyIdAsync

## -description
Gets an app token binding key Id.

## -parameters
### -param keyType
The type of token binding key.

### -param target
The target Uri for the key.

## -returns
When this method completes, it returns the token binding key Id.
