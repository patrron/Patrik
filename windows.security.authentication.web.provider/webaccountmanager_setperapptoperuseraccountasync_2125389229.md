# Windows.Security.Authentication.Web.Provider.WebAccountManager.SetPerAppToPerUserAccountAsync

## -description
Creates a per-app to per-user account link.

## -parameters
### -param perAppAccount
The per-app web account.

### -param perUserWebAccountId
The web account Id of the per-user web account to link.

## -returns
This method does not return a value.
