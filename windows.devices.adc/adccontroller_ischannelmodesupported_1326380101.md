# Windows.Devices.Adc.AdcController.IsChannelModeSupported

## -description
Verifies that the specified channel mode is supported by the controller.

## -parameters
### -param channelMode
The channel mode.

## -returns
True if the specified channel mode is supported, otherwise false.
