# Windows.Devices.Scanners.IImageScannerSourceConfiguration.IsAutoCroppingModeSupported

## -description
Determines if the scan source supports the specified [ImageScannerAutoCroppingMode](imagescannerautocroppingmode.md).

## -parameters
### -param value
The auto crop mode of the image to scan.

## -returns
True if the device supports the *value* mode; otherwise False.
