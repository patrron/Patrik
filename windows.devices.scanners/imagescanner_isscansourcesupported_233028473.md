# Windows.Devices.Scanners.ImageScanner.IsScanSourceSupported

## -description
Determines if the specified scan source is available on the scanner.

## -parameters
### -param value
The scan source.

## -returns
True if the scan source, Flatbed or Feeder, is available; otherwise False.
