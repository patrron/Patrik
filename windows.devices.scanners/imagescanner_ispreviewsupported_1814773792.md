# Windows.Devices.Scanners.ImageScanner.IsPreviewSupported

## -description
Determines if the specified scan source supports scan preview.

## -parameters
### -param scanSource
The scan source.

## -returns
Returns True if the specified scan source supports preview; otherwise False.
