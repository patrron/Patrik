# DomainNameType

## -description
The domain type of the [HostName](hostname.md) object when the [HostNameType](hostnametype.md) is a domain name.

## -enum-fields
### -field Suffix:0
The suffix of a domain name.

### -field FullyQualified:1
A fully-qualified domain name.
