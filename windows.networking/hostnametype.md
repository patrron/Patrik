# HostNameType

## -description
The type of a [HostName](hostname.md) object.

## -enum-fields
### -field DomainName:0
A domain name.

### -field Ipv4:1
An IPv4 address.

### -field Ipv6:2
An IPv6 address.

### -field Bluetooth:3
A Bluetooth address. This represents the hardware or media access control (MAC) address for a Bluetooth device.
