# Windows.Web.Http.Filters.HttpBaseProtocolFilter.AllowUI

## -description
Get or set a value that indicates whether the [HttpBaseProtocolFilter](httpbaseprotocolfilter.md) can prompt for user credentials when requested by the server.

## -property-value
A value that indicates whether [HttpBaseProtocolFilter](httpbaseprotocolfilter.md) can prompt for user credentials when requested by the server.

This value is **true** if [HttpBaseProtocolFilter](httpbaseprotocolfilter.md) can prompt for user credentials when requested; otherwise **false**. The default value is **true**.
