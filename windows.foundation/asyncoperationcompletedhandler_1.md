# Windows.Foundation.AsyncOperationCompletedHandler<TResult>

## -description
Represents a method that handles the completed event of an asynchronous operation.

## -parameters
### -param asyncInfo
The asynchronous operation.

### -param asyncStatus
One of the enumeration values.
