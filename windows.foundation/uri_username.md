# Windows.Foundation.Uri.UserName

## -description
Gets the user name component of the Uniform Resource Identifier (URI) as stored in this [Uri](uri.md) instance.

## -property-value
The user name.

## -remarks
> [!NOTE]
>  *username*:*password***@** preceding the *host* part of a URI are mainly relevant for File Transfer Protocol (FTP)  Uniform Resource Identifier (URI). File Transfer Protocol (FTP)  Uniform Resource Identifier (URI) have limited support in Windows Runtime; see "Transferring data in the background" ([HTML](http://msdn.microsoft.com/library/9e2ed5b4-af57-456a-884f-1e1d2136a8e8) or [XAML](http://msdn.microsoft.com/library/dfc4d2a2-8825-4c12-87db-9743ba3bd167)).
