# Windows.Foundation.IStringable

## -description
Provides a way to represent the current object as a string.

## -remarks
> [!IMPORTANT]
> Managed types should not implement the IStringable interface. For more info, see [Object.ToString Method](https://go.microsoft.com/fwlink/p/?LinkID=85776).
