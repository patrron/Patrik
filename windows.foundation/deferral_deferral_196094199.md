# Windows.Foundation.Deferral.Deferral

## -description
Initializes a new [Deferral](deferral.md) object and specifies a [DeferralCompletedHandler](deferralcompletedhandler.md) to be called upon completion of the deferral.

## -parameters
### -param handler
A [DeferralCompletedHandler](deferralcompletedhandler.md) to be called upon completion of the deferral.
