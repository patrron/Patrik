# Windows.Storage.AccessCache.StorageItemMostRecentlyUsedList.GetFolderAsync

## -description
Retrieves the specified [StorageFolder](../windows.storage/storagefolder.md) from the most recently used (MRU) list using the specified options.

## -parameters
### -param token
The token of the [StorageFolder](../windows.storage/storagefolder.md) to retrieve.

### -param options
The enum value that describes the behavior to use when the app accesses the item.

## -returns
When this method completes successfully, it returns the [StorageFolder](../windows.storage/storagefolder.md) that is associated with the specified *token*.
