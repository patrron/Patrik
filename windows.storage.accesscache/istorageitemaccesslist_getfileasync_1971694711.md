# Windows.Storage.AccessCache.IStorageItemAccessList.GetFileAsync

## -description
Retrieves the specified [StorageFile](../windows.storage/storagefile.md) from the list using the specified options.

## -parameters
### -param token
The token of the [StorageFile](../windows.storage/storagefile.md) to retrieve.

### -param options
The enum value that describes the behavior to use when the app accesses the item.

## -returns
When this method completes successfully, it returns the [StorageFile](../windows.storage/storagefile.md) that is associated with the specified *token*.
