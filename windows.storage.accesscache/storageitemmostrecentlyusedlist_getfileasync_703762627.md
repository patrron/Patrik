# Windows.Storage.AccessCache.StorageItemMostRecentlyUsedList.GetFileAsync

## -description
Retrieves the specified [storageFile](../windows.storage/storagefile.md) from the most recently used (MRU) list.

## -parameters
### -param token
The token of the [storageFile](../windows.storage/storagefile.md) to retrieve.

## -returns
When this method completes successfully, it returns the [storageFile](../windows.storage/storagefile.md) that is associated with the specified *token*.
