# SmartCardCryptogramStorageKeyAlgorithm

## -description
Defines cryptogram storage key algorithms.

## -enum-fields
### -field None:0
The storage key is not encrypted.

### -field Rsa2048:1
The storage key is encrypted by RSA-2048.
