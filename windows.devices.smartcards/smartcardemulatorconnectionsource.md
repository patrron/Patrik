# SmartCardEmulatorConnectionSource

## -description
Defines the valid types of smart card connection sources.

## -enum-fields
### -field Unknown:0
Specifies a connection to something other than an NFC reader.

### -field NfcReader:1
Specifies an NFC reader connection.
