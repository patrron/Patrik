# Windows.UI.Xaml.Automation.Provider.IDragProvider.IsGrabbed

## -description
Gets a value indicating whether an item is currently being dragged.

Equivalent WinUI property: [Microsoft.UI.Xaml.Automation.Provider.IDragProvider.IsGrabbed](/windows/winui/api/microsoft.ui.xaml.automation.provider.idragprovider.isgrabbed).

## -property-value
**true** if the item is being dragged. Otherwise, **false**.
