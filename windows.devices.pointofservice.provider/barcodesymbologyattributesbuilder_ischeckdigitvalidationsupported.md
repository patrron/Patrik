# Windows.Devices.PointOfService.Provider.BarcodeSymbologyAttributesBuilder.IsCheckDigitValidationSupported

## -description
Gets or sets whether the barcode symbology supports check digit validation.

## -property-value
True if the barcode symbology supports check digit validation; otherwise, false.
