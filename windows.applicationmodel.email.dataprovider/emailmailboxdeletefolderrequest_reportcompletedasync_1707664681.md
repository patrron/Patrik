# Windows.ApplicationModel.Email.DataProvider.EmailMailboxDeleteFolderRequest.ReportCompletedAsync

## -description
Informs the email client that the request was processed successfully.

Successful completion of the asynchronous operation indicates that the folder was deleted successfully.

## -returns
An asynchronous delete operation on an [EmailMailboxDeleteFolderRequest](emailmailboxdeletefolderrequest.md) object.
