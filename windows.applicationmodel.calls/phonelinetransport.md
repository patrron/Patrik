# PhoneLineTransport

## -description
The transport used by a particular phone line.

## -enum-fields
### -field Cellular:0
The phone line uses a cellular transport.

### -field VoipApp:1
The phone line uses a voice over IP transport.

### -field Bluetooth:2
The phone line uses a bluetooth transport.
