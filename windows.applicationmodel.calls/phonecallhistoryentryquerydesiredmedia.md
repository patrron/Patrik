# PhoneCallHistoryEntryQueryDesiredMedia

## -description
A filter used to retrieve call entries based on the contained media.

## -enum-fields
### -field None:0
No associated media.

### -field Audio:1
Audio media.

### -field Video:2
Video media.

### -field All:4294967295
All media types.
