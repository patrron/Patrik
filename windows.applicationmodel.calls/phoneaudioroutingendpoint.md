# PhoneAudioRoutingEndpoint

## -description
The available audio endpoints you can use for dialing a phone call.

## -enum-fields
### -field Default:0
The default audio endpoint for the phone.

### -field Bluetooth:1
A Bluetooth connection.

### -field Speakerphone:2
A speaker phone.
