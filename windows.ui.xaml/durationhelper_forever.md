# Windows.UI.Xaml.DurationHelper.Forever

## -description
 Returns a generated [Duration](duration.md) value that indicates a "Forever"  [Duration](duration.md). C# and Microsoft Visual Basic code should use [Forever](duration_forever.md) instead. 

## -property-value
A [Duration](duration.md) value that indicates a "Forever"**[Duration](duration.md).
