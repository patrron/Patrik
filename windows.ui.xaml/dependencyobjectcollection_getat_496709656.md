# Windows.UI.Xaml.DependencyObjectCollection.GetAt

## -description
Returns the [DependencyObject](dependencyobject.md) located at the specified index.

Equivalent WinUI method: [Microsoft.UI.Xaml.DependencyObjectCollection.GetAt](/windows/winui/api/microsoft.ui.xaml.dependencyobjectcollection.getat).

## -parameters
### -param index
The integer index for the value to retrieve.

## -returns
The [DependencyObject](dependencyobject.md) value at the specified index.

## -remarks
> [!NOTE]
> The equivalent Microsoft .NET  API is [Item](dependencyobjectcollection_item.md) (the indexer).
