# Windows.UI.Xaml.RectHelper.Contains

## -description
Returns whether a given [Point](../windows.foundation/point.md) is within the bounds of a given [Rect](../windows.foundation/rect.md), for a shared coordinate reference. C# and Microsoft Visual Basic code should use [Contains](../windows.foundation/rect_contains.md) instead. C# and Microsoft Visual Basic code should use the = operator instead. 
<!--Link if we ever add Rect.Equals-->

## -parameters
### -param target
The [Rect](../windows.foundation/rect.md) to evaluate.

### -param point
The [Point](../windows.foundation/point.md) to check for containment.

## -returns
**true** if *point* is within the *target* bounds; otherwise, **false**.
