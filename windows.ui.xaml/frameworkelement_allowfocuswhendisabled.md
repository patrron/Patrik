# Windows.UI.Xaml.FrameworkElement.AllowFocusWhenDisabled

## -description
Gets or sets whether a disabled control can receive focus.

Equivalent WinUI property: [Microsoft.UI.Xaml.FrameworkElement.AllowFocusWhenDisabled](/windows/winui/api/microsoft.ui.xaml.frameworkelement.allowfocuswhendisabled).

## -property-value
**true** if a disabled control can receive focus; otherwise, **false**.
