# Windows.UI.Xaml.DependencyObject.ClearValue

## -description
Clears the local value of a dependency property.

## -parameters
### -param dp
The [DependencyProperty](dependencyproperty.md) identifier of the property for which to clear the value.

## -remarks
[ClearValue](dependencyobject_clearvalue_1095854009.md) is sometimes used as part of a property-changed callback method. For more info, see [Custom dependency properties](http://msdn.microsoft.com/library/5adf7935-f2cf-4bb6-b1a5-f535c2ed8ef8).
