# Windows.UI.Xaml.RectHelper.FromCoordinatesAndDimensions

## -description
Creates a new [Rect](../windows.foundation/rect.md) value based on the element values of a [Rect](../windows.foundation/rect.md) structure. C# and Microsoft Visual Basic code should use [Rect(Double,Double,Double,Double)](../windows.foundation/rect_rect_3.md) instead.

## -parameters
### -param x
The pixel value to set for **X**.

### -param y
The pixel value to set for **Y**.

### -param width
The pixel value to set for **Width**.

### -param height
The pixel value to set for **Height**.

## -returns
The created [Rect](../windows.foundation/rect.md).
