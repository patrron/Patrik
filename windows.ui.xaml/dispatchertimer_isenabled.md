# Windows.UI.Xaml.DispatcherTimer.IsEnabled

## -description
Gets a value that indicates whether the timer is running.

Equivalent WinUI property: [Microsoft.UI.Xaml.DispatcherTimer.IsEnabled](/windows/winui/api/microsoft.ui.xaml.dispatchertimer.isenabled).

## -property-value
**true** if the timer is enabled and running; otherwise, **false**.
