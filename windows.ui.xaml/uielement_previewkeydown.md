# Windows.UI.Xaml.UIElement.PreviewKeyDown

## -description

Occurs when a keyboard key is pressed while the [UIElement](uielement.md) has focus.

## -xaml-syntax

```xaml
<uiElement PreviewKeyDown="eventhandler"/>
```
