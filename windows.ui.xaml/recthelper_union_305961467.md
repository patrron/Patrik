# Windows.UI.Xaml.RectHelper.Union

## -description
Creates a rectangle that is exactly large enough to contain the a specified rectangle and a specified point. C# and Microsoft Visual Basic code should use [Union(Point)](../windows.foundation/rect_union_1.md) instead.

## -parameters
### -param target
The rectangle to include.

### -param point
The point to include.

## -returns
A rectangle that is exactly large enough to contain the specified rectangle and point.
