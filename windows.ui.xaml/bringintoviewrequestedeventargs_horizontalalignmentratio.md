# Windows.UI.Xaml.BringIntoViewRequestedEventArgs.HorizontalAlignmentRatio

## -description
Gets the horizontal distance that should be added to the viewport-relative position of the TargetRect after satisfying the requested [HorizontalAlignmentRatio](bringintoviewrequestedeventargs_horizontalalignmentratio.md).

## -property-value

The horizontal distance that should be added to the viewport-relative position of the TargetRect after satisfying the requested [HorizontalAlignmentRatio](bringintoviewrequestedeventargs_horizontalalignmentratio.md).
