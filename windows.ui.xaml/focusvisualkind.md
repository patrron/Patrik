# FocusVisualKind

## -description
Specifies the type of visual feedback that an app can use to indicate the UI element with focus.

## -enum-fields
### -field DottedLine:0
A focus visual for legacy apps.

### -field HighVisibility:1
A focus visual that is easily discerned across devices.
