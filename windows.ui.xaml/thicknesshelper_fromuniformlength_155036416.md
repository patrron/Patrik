# Windows.UI.Xaml.ThicknessHelper.FromUniformLength

## -description
Creates a new [Thickness](thickness.md) value using a uniform value for all the element values. C# and Microsoft Visual Basic code should use [Thickness(Double)](thickness_thickness_1.md) instead.

## -parameters
### -param uniformLength
The uniform value to apply to all four of the [Thickness](thickness.md) element values.

## -returns
The created [Thickness](thickness.md).
