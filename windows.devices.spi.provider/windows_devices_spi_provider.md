# Windows.Devices.Spi.Provider

## -description
This API exposes a Provider interface and associated classes for the user to implement for a specific controller. The [SPI API](../windows.devices.spi/windows_devices_spi.md) needs a SPI Provider implementation to talk to the SPI controller.
