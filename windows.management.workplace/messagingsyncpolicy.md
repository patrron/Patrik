# MessagingSyncPolicy

## -description
An enumeration that indicates the messaging sync policy.

## -enum-fields
### -field Disallowed:0
Message syncing is disallowed.

### -field Allowed:1
Message syncing is allowed.

### -field Required:2
Message syncing is required.
