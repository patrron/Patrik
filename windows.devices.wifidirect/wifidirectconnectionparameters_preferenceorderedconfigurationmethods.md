# Windows.Devices.WiFiDirect.WiFiDirectConnectionParameters.PreferenceOrderedConfigurationMethods

## -description
Gets a list of supported configuration methods, in order of preference.

## -property-value
A list of [WiFiDirectConfigurationMethod](wifidirectconfigurationmethod.md) values representing supported configuration methods, in order of preference (most preferred first.)
