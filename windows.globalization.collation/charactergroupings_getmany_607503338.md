# Windows.Globalization.Collation.CharacterGroupings.GetMany

## -description
Returns the [CharacterGrouping](charactergrouping.md) objects that start at the specified index in the set of character groups.

## -parameters
### -param startIndex
The zero-based index of the start of the [CharacterGrouping](charactergrouping.md) objects in the set to return.

### -param items
The [CharacterGrouping](charactergrouping.md) objects in the set that start at *startIndex*.

## -returns
The number of objects returned.
