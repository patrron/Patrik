# Windows.Globalization.Collation.CharacterGroupings.GetAt

## -description
Returns the [CharacterGrouping](charactergrouping.md) at the specified index in the set of character groups.

## -parameters
### -param index
The zero-based index of the [CharacterGrouping](charactergrouping.md) in the set to return.

## -returns
The [CharacterGrouping](charactergrouping.md) at the specified index.
