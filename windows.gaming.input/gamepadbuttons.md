# GamepadButtons

## -description
Specifies the button type.

## -enum-fields
### -field None:0
No button.

### -field Menu:1
Menu button.

### -field View:2
View button.

### -field A:4
A button.

### -field B:8
B button.

### -field X:16
X button.

### -field Y:32
Y button.

### -field DPadUp:64
D-pad up.

### -field DPadDown:128
D-pad down.

### -field DPadLeft:256
D-pad left.

### -field DPadRight:512
D-pad right.

### -field LeftShoulder:1024
Left bumper.

### -field RightShoulder:2048
Right bumper.

### -field LeftThumbstick:4096
Left stick.

### -field RightThumbstick:8192
Right stick.

### -field Paddle1:16384
The first paddle.

### -field Paddle2:32768
The second paddle.

### -field Paddle3:65536
The third paddle.

### -field Paddle4:131072
The fourth paddle.
