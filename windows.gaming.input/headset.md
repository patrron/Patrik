# Windows.Gaming.Input.Headset

## -description
Contains information about an audio headset attached to a gamepad.

## -remarks
Instances of the [Headset](headset.md) class cannot be created directly; instead, instances of the [Headset](headset.md) class are retrieved through the [Gamepad.Headset](gamepad_headset.md) property or the [Gamepad.HeadsetConnected](gamepad_headsetconnected.md) event.
