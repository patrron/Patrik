# ExtendedExecutionForegroundReason

## -description
Describes the reason for the extended execution request.

## -enum-fields
### -field Unspecified:0
No reason specified.

### -field SavingData:1
To save data.

### -field BackgroundAudio:2
To play audio.

### -field Unconstrained:3
To perform any task that requires extended execution time.
