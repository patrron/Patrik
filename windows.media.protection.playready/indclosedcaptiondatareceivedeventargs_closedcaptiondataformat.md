# Windows.Media.Protection.PlayReady.INDClosedCaptionDataReceivedEventArgs.ClosedCaptionDataFormat

## -description
Gets the format of the closed caption data.

## -property-value
The closed caption data format. This value can be **ATSC**, **SCTE20**, or **Unknown**.
