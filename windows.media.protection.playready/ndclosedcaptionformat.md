# NDClosedCaptionFormat

## -description
Indicates the type of format used for closed captions.

## -enum-fields
### -field ATSC:0
ATSC digital television closed captions.

### -field SCTE20:1
SCTE 20 MPEG2 closed captions.

### -field Unknown:2
Invalid format.
