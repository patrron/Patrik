# Windows.UI.Xaml.Documents.TextElement.IsTextScaleFactorEnabled

## -description
Gets or sets whether automatic text enlargement, to reflect the system text size setting, is enabled.

Equivalent WinUI property: [Microsoft.UI.Xaml.Documents.TextElement.IsTextScaleFactorEnabled](/windows/winui/api/microsoft.ui.xaml.documents.textelement.istextscalefactorenabled).

## -xaml-syntax
```xaml
<textElement IsTextScaleFactorEnabled="bool"/>
```


## -property-value
**true** if automatic text enlargement is enabled; otherwise, **false**.
