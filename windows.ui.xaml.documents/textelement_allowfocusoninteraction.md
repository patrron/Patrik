# Windows.UI.Xaml.Documents.TextElement.AllowFocusOnInteraction

## -description
Gets or sets a value that indicates whether the element automatically gets focus when the user interacts with it.

Equivalent WinUI property: [Microsoft.UI.Xaml.Documents.TextElement.AllowFocusOnInteraction](/windows/winui/api/microsoft.ui.xaml.documents.textelement.allowfocusoninteraction).

## -xaml-syntax
```xaml
<textElementAllowFocusOnInteraction="bool"/>
```


## -property-value
**true** if the element gets focus when a user interacts with it; otherwise, **false**.
