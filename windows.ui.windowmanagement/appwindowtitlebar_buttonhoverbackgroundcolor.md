# Windows.UI.WindowManagement.AppWindowTitleBar.ButtonHoverBackgroundColor

## -description

Gets or sets the background color of a title bar button when the pointer is over it.

## -property-value

The background color of a title bar button when the pointer is over it. (See [Color](../windows.ui/color.md).)
