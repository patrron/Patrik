# Windows.UI.WindowManagement.AppWindowCloseRequestedEventArgs.Cancel

## -description

Gets or sets a value that indicates whether the close operation should be canceled.

## -property-value

**true** to cancel the close operation; otherwise, **false**. The default is **false**.
