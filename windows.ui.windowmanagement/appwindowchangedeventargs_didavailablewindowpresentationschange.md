# Windows.UI.WindowManagement.AppWindowChangedEventArgs.DidAvailableWindowPresentationsChange

## -description

Gets a value that indicates whether the available window presentations changed.

## -property-value

**true** if the available window presentations changed; otherwise, **false**.
