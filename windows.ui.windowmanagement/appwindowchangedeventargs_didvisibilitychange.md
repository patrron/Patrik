# Windows.UI.WindowManagement.AppWindowChangedEventArgs.DidVisibilityChange

## -description

Gets a value that indicates whether the AppWindow.IsVisible property changed.

## -property-value

**true** if the AppWindow.IsVisible property changed; otherwise, **false**.
