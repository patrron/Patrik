# Windows.UI.WindowManagement.AppWindowTitleBar.ButtonInactiveBackgroundColor

## -description

Gets or sets the background color of a title bar button when it's inactive.

## -property-value

The background color of a title bar button when it's inactive. (See [Color](../windows.ui/color.md).)
