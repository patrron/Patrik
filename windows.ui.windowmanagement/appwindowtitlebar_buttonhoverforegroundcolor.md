# Windows.UI.WindowManagement.AppWindowTitleBar.ButtonHoverForegroundColor

## -description

Gets or sets the foreground color of a title bar button when the pointer is over it.

## -property-value

The foreground color of a title bar button when the pointer is over it. (See [Color](../windows.ui/color.md).)
