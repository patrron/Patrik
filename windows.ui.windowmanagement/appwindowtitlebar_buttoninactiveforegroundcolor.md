# Windows.UI.WindowManagement.AppWindowTitleBar.ButtonInactiveForegroundColor

## -description

Gets or sets the foreground color of a title bar button when it's inactive.

## -property-value

The foreground color of a title bar button when it's inactive. (See [Color](../windows.ui/color.md).)
