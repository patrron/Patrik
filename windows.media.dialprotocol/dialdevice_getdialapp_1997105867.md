# Windows.Media.DialProtocol.DialDevice.GetDialApp

## -description
Creates a new [DialApp](dialapp.md) object. This method does not establish a connection to the device or validate that the app exists. That is done when any function is called on the resulting [DialApp](dialapp.md) object.

## -parameters
### -param appName
The name of the app. This becomes the [AppName](dialapp_appname.md) property of the new object.

## -returns
The [DialApp](dialapp.md) object for the app.
