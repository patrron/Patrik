# Windows.Media.DialProtocol.DialDevice.FromIdAsync

## -description
Returns a [DialDevice](dialdevice.md) object for a given a device ID (acquired from a query using the [Windows.Devices.Enumeration](../windows.devices.enumeration/windows_devices_enumeration.md) APIs).

## -parameters
### -param value
The ID of the device you want a [DialDevice](dialdevice.md) object for.

## -returns
The [DialDevice](dialdevice.md) object for the given device ID.
