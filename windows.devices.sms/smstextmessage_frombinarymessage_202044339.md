# Windows.Devices.Sms.SmsTextMessage.FromBinaryMessage

## -description
Reads a binary message and decodes it. The results are placed in a new instance of a text message.

## -parameters
### -param binaryMessage
An instance of a binary message to decode.

## -returns
A new [SmsTextMessage](smstextmessage.md) object.
