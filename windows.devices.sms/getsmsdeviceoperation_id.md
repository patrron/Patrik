# Windows.Devices.Sms.GetSmsDeviceOperation.Id

## -description
Specifies the ID of the asynchronous [SmsDevice](smsdevice.md) object retrieval operation.

## -property-value
An integer that uniquely represents the asynchronous [SmsDevice](smsdevice.md) object retrieval operation.
