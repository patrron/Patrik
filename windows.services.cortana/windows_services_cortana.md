# Windows.Services.Cortana

## -description

Provides types and members for accessing and managing the information an app shares with the Cortana service.

For example, apps that require the user to enable a particular Cortana permission can display in-app consent UI and can then grant the appropriate permissions to Cortana on behalf of the user.Â
