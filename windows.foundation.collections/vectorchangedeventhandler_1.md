# Windows.Foundation.Collections.VectorChangedEventHandler<T>

## -description
Represents the method that handles the changed event of an observable vector.

## -parameters
### -param sender
The observable vector that changed.

### -param event
The description of the change that occurred in the vector.


## -remarks
This delegate is used by the [IObservableVector.VectorChanged](iobservablevector_1_vectorchanged.md) event.
