# Windows.UI.Xaml.Media.Animation.DoubleKeyFrameCollection.First

## -description
Returns the iterator for iteration over the items in the collection.

Equivalent WinUI method: [Microsoft.UI.Xaml.Media.Animation.DoubleKeyFrameCollection.First](/windows/winui/api/microsoft.ui.xaml.media.animation.doublekeyframecollection.first).

## -returns
The iterator object. The iterator's current position is at the 0-index position, or at the collection end if the collection is empty.
