# Windows.UI.Xaml.Media.Animation.SplitOpenThemeAnimation.OpenedTarget

## -description
Gets or sets the UI element that will be clipped.

Equivalent WinUI property: [Microsoft.UI.Xaml.Media.Animation.SplitOpenThemeAnimation.OpenedTarget](/windows/winui/api/microsoft.ui.xaml.media.animation.splitopenthemeanimation.openedtarget).

## -property-value
The UI element that will be clipped.

## -remarks
Don't set this in XAML. For a XAML declaration, use [OpenedTargetName](splitopenthemeanimation_openedtargetname.md) instead.
