# Windows.Media.Protection.RebootNeededEventHandler

## -description
Represents the method that handles the data passed by the [MediaProtectionManager](mediaprotectionmanager.md) that requires a reboot.

## -parameters
### -param sender
Contains the [MediaProtectionManager](mediaprotectionmanager.md) object that requires a reboot to occur.
