# Windows.Media.Protection.ComponentLoadFailedEventArgs.Information

## -description
Returns a list of components in the [RevocationAndRenewalInformation](revocationandrenewalinformation.md) object that need to be revoked and renewed with updated components.

## -property-value
An object that contains a list of components to be revoked and renewed with updated components.
