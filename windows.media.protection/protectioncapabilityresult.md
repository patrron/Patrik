# Windows.Media.Protection.ProtectionCapabilityResult

## -description
Specifies the result of a query for hardware DRM capability support using [IsTypeSupported](protectioncapabilities_istypesupported_2024884020.md).


## -enum-fields
### -field Probably:2
The queried capabilities are probably supported.

### -field NotSupported:0
The queried capabilities are not supported.

### -field Maybe:1
The queried capabilities may be supported, but the support status is not known.
