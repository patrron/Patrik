# Windows.Media.Protection.ServiceRequestedEventHandler

## -description
Represents the method that handles the data passed by the [MediaProtectionManager](mediaprotectionmanager.md) when the resume operation is completed.

## -parameters
### -param sender
Contains the [MediaProtectionManager](mediaprotectionmanager.md) object that is requesting the service.

### -param e
Contains event data for a [MediaProtectionManager](mediaprotectionmanager_mediaprotectionmanager_1221375020.md) object when a service is requested.
