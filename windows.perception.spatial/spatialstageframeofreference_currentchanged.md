# Windows.Perception.Spatial.SpatialStageFrameOfReference.CurrentChanged

## -description
Occurs when the user changes or redefines the current spatial stage.

## -remarks
When this occurs, inspect the [Current](spatialstageframeofreference_current.md) property again to get the latest stage instance.
