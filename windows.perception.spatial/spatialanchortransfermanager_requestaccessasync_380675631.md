# Windows.Perception.Spatial.SpatialAnchorTransferManager.RequestAccessAsync

## -description
Requests access for an app to export or import spatial anchors. This requires the spatialPerception capability.

## -returns
Operation that triggers once the user has completed any consent dialog, providing the access status.
