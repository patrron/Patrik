# Windows.Networking.NetworkOperators.MobileBroadbandAntennaSar.MobileBroadbandAntennaSar

## -description
Creates a new instance of a [MobileBroadbandAntennaSar](mobilebroadbandantennasar.md).

## -parameters
### -param antennaIndex
The identifier number of the antenna, in the form of a unique, zero-based index.

### -param sarBackoffIndex
The OEM/IHV-variant value for SAR backoff, in the form of an index entry in the device OEM/IHV's mapping table.
