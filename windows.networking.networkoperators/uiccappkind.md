# UiccAppKind

## -description
Describes the kinds of UICC applications.

## -enum-fields
### -field Unknown:0
An unknown kind.

### -field MF:1
Master File application.

### -field MFSim:2
A GSM application.

### -field MFRuim:3
Removable user identity card application.

### -field USim:4
UMTS application.

### -field CSim:5
CDMA application.

### -field ISim:6
IP multimedia services application.
