# Windows.Networking.NetworkOperators.NetworkOperatorDataUsageNotificationKind

## -description
Defines constants that specify the kind of data usage change that activated a [NetworkOperatorDataUsageTrigger](../windows.applicationmodel.background/networkoperatordatausagetrigger.md).

> [!NOTE]
> This functionality is only available to mobile operator apps and UWP apps given privileged access by mobile network operators.
>
> If you want to use this API and publish your app to the Store, you will need special approval. For more information, see the **Special and restricted capabilities** section under [App capability declarations](https://docs.microsoft.com/windows/uwp/packaging/app-capability-declarations), and [Mobile Broadband](https://docs.microsoft.com/windows-hardware/drivers/mobilebroadband/index).

## -enum-fields
### -field DataUsageProgress:0
Indicates that actionable progress in data usage has occurred.
