# MobileBroadbandPinFormat

## -description
Describes the possible formats for a mobile broadband PIN.

## -enum-fields
### -field Unknown:0
No PIN format has been defined. This is normally not used.

### -field Numeric:1
The PIN is numeric.

### -field Alphanumeric:2
The PIN is alphanumeric.
