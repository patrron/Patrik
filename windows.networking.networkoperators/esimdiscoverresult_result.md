# Windows.Networking.NetworkOperators.ESimDiscoverResult.Result

## -description
Gets the result object from the eSIM profile discovery operation. This is a general operation result indicating success or failure. Also see [Kind](esimdiscoverresult_kind.md).

> [!NOTE]
> This functionality is available only to mobile operator apps and UWP apps given privileged access by mobile network operators.
>
> If you want to use this API and publish your app to the Microsoft Store, then you'll need to request special approval to use the custom capability **Microsoft.eSIMManagement_8wekyb3d8bbwe**. For more info, see [Custom capabilities](/windows/uwp/packaging/app-capability-declarations#custom-capabilities).

## -property-value
An [ESimOperationResult](esimoperationresult.md) object representing the result object.
