# Windows.Devices.HumanInterfaceDevice.HidDevice.Version

## -description
Gets the version, or revision, number for the given HID device.

## -property-value
The version number.

## -remarks
The version is retrieved from the device descriptor.

Valid values are 0 through 0xFFFF.
