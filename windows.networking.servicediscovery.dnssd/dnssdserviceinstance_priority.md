# Windows.Networking.ServiceDiscovery.Dnssd.DnssdServiceInstance.Priority

## -description
Gets or sets the priority value to assign to this service instance, as defined in [RFC 2782](https://go.microsoft.com/fwlink/?LinkId=524158). Clients use **Priority** to decide which of several equivalent service instances to attach to.

## -property-value
The priority value for this service instance. See [RFC 2782](https://go.microsoft.com/fwlink/?LinkId=524158) for details on how clients use this value.
