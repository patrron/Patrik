# Windows.Networking.ServiceDiscovery.Dnssd.DnssdServiceInstanceCollection.IndexOf

## -description
Gets a value indicating whether a given [DnssdServiceInstance](dnssdserviceinstance.md) is at the specified index in this service instance collection.

> [!NOTE]
> DnssdServiceInstanceCollection is not supported and may be altered or unavailable in the future. Instead, use the Windows.Devices.Enumeration API.

## -parameters
### -param value
The [DnssdServiceInstance](dnssdserviceinstance.md) to get the index of.

### -param index
The index, if the [DnssdServiceInstance](dnssdserviceinstance.md) is found.

## -returns
**true** if *value* is found at *index*, **false** otherwise.
