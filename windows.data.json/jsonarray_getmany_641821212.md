# Windows.Data.Json.JsonArray.GetMany

## -description
Gets all array items of type [IJsonValue](ijsonvalue.md), starting from a specified index.

## -parameters
### -param startIndex
The starting index.

### -param items
Returns items of the [IJsonValue](ijsonvalue.md).

## -returns
The index of *items* within the range of elements in an array that starts at *startIndex*.
