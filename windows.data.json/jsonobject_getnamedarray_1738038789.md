# Windows.Data.Json.JsonObject.GetNamedArray

## -description
Gets the [JsonArray](jsonarray_jsonarray_1221375020.md) value with the specified name, or the provided default value if no such named value is found.

## -parameters
### -param name
The name.

### -param defaultValue
The default value to use if the JSON property is not found.

## -returns
The [JsonArray](jsonarray.md) with the specified *name*, or if this value wasn't found, the *defaultValue* is returned.
