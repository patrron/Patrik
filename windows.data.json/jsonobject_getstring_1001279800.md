# Windows.Data.Json.JsonObject.GetString

## -description
Gets the [String](https://msdn.microsoft.com/library/system.string.aspx) if the [ValueType](ijsonvalue_valuetype.md) of the encapsulated value is [String](https://msdn.microsoft.com/library/system.string.aspx).

## -returns
The [String](https://msdn.microsoft.com/library/system.string.aspx) value.
