# VpnChannelRequestCredentialsOptions

## -description
Specifies the values defining whether a requested credential is a retry case, or should be used for single sign on.

## -enum-fields
### -field None:0
None.

### -field Retrying:1
Retrying.

### -field UseForSingleSignIn:2
Use for single sign on.
