# Windows.Networking.Vpn.IVpnCustomPromptElement.Emphasized

## -description
Gets or sets whether the UI element is emphasized (bordered) or not.

## -property-value
If the input UI element is emphasized, it is TRUE; otherwise , it is FALSE.
