# VpnChannelActivityEventType

## -description
Specifies the type of activity events.

## -enum-fields
### -field Idle:0
Mode in which no traffic is being sent over the VPN channel.

### -field Active:1
Mode in which traffic is currently being sent over the VPN channel.
