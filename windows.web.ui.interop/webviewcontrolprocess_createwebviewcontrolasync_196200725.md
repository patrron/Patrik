# Windows.Web.UI.Interop.WebViewControlProcess.CreateWebViewControlAsync

## -description
Creates a [Windows.Web.UI.Interop.WebViewControl](webviewcontrol.md) within the context of this process.

## -parameters
### -param hostWindowHandle
The parent window.

### -param bounds
The initial bounds of the window.

## -returns
An asynchronous operation that completes with a [Windows.Web.UI.Interop.WebViewControl](webviewcontrol.md).
