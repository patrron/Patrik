# Windows.UI.Popups.UICommand.Invoked

## -description
Gets or sets the handler for the event that is fired when the user selects the [UICommand](uicommand.md).

## -property-value
The event handler associated with the [UICommand](uicommand.md).

## -remarks
This property may be **null**.
