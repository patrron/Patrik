# Windows.Security.Credentials.UI.CredentialPickerOptions.CredentialSaveOption

## -description
Gets or sets the option on saving credentials.

## -property-value
The option specifies the initial state of the dialog's "Save Credential" check box. The default value is clear (unchecked). This property is ignored if the caller specifies **Hidden** in the [CredentialSaveOption](credentialsaveoption.md) enumeration.
