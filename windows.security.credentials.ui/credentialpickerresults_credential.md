# Windows.Security.Credentials.UI.CredentialPickerResults.Credential

## -description
Gets the opaque credential.

## -property-value
A byte buffer that contains the opaque credential. If the credential is not available, a valid, but empty [IBuffer](../windows.storage.streams/ibuffer.md) object is returned.
