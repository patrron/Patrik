# Windows.Security.Credentials.UI.CredentialPickerResults.CredentialSaved

## -description
Gets the status of the credential save operation.

## -property-value
Indicates whether the credential was successfully saved by the credential manager. This value is always False if the caller specified the [CallerSavesCredential](credentialpickeroptions_callersavescredential.md) property.
