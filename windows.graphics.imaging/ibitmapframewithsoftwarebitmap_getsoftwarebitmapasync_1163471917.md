# Windows.Graphics.Imaging.IBitmapFrameWithSoftwareBitmap.GetSoftwareBitmapAsync

## -description
Asynchronously retrieves a SoftwareBitmap representation of an [BitmapFrame](bitmapframe.md).

## -returns
An asynchronous operation that returns a [SoftwareBitmap](softwarebitmap.md) on successful completion.
