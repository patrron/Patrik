# Windows.Graphics.Imaging.BitmapPropertySet.Insert

## -description
Adds an item to the property set.

## -parameters
### -param key
The key to insert.

### -param value
The value to insert.

## -returns
True if the method replaces a value that already exists for the key; false if this is a new key.
