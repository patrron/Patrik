# Windows.Graphics.Imaging.BitmapFrame.GetThumbnailAsync

## -description
Asynchronously returns a stream containing the thumbnail image.

## -returns
Object that manages the asynchronous retrieval of the thumbnail data.

## -remarks
Many image formats, like JPEG and TIFF, allow a small sized thumbnail of the full image to be embedded within a frame. If the image does not contain a thumbnail, then this operation will fail with HRESULT [WINCODEC_ERR_CODECNOTHUMBNAIL](https://docs.microsoft.com/windows/desktop/wic/-wic-codec-error-codes). If it exists, the EXIF orientation flag on the frame is automatically applied to the thumbnail before it is returned.
