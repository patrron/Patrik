# Windows.Graphics.Imaging.BitmapDecoder.PixelWidth

## -description
The width of the first frame in pixels.

## -property-value
The width of the first frame in pixels.

## -remarks
Use PixelWidth for pixel dimensions of the image as it is natively stored.
