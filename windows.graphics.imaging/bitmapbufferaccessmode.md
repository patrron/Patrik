# BitmapBufferAccessMode

## -description
Specifies the access mode of a [BitmapBuffer](bitmapbuffer.md).

## -enum-fields
### -field Read:0
The buffer is read-only.

### -field ReadWrite:1
The buffer can be read from and written to.

### -field Write:2
The buffer is write-only.
