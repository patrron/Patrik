# Windows.Graphics.Imaging.BitmapTransform.InterpolationMode

## -description
Specifies the interpolation mode that is used to scale the bitmap.

## -property-value
The interpolation mode. One of the values of the [BitmapInterpolationMode](bitmapinterpolationmode.md) enumeration.
