# Windows.Graphics.Imaging.BitmapFrame.GetSoftwareBitmapAsync

## -description
Asynchronously gets a [SoftwareBitmap](softwarebitmap.md) representation of the bitmap frame.

## -returns
An asynchronous operation that returns a [SoftwareBitmap](softwarebitmap.md) on successful completion.
