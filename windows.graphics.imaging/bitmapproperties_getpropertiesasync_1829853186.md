# Windows.Graphics.Imaging.BitmapProperties.GetPropertiesAsync

## -description
Asynchronously retrieves one or more bitmap properties.

## -parameters
### -param propertiesToRetrieve
A collection of strings containing the specified property keys.

## -returns
Object that manages the asynchronous retrieval of the bitmap properties.

## -remarks
This method is identical to [BitmapPropertiesView.GetPropertiesAsync](bitmappropertiesview_getpropertiesasync_1829853186.md).
