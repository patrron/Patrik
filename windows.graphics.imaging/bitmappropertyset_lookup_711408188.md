# Windows.Graphics.Imaging.BitmapPropertySet.Lookup

## -description
Retrieves the value for the specified key.

## -parameters
### -param key
The key.

## -returns
The value, if an item with the specified key exists. Use the **HasKey** method to determine whether the key exists.
