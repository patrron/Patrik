# Windows.Networking.Proximity.PeerFinder.SupportedDiscoveryTypes

## -description
Gets a value that indicates which discovery options are available to use with the [PeerFinder](peerfinder.md) class.

## -property-value
A value that indicates which discovery options are available to use with the [PeerFinder](peerfinder.md) class.
