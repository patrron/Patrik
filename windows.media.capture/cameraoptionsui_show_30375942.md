# Windows.Media.Capture.CameraOptionsUI.Show

## -description
Displays a UI that contains options for the capture of photos, audio recordings and videos.

## -parameters
### -param mediaCapture
The [MediaCapture](mediacapture.md) object that provides methods for the capture of photos, audio recordings and videos.
