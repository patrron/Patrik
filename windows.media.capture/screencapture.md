# Windows.Media.Capture.ScreenCapture

## -description
Enables an app to capture audio and video of the contents being displayed on the device.

## -remarks
> [!IMPORTANT]
> The **ScreenCapture** feature is only supported for mobile devices running Windows Phone 8.1. This API is not supported on WindowsÂ 10.
