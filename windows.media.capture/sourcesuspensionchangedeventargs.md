# Windows.Media.Capture.SourceSuspensionChangedEventArgs

## -description
Provides information about a [SourceSuspensionChanged](screencapture_sourcesuspensionchanged.md) event which is triggered when the capture of audio or video is suspended or resumed.

## -remarks
Handle the [ScreenCapture.SourceSuspensionChanged](screencapture_sourcesuspensionchanged.md) event to obtain an instance of this class.
