# Windows.Media.Capture.ScreenCapture.GetForCurrentView

## -description
Gets the [ScreenCapture](screencapture.md) object associated with the app's current view.

## -returns
The [ScreenCapture](screencapture.md) object associated with the app's current view.

## -remarks
> [!IMPORTANT]
> The **ScreenCapture** feature is only supported for mobile devices running Windows Phone 8.1. This API is not supported on WindowsÂ 10.

Use this method to obtain an instance of the [ScreenCapture](screencapture.md) class.
