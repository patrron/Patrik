# Windows.Media.Capture.MediaCaptureVideoProfileMediaDescription.Subtype

## -description
Gets a string containing the standard name for the media encoding subtype supported by the media profile, such as "Aac", "Jpeg", or "Hevc".

## -property-value
A string containing the standard name for the media encoding subtype supported by the media profile.
