# Windows.Media.Capture.ScreenCapture.VideoSource

## -description
Gets the video source used for screen capture.

## -property-value
The video source used for screen capture.

## -remarks
> [!IMPORTANT]
> The **ScreenCapture** feature is only supported for mobile devices running Windows Phone 8.1. This API is not supported on WindowsÂ 10.

Use this property to set the [VideoSource](mediacaptureinitializationsettings_videosource.md) property of the [MediaCaptureInitializationSettings](mediacaptureinitializationsettings.md) object that you use to initialize a [MediaCapture](mediacapture.md) object.
