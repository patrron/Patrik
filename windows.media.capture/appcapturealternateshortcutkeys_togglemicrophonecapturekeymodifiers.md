# Windows.Media.Capture.AppCaptureAlternateShortcutKeys.ToggleMicrophoneCaptureKeyModifiers

## -description
Gets or sets the toggle microphone capture shortcut key modifiers.

> [!NOTE]
> This API is not available to all Windows apps. Unless your developer account is specially provisioned by Microsoft, calls to these APIs will fail at runtime.

## -property-value
The toggle microphone capture shortcut key modifiers.
