# MediaCategory

## -description
Defines values for types of media categories.

## -enum-fields
### -field Other:0
Media category is other.

### -field Communications:1
Media is intended for real-time communications.

### -field Media:2
General media.

### -field GameChat:3
Media is game chat.

### -field Speech:4
Media is speech.
