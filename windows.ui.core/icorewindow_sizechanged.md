# Windows.UI.Core.ICoreWindow.SizeChanged

## -description
Specifies the event that raises when the window size is changed.

## -remarks
This event fires when the window size changes or when one of the following properties changes:


+ [AdjacentToLeftDisplayEdge](../windows.ui.viewmanagement/applicationview_adjacenttoleftdisplayedge.md)
+ [AdjacentToRightDisplayEdge](../windows.ui.viewmanagement/applicationview_adjacenttorightdisplayedge.md)
+ [IsFullScreen](../windows.ui.viewmanagement/applicationview_isfullscreen.md)
You can use this event to listen for changes to the size or position of the current window.
