# CoreInputDeviceTypes

## -description
Specifies user input modalities. These values can be combined.

## -enum-fields
### -field None:0
No input.

### -field Touch:1
Expose touch input events.

### -field Pen:2
Expose pen input events.

### -field Mouse:4
Expose mouse input events.
