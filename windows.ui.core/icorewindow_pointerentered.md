# Windows.UI.Core.ICoreWindow.PointerEntered

## -description
Specifies the event that occurs when a pointer moves into the bounding box of the window.

## -remarks
For touch points this event will be raised before the [PointerPressed](icorewindow_pointerpressed.md) event.
