# Windows.Media.Casting.CastingDevice.DeviceInfoSupportsCastingAsync

## -description
Indicates whether the given device (a [DeviceInformation](../windows.devices.enumeration/deviceinformation.md) object) supports casting.

## -parameters
### -param device
The device you want to know about.

## -returns
True if the device supports casting; false otherwise.
