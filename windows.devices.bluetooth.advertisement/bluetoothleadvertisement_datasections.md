# Windows.Devices.Bluetooth.Advertisement.BluetoothLEAdvertisement.DataSections

## -description
Gets the list of raw data sections.

## -property-value
The list of raw data sections.

## -remarks
When a [BluetoothLEAdvertisement](bluetoothleadvertisement.md) is generated by the system, the other properties are different representations of the sections contained within this list. As such, only the DataSections property needs to be copied when switching between the different usages.
