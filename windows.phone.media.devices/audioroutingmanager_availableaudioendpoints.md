# Windows.Phone.Media.Devices.AudioRoutingManager.AvailableAudioEndpoints

## -description
Gets the list of available audio routing endpoints.

## -property-value
The list of available audio routing endpoints. When multiple endpoints are available, values from the [AvailableAudioRoutingEndpoints](availableaudioroutingendpoints.md) enumeration are combined using bitwise OR.
