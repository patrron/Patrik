# Windows.Media.Miracast.MiracastReceiverConnection.InputDevices

## -description

Gets the devices from which the Miracast Receiver can collect input and transmit to the Miracast Transmitter.

## -property-value

A [MiracastReceiverInputDevices](miracastreceiverinputdevices.md) representing the input devices.
