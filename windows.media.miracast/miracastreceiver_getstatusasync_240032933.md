# Windows.Media.Miracast.MiracastReceiver.GetStatusAsync

## -description

Asynchronously retrieves the current status of the [MiracastReceiver](miracastreceiver.md).

## -returns

An asynchronous operation that returns a [MiracastReceiverStatus](miracastreceiverstatus.md) object upon completion.
