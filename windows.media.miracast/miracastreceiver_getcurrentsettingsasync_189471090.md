# Windows.Media.Miracast.MiracastReceiver.GetCurrentSettingsAsync

## -description

Returns the settings currently used by the [MiracastReceiver](miracastreceiver.md).

## -returns

An asynchronous operation that returns a [MiracastReceiverApplySettingsResult](miracastreceiverapplysettingsresult.md) object upon completion.
