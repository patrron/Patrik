# Windows.Media.Miracast.MiracastReceiverGameControllerDevice.Mode

## -description

Specifies how game controller input is being used when transmitted to a Miracast Transmitter. Game controller input can be transmitted as-is, or it can be mapped to mouse and keyboard input.

## -property-value

The current [MiracastReceiverGameControllerDeviceUsageMode](miracastreceivergamecontrollerdeviceusagemode.md) value.
