# Windows.Perception.Automation.Monitoring.PerceptionMonitorDataReceivedEventArgs.Data

## -description
Gets the data buffer.

## -property-value
The data buffer.

## -remarks
The data buffer remains active until the [DataReceived](perceptionmonitorchannel_datareceived.md) call completes, unless a deferral is acquired.
