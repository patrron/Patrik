# Windows.Perception.Automation.Monitoring.PerceptionMonitor.Start

## -description
Start monitoring data channels.

## -remarks
Call the Start method after initializing all of the channels by handling the [DataReceived](perceptionmonitorchannel_datareceived.md) event.
