# Windows.Web.UI.WebViewControlNewWindowRequestedEventArgs.Uri

## -description
Gets the Uniform Resource Identifier (URI) of the content the [IWebViewControl](iwebviewcontrol.md) is attempting to navigate to.

## -property-value
The Uniform Resource Identifier (URI) of the content the [IWebViewControl](iwebviewcontrol.md) is attempting to navigate to.
