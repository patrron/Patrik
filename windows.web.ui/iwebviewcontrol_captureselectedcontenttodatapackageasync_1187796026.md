# Windows.Web.UI.IWebViewControl.CaptureSelectedContentToDataPackageAsync

## -description
Asynchronously gets a [DataPackage](../windows.applicationmodel.datatransfer/datapackage.md) that contains the selected content within the IWebViewControl.

## -returns
When this method completes, it returns the selected content as a [DataPackage](../windows.applicationmodel.datatransfer/datapackage.md).
