# Windows.Web.UI.WebViewControlSettings.IsJavaScriptEnabled

## -description
Gets or sets a value that indicates whether the use of JavaScript is allowed.

## -property-value
**true** if JavaScript is allowed in the [IWebViewControl](iwebviewcontrol.md); otherwise, **false**. The default is **true**.
