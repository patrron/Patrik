# Windows.Web.UI.WebViewControlNavigationStartingEventArgs.Cancel

## -description
Gets or sets a value indicating whether to cancel the [IWebViewControl](iwebviewcontrol.md) navigation.

## -property-value
**true** to cancel the navigation; otherwise, **false**.
