# Windows.Web.UI.IWebViewControl.DeferredPermissionRequests

## -description
Gets a collection of permission requests that are waiting to be granted or denied.

## -property-value
A collection of [WebViewDeferredPermissionRequest](webviewcontroldeferredpermissionrequest.md) objects that are waiting to be granted or denied.
