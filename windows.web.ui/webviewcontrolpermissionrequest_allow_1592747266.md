# Windows.Web.UI.WebViewControlPermissionRequest.Allow

## -description
Grants the requested permission.

## -remarks
When you allow the requested permission, subsequent calls to [Allow](webviewcontrolpermissionrequest_allow_1592747266.md), [Deny](webviewcontrolpermissionrequest_deny_445869776.md), or [Defer](webviewcontrolpermissionrequest_defer_1945301616.md) the permission request result in an error.
