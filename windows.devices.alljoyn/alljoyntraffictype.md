# AllJoynTrafficType

## -description
Defines values used to indicate the type of data carried in session traffic.

## -enum-fields
### -field Unknown:0
Traffic type currently unknown.

### -field Messages:1
Message traffic.

### -field RawUnreliable:2
Unreliable (lossy) byte stream.

### -field RawReliable:4
Session carries a reliable byte stream.
