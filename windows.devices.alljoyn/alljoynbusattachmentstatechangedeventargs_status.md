# Windows.Devices.AllJoyn.AllJoynBusAttachmentStateChangedEventArgs.Status

## -description
Contains a status code that provides status information for [AllJoynBusAttachment](alljoynbusattachment.md) change events. Properties used in the generation of possible values are defined by [AllJoynStatus](alljoynstatus.md).

## -property-value
The AllJoyn status code associated with the event.
