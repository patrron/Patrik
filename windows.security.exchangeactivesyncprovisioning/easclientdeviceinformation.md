# Windows.Security.ExchangeActiveSyncProvisioning.EasClientDeviceInformation

## -description

Provides the app with the ability to retrieve device information from the local device.

For a Windows 8.x sample application showing how mail clients can retrieve device information, see the [EAS policies for mail clients Windows 8.x sample](https://code.msdn.microsoft.com/windowsapps/Web-authentication-for-b9b8ed1a).
