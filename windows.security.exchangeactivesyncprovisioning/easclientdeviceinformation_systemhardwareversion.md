# Windows.Security.ExchangeActiveSyncProvisioning.EasClientDeviceInformation.SystemHardwareVersion

## -description
Returns the system hardware version of the local device.

## -property-value
The system hardware version of the local device.

## -remarks
> [!NOTE]
> The [SystemHardwareVersion](easclientdeviceinformation_systemhardwareversion.md) property is retrieved from the registry. If the registry can't be read for any reason, [SystemHardwareVersion](easclientdeviceinformation_systemhardwareversion.md) is returned as an empty string.
