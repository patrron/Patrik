# Windows.Security.Cryptography.Core.CryptographicKey.ExportPublicKey

## -description
Exports a public key to a buffer given a specified format.

## -parameters
### -param BlobType
A [CryptographicPublicKeyBlobType](cryptographicpublickeyblobtype.md) enumeration value that specifies the format of the key in the buffer. The default value is **X509SubjectPublicKeyInfo**.

## -returns
Buffer that contains the public key.
