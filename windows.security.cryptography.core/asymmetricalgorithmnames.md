# Windows.Security.Cryptography.Core.AsymmetricAlgorithmNames

## -description
Contains static properties that enable you to retrieve algorithm names that can be used in the [OpenAlgorithm](asymmetrickeyalgorithmprovider_openalgorithm.md) method of the [AsymmetricKeyAlgorithmProvider](asymmetrickeyalgorithmprovider.md) class.
