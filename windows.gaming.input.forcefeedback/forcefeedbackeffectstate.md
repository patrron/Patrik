# ForceFeedbackEffectState

## -description
The current state of a force feedback effect.

## -enum-fields
### -field Stopped:0
The effect is stopped.

### -field Running:1
The effect is running.

### -field Paused:2
The effect is paused.

### -field Faulted:3
The effect has faulted.
