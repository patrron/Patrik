# Windows.Globalization.NumberFormatting.INumberRounder.RoundDouble

## -description
Rounds a **Double** number.

## -parameters
### -param value
The **Double** value to be rounded.

## -returns
The rounded number. This result is a double precision floating point number, which is not necessarily an integer.

## -remarks
> [!NOTE]
> This method is precise only within the limits of the [Double](https://docs.microsoft.com/dotnet/api/system.double?redirectedfrom=MSDN) data type: approximately 15 digits.
