# Windows.Globalization.NumberFormatting.IncrementNumberRounder.RoundingAlgorithm

## -description
Gets or sets the rounding strategy value for this [IncrementNumberRounder](incrementnumberrounder.md) object.

## -property-value
A value of the [RoundingAlgorithm](roundingalgorithm.md) enumeration.

## -remarks
This property is initialized to a default value of [RoundingAlgorithm](roundingalgorithm.md).**RoundHalfUp**.
