# Windows.UI.ApplicationSettings.SettingsPaneCommandsRequestedEventArgs

## -description
> [!NOTE]
> [SettingsPaneCommandsRequestedEventArgs](settingspanecommandsrequestedeventargs.md) is deprecated, and might not work on all versions of Windows 10. Instead of using a [SettingsPane](settingspane.md), integrate settings options into the app experience. For more info, see [Guidelines for app settings](http://msdn.microsoft.com/library/2d765e90-3fa0-42f5-a5cb-bedc14c3f60a).

Contains arguments that are available from the event object during the [CommandsRequested](settingspane_commandsrequested.md) event.

## -remarks
This class object is obtained from the event object during the [CommandsRequested](settingspane_commandsrequested.md) event.
