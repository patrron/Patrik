# Windows.Graphics.Holographic.HolographicViewConfiguration.Kind

## -description
Gets the kind of rendering this HolographicViewConfiguration is for.

## -property-value
A [HolographicViewConfigurationKind](holographicviewconfigurationkind.md) specifying a kind of HolographicViewConfiguration for a [HolographicDisplay](HolographicDisplay.md).
