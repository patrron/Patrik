# Windows.Graphics.Holographic.HolographicViewConfiguration.IsStereo

## -description
Gets whether this HolographicViewConfiguration is stereoscopic.

## -property-value
This value is `true` if the HolographicViewConfiguration is stereoscopic, otherwise `false`.
