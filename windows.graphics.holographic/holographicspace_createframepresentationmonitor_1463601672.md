# Windows.Graphics.Holographic.HolographicSpace.CreateFramePresentationMonitor

## -description
Creates an object to monitor an independent stream of frame presentation reports from this HolographicSpace.

## -parameters
### -param maxQueuedReports
The maximum number of frame reports that can queue up without being read before reports are discarded.

## -returns
The frame presentation monitor.
