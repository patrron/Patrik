# Windows.Graphics.Holographic.HolographicViewConfiguration.IsEnabled

## -description
Gets or sets whether this HolographicViewConfiguration is enabled.

## -property-value
This value is `true` if the HolographicViewConfiguration is enabled, otherwise `false`.
