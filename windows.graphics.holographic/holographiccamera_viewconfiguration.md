# Windows.Graphics.Holographic.HolographicCamera.ViewConfiguration

## -description
Gets the [HolographicViewConfiguration](holographicviewconfiguration.md) that this [HolographicCamera](HolographicCamera.md) provides rendering for.

## -property-value
The [HolographicViewConfiguration](holographicviewconfiguration.md) that can be used to configure rendering parameters for this [HolographicCamera](HolographicCamera.md).
