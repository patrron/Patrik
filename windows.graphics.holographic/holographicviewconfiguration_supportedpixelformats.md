# Windows.Graphics.Holographic.HolographicViewConfiguration.SupportedPixelFormats

## -description
Gets a list of the supported pixel formats for this HolographicViewConfiguration.

## -property-value
A collection of [Windows.Graphics.DirectX.DirectXPixelFormat](../windows.graphics.directx/directxpixelformat.md) values, each of which can be set using the [HolographicViewConfiguration.PixelFormat](holographicviewconfiguration_pixelformat.md) property.
