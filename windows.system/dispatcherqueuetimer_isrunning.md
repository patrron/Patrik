# Windows.System.DispatcherQueueTimer.IsRunning

## -description
Indicates whether the timer is currently running.

Equivalent WinUI property: [Microsoft.System.DispatcherQueueTimer.IsRunning](/windows/winui/api/microsoft.system.dispatcherqueuetimer.isrunning).

## -property-value
**True** indicates that the timer is running; otherwise **false**
