# Windows.System.Launcher.LaunchUriForUserAsync

## -description
Starts the default multi-user app associated with the URI scheme name for the specified URI using the specified options, and provides a specific [User](user.md) as context.

## -parameters
### -param user
The user context to pass to the launched app.

### -param uri
The URI scheme.

### -param options
The launch options for the app.

## -returns
The status of the launch attempt.
