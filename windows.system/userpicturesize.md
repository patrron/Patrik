# UserPictureSize

## -description
Represents user picture sizes.

## -enum-fields
### -field Size64x64:0
The picture size is 64x64.

### -field Size208x208:1
The picture size is 208x208.

### -field Size424x424:2
The picture size is 424x424.

### -field Size1080x1080:3
The picture size is 1080x1080.
