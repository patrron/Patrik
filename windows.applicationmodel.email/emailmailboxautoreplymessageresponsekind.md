# EmailMailboxAutoReplyMessageResponseKind

## -description
Defines the encoding schema used for automatic replies.

## -enum-fields
### -field Html:0
The autoreply message is encoded as HTML.

### -field PlainText:1
The autoreply message is encoded as plain text.
