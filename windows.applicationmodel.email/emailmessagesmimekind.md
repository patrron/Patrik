# EmailMessageSmimeKind

## -description
Defines the S/MIME type of an email message.

## -enum-fields
### -field None:0
No MIME encryption or signature.

### -field ClearSigned:1
Clear signed MIME (two MIME entities).

### -field OpaqueSigned:2
Opaque signed MIME (a single MIME entity).

### -field Encrypted:3
Encrypted MIME.
