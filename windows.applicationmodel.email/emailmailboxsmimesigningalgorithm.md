# EmailMailboxSmimeSigningAlgorithm

## -description
Defines the algorithm used to sign an email.

## -enum-fields
### -field Any:0
No specific algorithm is required.

### -field Sha1:1
Use Secure Hash Algorithm 1 (160-bit).

### -field MD5:2
Use Message Digest algorithm 5 (128-bit).
