# PrintOptionStates

## -description
Specifies the print task option states.

## -enum-fields
### -field None:0
The UI for the print tasks has not been enabled.

### -field Enabled:1
The UI for the print tasks has been enabled.

### -field Constrained:2
The UI for the print tasks is visible to the user.
