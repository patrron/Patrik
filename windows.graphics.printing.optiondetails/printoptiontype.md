# PrintOptionType

## -description
Specifies the print task option types.

## -enum-fields
### -field Unknown:0
Unknown option type.

### -field Number:1
A type of option that is numerical.

### -field Text:2
A type of option that is a string or some text.

### -field ItemList:3
A type of option that is a list of items.
