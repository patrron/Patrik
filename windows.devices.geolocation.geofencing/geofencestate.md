# GeofenceState

## -description
Indicates the current state of a [Geofence](geofence.md).

## -enum-fields
### -field None:0
No flag is set.

### -field Entered:1
The device has entered the geofence area.

### -field Exited:2
The device has left the geofence area.

### -field Removed:4
The geofence was removed.
