# Windows.System.UserProfile.AdvertisingManager.GetForUser

## -description
Gets the advertising ID for the specified user.

## -parameters
### -param user
The user to get the advertising ID for.

## -returns
An instance of [AdvertisingManagerForUser](advertisingmanagerforuser.md) that associates a user ID with their advertising ID.
