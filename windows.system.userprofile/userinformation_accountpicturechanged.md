# Windows.System.UserProfile.UserInformation.AccountPictureChanged

## -description
Occurs when the user's image or name changes.

> [!IMPORTANT]
> The [UserInformation](userinformation.md) class is not supported on Windows 10 or later. Use the [User](../windows.system/user.md) class instead. See Remarks in the [UserInformation](userinformation.md) class documentation.

## -remarks
> [!IMPORTANT]
> The [UserInformation](userinformation.md) class is not supported on Windows 10 or later. Use the [User](../windows.system/user.md) class instead. See Remarks in the [UserInformation](userinformation.md) class documentation.

Use this event to monitor changes to the user's account picture.

Access to this event can be blocked by privacy settings. If access is blocked, the [AccountPictureChangeEnabled](userinformation_accountpicturechangeenabled.md) property will be false.
