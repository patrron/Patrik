# Windows.Media.Editing.MediaComposition.GeneratePreviewMediaStreamSource

## -description
Creates a new [MediaStreamSource](../windows.media.core/mediastreamsource.md) used to preview the edited media.

## -parameters
### -param scaledWidth
The width of the preview media.

### -param scaledHeight
The height of the preview media.

## -returns
The new [MediaStreamSource](../windows.media.core/mediastreamsource.md) used to preview the edited media.
