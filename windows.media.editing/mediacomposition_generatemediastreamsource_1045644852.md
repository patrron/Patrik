# Windows.Media.Editing.MediaComposition.GenerateMediaStreamSource

## -description
Creates a new [MediaStreamSource](../windows.media.core/mediastreamsource.md) using the specified [MediaEncodingProfile](../windows.media.mediaproperties/mediaencodingprofile.md).

## -parameters
### -param encodingProfile
The [MediaEncodingProfile](../windows.media.mediaproperties/mediaencodingprofile.md) used to specify the encoding properties of the new [MediaStreamSource](../windows.media.core/mediastreamsource.md).

## -returns
The newly created [MediaStreamSource](../windows.media.core/mediastreamsource.md).
