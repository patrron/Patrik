# Windows.Security.Authentication.Web.Core.WebTokenRequestResult.InvalidateCacheAsync

## -description
Invalidates the current cached [WebTokenRequestResult](webtokenrequestresult.md). Use this method if an access denied error is returned when trying to use a token.

## -returns
This method does not return an object or value.
