# Windows.Foundation.Metadata.ContractVersionAttribute.ContractVersionAttribute

## -description
Creates and initializes a new instance of the attribute.  

## -parameters

### -param contract
The type to associate with the API contract.  

### -param version
The version of the API contract.

## -remarks
This constructor applies to any type that does not have the [ApiContractAttribute](apicontractattribute.md) and indicates the API contract version in which this type was added to the specified API contract.
