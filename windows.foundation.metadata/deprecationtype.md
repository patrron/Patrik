# DeprecationType

## -description
Indicates a deprecation as **Deprecate** or **Remove**.

## -enum-fields
### -field Deprecate:0
Compilers and other tools should treat the entity as deprecated. This is the default.

### -field Remove:1
Compilers and other tools should treat the entity as removed.
