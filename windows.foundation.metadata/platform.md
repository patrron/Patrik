# Platform

## -description
Specifies the platforms that a specified type should be supported in, as used by Windows Runtime attributes and metadata.

## -enum-fields
### -field Windows:0
For use by Windows metadata.

### -field WindowsPhone:1
For use by Windows Phone metadata.
