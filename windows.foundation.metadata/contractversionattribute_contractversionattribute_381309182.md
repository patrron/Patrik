# Windows.Foundation.Metadata.ContractVersionAttribute.ContractVersionAttribute

## -description
Creates and initializes a new instance of the attribute.  

## -parameters

### -param contract
The type to associate with the API contract.  

### -param version
The version of the API contract.  

## -remarks
This constructor applies to a type with the [ApiContractAttribute](apicontractattribute.md) and specifies the contract version of that API contract.
