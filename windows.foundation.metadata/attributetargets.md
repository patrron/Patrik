# AttributeTargets

## -description
Specifies the programming construct to which an attribute applies.



> **.NET**
> This type appears as [System.AttributeTargets](https://docs.microsoft.com/dotnet/api/system.attributetargets?redirectedfrom=MSDN).

## -enum-fields
### -field All:4294967295
The attribute applies to all programming constructs.

### -field Delegate:1
The attribute applies to a delegate.

### -field Enum:2
The attribute applies to an enumeration.

### -field Event:4
The attribute applies to an event.

### -field Field:8
The attribute applies to a field.

### -field Interface:16
The attribute applies to an interface.

### -field Method:64
The attribute applies to a method.

### -field Parameter:128
The attribute applies to a parameter.

### -field Property:256
The attribute applies to a property.

### -field RuntimeClass:512
The attribute applies to a runtime class.

### -field Struct:1024
The attribute applies to a struct.

### -field InterfaceImpl:2048
The attribute applies to an implementation of an interface.

### -field ApiContract:8192
The attribute applies to an API contract.


## -remarks
When programming with .NET, this type is hidden and developers should use [System.AttributeTargets](https://docs.microsoft.com/dotnet/api/system.attributetargets?redirectedfrom=MSDN).
