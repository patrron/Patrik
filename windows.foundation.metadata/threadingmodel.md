# ThreadingModel

## -description
Specifies the threading model.

## -enum-fields
### -field STA:1
Single-threaded apartment.

### -field MTA:2
Multithreaded apartment.

### -field Both:3
Both single-threaded and multithreaded apartments.

### -field InvalidThreading:0
No valid threading model applies.
