# Windows.UI.Xaml.Media.RevealBrushState

## -description

Defines constants that specify the pointer state of an element.

## -enum-fields

### -field Pressed:2

The element is pressed.

### -field PointerOver:1

The pointer is over the element.

### -field Normal:0

The element is in its default state.
