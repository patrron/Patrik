# Windows.UI.Xaml.Media.GradientStopCollection.ReplaceAll

## -description
Initially clears the collection, then inserts the provided array as new items.

Equivalent WinUI method: [Microsoft.UI.Xaml.Media.GradientStopCollection.ReplaceAll](/windows/winui/api/microsoft.ui.xaml.media.gradientstopcollection.replaceall).

## -parameters
### -param items
The new collection items.

## -remarks
For implementations that track a "changed" event, the initial reset fires an event, but the items added do not fire discrete events.
