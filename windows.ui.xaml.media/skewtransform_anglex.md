# Windows.UI.Xaml.Media.SkewTransform.AngleX

## -description
Gets or sets the x-axis skew angle, which is measured in degrees counterclockwise from the y-axis.

Equivalent WinUI property: [Microsoft.UI.Xaml.Media.SkewTransform.AngleX](/windows/winui/api/microsoft.ui.xaml.media.skewtransform.anglex).
