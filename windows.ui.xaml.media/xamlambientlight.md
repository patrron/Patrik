# Windows.UI.Xaml.Media.XamlAmbientLight

## -description

Represents a XAML light that uses an AmbientLight to illuminate every targeted Visual equally.

## -remarks

Set the XamlAmbientLight.IsTarget attached property to **true** on an a UIElement or Brush to apply an ambient light.
