# Windows.UI.Xaml.Media.PointCollection.GetAt

## -description
Returns the item located at the specified index.

Equivalent WinUI method: [Microsoft.UI.Xaml.Media.PointCollection.GetAt](/windows/winui/api/microsoft.ui.xaml.media.pointcollection.getat).

## -parameters
### -param index
The integer index for the value to retrieve.

## -returns
The [Point](../windows.foundation/point.md) value at the specified index.
