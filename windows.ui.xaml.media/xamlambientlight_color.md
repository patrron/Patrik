# Windows.UI.Xaml.Media.XamlAmbientLight.Color

## -description

Gets or sets the color of the ambient light.

## -property-value

The color of the ambient light.

## -remarks

This property sets the corresponding Color property on the composition AmbientLight used to implement the light. This wrapper property lets you set the value in XAML markup.
