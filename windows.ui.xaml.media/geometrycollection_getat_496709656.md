# Windows.UI.Xaml.Media.GeometryCollection.GetAt

## -description

Equivalent WinUI method: [Microsoft.UI.Xaml.Media.GeometryCollection.GetAt](/windows/winui/api/microsoft.ui.xaml.media.geometrycollection.getat).

## -parameters
### -param index
The integer index for the value to retrieve.

## -returns
The value at the specified index.
