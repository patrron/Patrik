# Windows.UI.Xaml.Media.TranslateTransform.Y

## -description
Gets or sets the distance to translate (move) an object along the y-axis.

Equivalent WinUI property: [Microsoft.UI.Xaml.Media.TranslateTransform.Y](/windows/winui/api/microsoft.ui.xaml.media.translatetransform.y).
