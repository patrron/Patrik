# Windows.Web.Http.HttpResponseMessage.IsSuccessStatusCode

## -description
Gets a value that indicates whether the HTTP response was successful.

## -property-value
A value that indicates if the HTTP response was successful. true if [HttpStatusCode](httpstatuscode.md) was in the **Successful** range (200-299); otherwise false.
