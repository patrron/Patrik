# Windows.Web.Http.HttpMultipartContent.HttpMultipartContent

## -description
Initializes a new instance of the [HttpMultipartContent](httpmultipartcontent.md) class with the specified MIME subtype.

## -parameters
### -param subtype
The MIME subtype of the multipart content.

## -remarks
Below are the exceptions that this function throws.

### E_INVALIDARG

The *subtype* parameter is **null** (**Nothing** in Visual Basic).
