# Windows.Web.Http.HttpMultipartContent.Add

## -description
Add HTTP content to the [HttpMultipartContent](httpmultipartcontent.md) instance.

## -parameters
### -param content
The HTTP content to add to [HttpMultipartContent](httpmultipartcontent.md).

## -remarks
Below are the exceptions that this function throws.

### E_INVALIDARG

The *content* parameter is **null** (**Nothing** in Visual Basic).
