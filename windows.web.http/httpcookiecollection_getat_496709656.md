# Windows.Web.Http.HttpCookieCollection.GetAt

## -description
Returns the [HttpCookie](httpcookie.md) at the specified index from the [HttpCookieCollection](httpcookiecollection.md).

## -parameters
### -param index
The zero-based index of a specified item in the [HttpCookieCollection](httpcookiecollection.md).

## -returns
The HTTP cookie at the specified index from the [HttpCookieCollection](httpcookiecollection.md).
