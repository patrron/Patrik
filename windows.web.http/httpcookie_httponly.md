# Windows.Web.Http.HttpCookie.HttpOnly

## -description
Get or set a value that controls whether a script or other active content can access this [HttpCookie](httpcookie.md).

## -property-value
Whether a script or other active content can access this [HttpCookie](httpcookie.md).

**true** if a script or other active content cannot access this HTTP cookie; otherwise, **false**. The default is **false**.

## -remarks
When this property is set to **true**, a script or other active content cannot access this HTTP cookie.
