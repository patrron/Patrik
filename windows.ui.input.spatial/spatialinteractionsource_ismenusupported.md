# Windows.UI.Input.Spatial.SpatialInteractionSource.IsMenuSupported

## -description
Gets whether the source supports Menu presses.

## -property-value
Whether the source supports menu presses.

## -remarks
This will be true for most spatial controllers and false for other interaction sources.
