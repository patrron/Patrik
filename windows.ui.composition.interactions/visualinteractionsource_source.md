# Windows.UI.Composition.Interactions.VisualInteractionSource.Source

## -description
The visual that is used for hit-testing and defines the co-ordinate space for gesture recognition.

Equivalent WinUI property: [Microsoft.UI.Composition.Interactions.VisualInteractionSource.Source](/windows/winui/api/microsoft.ui.composition.interactions.visualinteractionsource.source).

## -property-value
The visual that is used for hit-testing and defines the co-ordinate space for gesture recognition.
